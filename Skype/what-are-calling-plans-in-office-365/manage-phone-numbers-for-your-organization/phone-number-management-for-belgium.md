# ベルギーの電話番号の管理

> [!IMPORTANT]
> この記事は機械翻訳されています。機械翻訳についての「免責事項」をお読みください。 
  
取得と skype for Business Online ベルギーの電話番号を管理する情報を検索するのにには、次の表を使用します。
  
詳細については、[組織の電話番号を管理する](manage-phone-numbers-for-your-organization.md)を参照してください。
  
|**目的**|**正しいフォームをダウンロードして送信する**|
|:-----|:-----|
|新しい電話番号を取得する。  <br/> | ベルギーでは、Skype for Business 管理センターを使用して、所属する組織のために電話番号を取得することはできません。新しい電話番号の申請を送信する必要があります。次のいずれかのフォームをダウンロードします。 <br/> [ベルギー (EN-US) で、新しい電話番号を取得します。](http://download.microsoft.com/download/3/B/D/3BDD4575-EAFA-4777-B4C6-A42E8F235AC9/New Phone Number Request for Belgium %28Geographic numbers%29 (v.2) (en.US).pdf) <br/> [ベルギーで新しい電話番号を取得する (de-する)](http://download.microsoft.com/download/3/B/D/3BDD4575-EAFA-4777-B4C6-A42E8F235AC9/New Phone Number Request for Belgium %28Geographic numbers%29 (v.3) (de.BE).pdf) <br/> [ベルギーで新しい電話番号を取得する (fr-する)](http://download.microsoft.com/download/3/B/D/3BDD4575-EAFA-4777-B4C6-A42E8F235AC9/New Phone Number Request for Belgium %28Geographic numbers%29 (v.3) (fr.BE).pdf) <br/> [ベルギーで新しい電話番号を取得する (nl-する)](http://download.microsoft.com/download/3/B/D/3BDD4575-EAFA-4777-B4C6-A42E8F235AC9/New Phone Number Request for Belgium %28Geographic numbers%29 (v.3) (nl.BE).pdf) <br/>  フォームを記入し、[送信](mailto:ptneu@microsoft.com)してください。  <br/> |
|電話番号を Skype for Business に移す必要がある。  <br/> | 次のいずれかのフォームをダウンロードします。 <br/> [ベルギー (地域番号) 用の承認状 (LOA) (v.1.0) (en-US)](http://download.microsoft.com/download/F/0/1/F01AE714-0F3C-4D9D-B41A-DFD180EC1622/Letter of Authorization %28LOA%29 for Belgium (Geographic numbers) (v.1.0) (en-US).pdf) <br/> [ベルギー (地域番号) 用の承認状 (LOA) (v.1.0) (nl-BE)](http://download.microsoft.com/download/F/0/1/F01AE714-0F3C-4D9D-B41A-DFD180EC1622/Letter of Authorization %28LOA%29 for Belgium(Geographic numbers) (v.1.0) (nl-BE).pdf) <br/> [ベルギー (地域番号) 用の承認状 (LOA) (v.1.0) (fr-BE)](http://download.microsoft.com/download/F/0/1/F01AE714-0F3C-4D9D-B41A-DFD180EC1622/Letter of Authorization %28LOA%29 for Belgium (Geographic numbers) (v.1.0) (fr-BE).pdf) <br/> [ベルギー (地域番号) 用の承認状 (LOA) (v.1.0) (de-BE)](http://download.microsoft.com/download/F/0/1/F01AE714-0F3C-4D9D-B41A-DFD180EC1622/Letter of Authorization %28LOA%29 for Belgium(Geographic numbers) (v.1.0) (de-BE).pdf) <br/>  フォームを記入し、[送信](mailto:ptneu@microsoft.com)してください。  <br/> |
|国内の電話帳に電話番号を公開/非公開にする必要がある。  <br/> |公開/非公開にしたい電話番号のリストを[送信](mailto:ptneu@microsoft.com)してください。           <br/> |
|ユーザーの電話番号をサービス番号に変更する必要がある。  <br/> |サービス番号に変換する番号のリストを[送信](mailto:ptneu@microsoft.com)してください。  <br/> |
