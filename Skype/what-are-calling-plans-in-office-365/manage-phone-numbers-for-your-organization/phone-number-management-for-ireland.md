# <a name="phone-number-management-for-ireland"></a>アイルランドの電話番号の管理

取得し、アイルランドの電話番号を Skype のオンライン ビジネスの管理上の情報を検索するのにには、次の表を使用します。 
  
詳細については、[組織の電話番号の管理](manage-phone-numbers-for-your-organization.md)を参照してください。
  
|**何かの操作を行います?**|**正しいフォームをダウンロードします。**|
|:-----|:-----|
|新しいユーザーの電話番号を取得します。  <br/> |アイルランドのビジネス管理センターは、Skype を使用して、組織の電話番号を取得できません。 新しい電話番号の要求を送信する必要があります。 ダウンロード、[アイルランド (地理的な番号) の電話番号の新しい要求 (v.3.0)(en.us)](../../downloads/new-phone-number-request-for-ireland-(geographic-numbers)-(v.3.0)-(en.us).pdf)フォームで、[弊社に送信](mailto:ptneu@microsoft.com)してを入力します。  <br/> |
|私たちに転送する電話番号を取得する必要があります。  <br/> |これらのフォームのいずれかをダウンロードします。 <br/> [文字の承認 (山) のアイルランド (地理的な番号) (v.3.0)(en.us)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-ireland-(geographic-numbers)-(v.1.0)-(en.us).pdf) <br/> [文字の承認 (山) のアイルランド (地理的ではない番号) (v.1.0)(en.us)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-ireland-(non-geographic-numbers)-(v.1.0)-(en.us).pdf) <br/>  [弊社に送信](mailto:ptneu@microsoft.com)して入力します。  <br/> |
|全国の電話帳の電話番号を公開非公開にする必要があります。  <br/> |[ご](mailto:ptneu@microsoft.com)公開非公開にする電話番号の一覧です。 <br/> |
|サービス番号にユーザーの数を変更する必要があります。  <br/> |[お送り](mailto:ptneu@microsoft.com )サービス番号に変換するユーザーの番号の一覧です。 <br/> |
