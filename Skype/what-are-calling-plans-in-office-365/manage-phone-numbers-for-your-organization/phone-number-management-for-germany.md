# <a name="phone-number-management-for-germany"></a>ドイツの電話番号の管理

取得して、ドイツの電話番号を Skype のオンライン ビジネスの管理に関する情報を検索するのにには、次の表を使用します。 
  
詳細については、[組織の電話番号の管理](manage-phone-numbers-for-your-organization.md)を参照してください。
  
|**何かの操作を行います?**|**正しいフォームをダウンロードします。**|
|:-----|:-----|
|新しいユーザーの電話番号を取得します。  <br/> | ドイツでは、ビジネス管理センターは、Skype を使用して、組織の電話番号を取得できません。 新しい電話番号の要求を送信する必要があります。 最初に、このフォームをダウンロードします。 <br/> [ドイツ (地理的な番号) の電話番号の新しい要求 (v.3.0)(de.de)](../../downloads/new-number-request-forms/new-phone-number-request-for-germany-(geographic-numbers)-(v.3.0)-(de.de).pdf)。 <br/>  [弊社に送信](mailto:ptneu@microsoft.com)して入力します。  <br/> |
|私たちに転送する電話番号を取得する必要があります。  <br/> | ダウンロード[状の承認 (山) (地理的な番号) をドイツの (v.1.0)(de.de)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-germany-(geographic-numbers)-(v.1.0)-(de.de).pdf) <br/>  [弊社に送信](mailto:ptneu@microsoft.com)して入力します。  <br/> |
|全国の電話帳の電話番号を公開非公開にする必要があります。  <br/> |[ご](mailto:ptneu@microsoft.com)公開非公開にする電話番号の一覧です。 <br/> |
|サービス番号にユーザーの数を変更する必要があります。  <br/> |[お送り](mailto:ptneu@microsoft.com )サービス番号に変換するユーザーの番号の一覧です。 <br/> |
