# ユーザーの電話会議プロバイダーを Microsoft に変更する

Skype for Business および Microsoft Teams を利用して、Office 365 で電話会議を使用するには、組織内のユーザーが自分に割り当てられた **電話会議ライセンス**を持つことと、電話会議プロバイダーを Microsoft に設定することが必要です。ライセンスの取得とその費用の詳細については「[Office 365 の電話会議を試用または購入する](try-or-purchase-audio-conferencing-in-office-365.md)」をご覧ください。
  
## ユーザーの電話会議プロバイダーを Microsoft に変更するには

 **電話会議**ライセンスが、電話会議プロバイダーを利用していないユーザーに割り当てられる場合は、ユーザーのプロバイダーは自動的に Microsoft に設定され、他に何も操作する必要がありません。ユーザーが既に電話会議プロバイダーを利用している場合は、ユーザーに **電話会議**ライセンスを割り当ててからユーザーのプロバイダーを Microsoft に変更する必要があります。
  
 **電話会議**ライセンスが割り当て済みで、別の電話会議プロバイダーを使用していたユーザーの電話会議プロバイダーとして Microsoft を設定するには、次の操作を実行します。
  
1. 職場または学校のアカウントを使用して、Office 365 にサインインします。
    
2. **Office 365 管理センター**、[ **Skype for Business**] の順に移動します。
    
3. [ **Skype for Business 管理センター**] で、[ **電話会議**] に移動します。
    
4. **電話会議**ライセンスが割り当てられているのに Microsoft を電話会議プロバイダーとして設定していないユーザーが存在することがバナーの表示で通知される場合は、[ **ここをクリックして移行**] をクリックします。バナーが表示されない場合は、[ **Skype for Business 管理センター**] で [ **ユーザー**] をクリックして [ **電話会議に移行できる状態のユーザー**] フィルターを選びます。
    
5. 移行するユーザーを選び、操作ウィンドウで [ **編集**] をクリックします。
    
6. [ **Microsoft**] を [ **プロバイダー名**] リストで選びます。
    
    > [!NOTE]
    > ユーザーの電話会議プロバイダーを Microsoft に変更すると、ユーザーの電話会議に関する情報も変更されます。この情報を保持する必要がある場合は、ユーザーのプロバイダーを変更する前に情報を任意の場所に記録するようにしてください。 
  
7. [ **保存**] をクリックします。
    
## 他に必要な情報はありますか?

- リストでユーザーを選ぶと、既定の電話会議情報を表示できますが、電話会議の PIN を表示することはできません。[ **リセット**] をクリックしてユーザーの電話会議の PIN をリセットすることはできます。
    
- ユーザーの電話会議の設定を変更する場合は、リストからユーザーを選んでから [ **編集**] をクリックして、[ **プロパティ**] ページで変更します。
