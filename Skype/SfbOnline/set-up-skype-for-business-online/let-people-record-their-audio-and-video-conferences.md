# <a name="let-people-record-their-audio-and-video-conferences"></a>音声やビデオ会議をレコーディングする

## <a name="choose-the-default-recording-setting-for-a-single-user-or-everyone-in-your-organization"></a>既定の組織内の 1 人のユーザーまたはすべてのユーザー設定のレコーディングを選択します。

1. **Office 365 管理センター**に移動 > **Skype for Business**の場合は、[**ユーザー**] をクリックします。
    
2. [**ユーザー** ] ページで、リストから、または複数のユーザーを選択し、[**編集**] をクリックします。
    
3. **] オプション ページ****レコードの会話と会議**] をクリックし、[**保存**] をクリックします。
    
記録がオンになっている場合、開催者または発表者は Skype for Business Online の会議または電話会議を記録できます。記録のオンまたはオフにするには、組織内のすべてのユーザーに影響します。
  
|**レコーディングに含まれる**|**レコーディングに含まれない**|
|:-----|:-----|
|音声とビデオの HD  <br/> PowerPoint およびホワイトボードによるプレゼンテーション  <br/> デスクトップまたはプログラム共有セッション  <br/> |OneNote の共有ページ  <br/> PowerPoint のコメント  <br/> 共有ファイル (添付ファイル)  <br/> |
   
詳細については、 [Lync のレコーディングの保存し、公開](https://support.office.com/en-US/Article/83cf1507-0f05-40f5-a520-5a6c133bc33e)」と「 [Lync のレコーディング マネージャー](https://support.office.com/en-US/Article/fa915c55-ff55-45c0-b372-1ef5596da10b)を参照してください。
