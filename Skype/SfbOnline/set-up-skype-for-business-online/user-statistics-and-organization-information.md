# <a name="user-statistics-and-organization-information"></a>ユーザーの統計および組織の情報

Skype for Business ダッシュボードでは、自社向けにサービスをセットアップする方法が簡単にわかります。 まだ Skype for Business を構成していない場合、具体的な手順については、「[Skype for Business Online のセットアップ](set-up-skype-for-business-online.md)」をご覧ください。
  
## <a name="how-to-change-skype-for-business-online-user-statistics"></a>Skype for Business Online のユーザーの統計情報を変更する方法

|||
|:-----|:-----|
|**プロパティ** <br/> |**変更方法** <br/> |
|Skype for Business Online のユーザー  <br/> |Skype for Business アプリを使用するユーザーを追加またはブロックするには、「一ビジネス向け [Microsoft 365](https://support.office.com/article/997596b5-4173-4627-b915-36abac6786dc) のライセンスを割り当てるまたは削除する」または「一ビジネス向け [Microsoft 365](https://support.office.com/article/9b497c85-d0a4-4735-80fa-d3565bc05bd1)ライセンスをユーザーから削除する」を参照してください。  <br/> ユーザーが会社を離れた場合の操作の完全な一覧については [、「Microsoft 365](https://support.office.com/article/44d96212-4d90-4027-9aa9-a95eddb367d1)から元従業員を削除する」を参照してください。  <br/> |
|電話会議に対応しているユーザー  <br/> |ユーザーが電話会議を有効にするには [、「Microsoft 365](../audio-conferencing-in-office-365/try-or-purchase-audio-conferencing-in-office-365.md)または Office 365 で電話会議を試用または購入する」を参照してください。  <br/> |
|クラウドのユーザー  <br/> |これは、Microsoft 365 または 365 アカウントを持つ Skype for Business Online ユーザー Office数です。  <br/> Microsoft 365 または Office 365 サブスクリプションにユーザーを追加するには (つまり、Microsoft 365 または Office 365 アカウントを作成する [)、「Office 365](https://support.office.com/article/1970f7d6-03b5-442f-b385-5880b9c256ec)にユーザーを個別または一括で追加する - 管理者向けヘルプ」を参照してください。  <br/> |
|同期済みでオンラインに所属のユーザー  <br/> |これは、Skype for Business Server オンプレミス展開で作成されたハイブリッド ユーザーの数ですが、Microsoft 365 または Office 365 と同期されます。  <br/> Skype for Business のハイブリッド接続をセットアップするには、「[Skype for Business Server と Skype for Business Online 間のハイブリッド接続を展開する](../../SfbHybrid/hybrid/configure-hybrid-connectivity.md?bc=%2fSkypeForBusiness%2fbreadcrumb%2ftoc.json&toc=%2fSkypeForBusiness%2ftoc.json)」をご覧ください。  <br/> |
   
> [!NOTE]
> 利用不可とリストに表示されている場合は、情報がないか、エラーが発生したことになります。 
  
## <a name="how-to-change-information-about-your-organization"></a>組織に関する情報を変更する方法

|||
|:-----|:-----|
|**プロパティ** <br/> |**変更方法** <br/> |
|組織名  <br/> |これは、Microsoft 365 または Office 365 にサインアップするときに入力した名前です。 変更するには、「[組織の住所およびその他のプロファイル情報を変更する](https://support.office.com/article/a36e5a52-4df2-479e-bb97-9e67b8483e10)」をご覧ください。  <br/> |
|組織 ID  <br/> |組織 ID は変更できません。  <br/> |
|組織の作成日時  <br/> |この値は変更できません。  <br/> |
|Active Directory の同期対象  <br/> |自分の組織が、オンプレミス環境である Skype for Business Online を利用している組織に同期されているかどうかを示します。  <br/> オンプレミスの Active Directory をセットアップするには [、「Microsoft 365](https://support.office.com/article/1b3b5318-6977-42ed-b5c7-96fa74b08846)または Office 365 のディレクトリ同期をセットアップする」を参照してください。  <br/> |
|組織のドメイン  <br/> |Microsoft 365 または Office 365 を最初にセットアップした場合、これらは登録されたドメインです。  <br/> ドメインを追加するには [、「Microsoft 365 または Office 365 に複数のドメインを](https://support.office.com/article/2d2fa996-b760-411d-a5cc-190d63f13207)追加する」を参照してください。  <br/> ドメインを削除するには [、「Microsoft 365 または Office 365](https://support.office.com/article/f09696b2-8c29-4588-a08b-b333da19810c)からドメインを削除する」を参照してください。  <br/> |
