# <a name="this-skype-for-business-online-feature-isnt-enabled"></a>この Skype for Business Online の機能が有効になっていません。

申し訳ございませんようこの機能のライセンスが付与されていないユーザーです。
  
|**考えられる原因**|**推奨される措置**|
|:-----|:-----|
|使用可能なライセンスがある場合は、このユーザーの機能をオンにしていません。  <br/> |**Office 365 管理センター**に移動 > **アクティブなユーザー**がこのユーザーのライセンスを取得します。 <br/> |
|ユーザーは、この機能が利用できない場所にします。  <br/> |可能であれば、 **Office 365 管理センター**に移動 > **アクティブなユーザー**機能が利用可能な場所にこのユーザーに割り当てるとします。 <br/> |
|組織の機能を有効にされていません。  <br/> |**Office 365 管理センター**に移動 > **Skype for Business**と組織全体の機能をオンにします。 <br/> |
