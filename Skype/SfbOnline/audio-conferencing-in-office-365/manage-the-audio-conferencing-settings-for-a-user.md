# <a name="manage-the-audio-conferencing-settings-for-a-user"></a>ユーザーの電話会議の設定を管理します。

Office 365 管理者の場合は、電話会議の設定を編集することができます: プロバイダー、既定の有料またはフリー ダイヤル番号、会議 ID、PIN など、組織内の個々 のユーザーのします。組織の設定を編集する場合は、[組織用の電話会議の設定を管理する](manage-the-audio-conferencing-settings-for-my-organization.md)を参照してください。
  
![ユーザーの音声会議のプロパティ ページを表示します。](../images/228550f7-92be-416d-9ab1-7c2ef54dd4e6.png)
  
1. 職場または学校のアカウントで Office 365 にサインインします。
    
2. **管理センター**] を選びます > **Skype for Business**します。
    
3. Skype for Business 管理センター] では、**ユーザー**を選択します。
    
4. 設定を管理するユーザーを選択し、[アクション] ウィンドウで、[**編集**] をクリックして![編集アイコンが表示](../images/4d8bea48-be68-4e0e-a54c-73decf7ea4ec.png)します。
    
5. 左のナビゲーションで**電話会議**を選択し、[ユーザーの [**プロパティ**] ページで、次のいずれかを変更します。
    
|**設定**|**{Description}**|
|:-----|:-----|
|**プロバイダー名** <br/> |リストから、プロバイダーを選択します。  <br/><br/> **メモ:**次の表の他の設定は、電話会議プロバイダーとして Microsoft を選ぶ場合にのみに適用します。           |
|**既定の電話番号**(必須) <br/> |サード パーティのプロバイダーでは、これらの電話番号は、電話会議プロバイダーから受信したものです。ユーザーが電話会議プロバイダーとして Microsoft を使用している場合は、電話会議ブリッジに設定されている数値これらがされます。[数値の書式を設定して、ビジネスや Microsoft チーム会議出席依頼の Skype で表示されるようにしたいとします。  <br/> |
|**既定のフリー ダイヤル umber** <br/> |サード パーティのプロバイダーでは、これらの電話番号は、電話会議プロバイダーから受信したものです。ユーザーが電話会議プロバイダーとして Microsoft を使用している場合は、電話会議ブリッジに設定されている数値これらがされます。[数値の書式を設定して、ビジネスや Microsoft チーム会議出席依頼の Skype で表示されるようにしたいとします。  <br/> |
|**組織の Microsoft ブリッジのフリー ダイヤルの番号を使用して、このユーザーの会議に参加する許可します。** <br/> |会議に参加するため、ユーザーのフリー ダイヤルの番号を入力できるようにする場合は、このオプションを選択します。  <br/> |
|**会議情報を電子メールで送信します。** <br/> |すぐにだれかの電話会議用の ID と電話番号をユーザーにメールを送信する場合にのみ、このリンクをクリックします。(このメールは含まれません PIN。)[この情報は、電話会議情報を持つユーザーに電子メールを送信](send-an-email-to-a-user-with-their-dial-in-information.md)を参照してください。<br/> |
|**電話会議 ID** <br/> |ユーザーの会議 ID をリセットする必要がある場合は、[**リセット**] を選びます。詳細については、[ユーザーの会議 ID をリセットする](reset-a-conference-id-for-a-user.md)を参照してください。<br/> |
|**暗証番号 (PIN)** <br/> |ユーザーの PIN をリセットする必要がある場合は、[**リセット**] を選びます。詳細については、[ユーザーの音声会議の PIN のリセット](reset-the-audio-conferencing-pin-for-a-user.md)を参照してください。<br/> |
|**会議内の最初のユーザーを認証されていない呼び出しを許可します。** <br/> |最初に会議に参加する認証されていない呼び出しを許可するには、このオプションを選択します。  <br/> |
|**このユーザーの会議からダイヤル アウトする制限** <br/> |ダイヤル アウトを国内のみに制限する場合、または会議からすべてダイヤル アウトできないようにする場合は、ボックスの一覧でオプションを選択します。  <br/> |
