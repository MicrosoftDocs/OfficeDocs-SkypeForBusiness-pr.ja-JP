# <a name="compare-audio-conferencing-providers"></a>電話会議プロバイダーを比較します。

設定すると音声会議をセットアップする skype for Business と Microsoft チーム、電話会議プロバイダーを選択する必要があります。電話会議プロバイダーとして Microsoft またはサード パーティを選択します。 
  
> [!IMPORTANT]
> Microsoft チームのユーザーには、サードパーティ電話会議プロバイダーを使用できません。 
  
電話会議プロバイダーは、次を操作します。 
  
- *電話会議ブリッジ*を提供します。ダイヤルイン電話番号、Pin、および会議の会議 Id、会議ブリッジが設定されます。
    
- 有料電話、フリー ダイヤル、ユーザーにダイヤルする国際電話番号を提供します。これらは、会議の出席依頼に含まれている電話番号です。
    
- ユーザーが、会議にダイヤルインするときに聞く言語を設定します。
    
- 会議のユーザーが会議に参加するときに使用されている会議 Id を作成します。
    
- 開催者、電話で会議を開始するのには、暗証番号 (pin) を提供します。
