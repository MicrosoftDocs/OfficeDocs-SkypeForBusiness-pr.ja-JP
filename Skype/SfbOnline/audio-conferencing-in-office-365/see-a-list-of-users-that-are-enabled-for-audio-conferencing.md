# <a name="see-a-list-of-users-that-are-enabled-for-audio-conferencing"></a>音声会議が有効なユーザーの一覧を参照してください。

有効にした後 Skype for Business または Microsoft チームのユーザー電話会議用に組織で、有効になっているユーザーの一覧を表示することができます。電話会議プロバイダーを使用しているが、ユーザーの既定のダイヤルイン電話番号の種類の一覧の各ユーザーにも表示されます] 一覧を確認して組織が動的会議 Id、静的会議 Id を有効になっていない場合整理、電話会議の会議します。
  
## <a name="viewing-a-list-of-users"></a>ユーザーの一覧を表示します。

1. 職場または学校のアカウントで Office 365 にサインインします。
    
2. **Office 365 管理センター**に移動 > **Skype for Business**します。
    
3. **Skype for Business 管理センター**、左のナビゲーションで**電話会議**に移動 > **ユーザー**。
    
## <a name="what-else-should-i-know"></a>他に必要な情報はありますか?

- 有効になっているユーザーのリストを表示すると、リストからユーザーを選択し、操作ウィンドウを使用して、そのユーザーの電話会議の設定を編集することができます。
    
- 電話会議プロバイダーとして Microsoft を使用するように構成されている 1 人のユーザーを選択し、既定の電話番号との動的会議 Id、組織が有効になっているかどうかを表示することができます、会議の会議 ID をリセットすることができますが、ユーザーを整理します。
    
- サードパーティ電話会議プロバイダーを使用するように構成されている 1 人のユーザーを選択すると、(設定されている) 場合、電話会議プロバイダー、有料電話番号、フリー ダイヤル電話番号の名前を確認できます。
    
- 持つユーザーを表示するのには、フィルター オプションを使用できます。
    
  - **[電話会議**
    
  - **電話会議をオフ**
    
  - **会議のプロバイダーから Microsoft**
    
  - **会議のプロバイダーの他のユーザー**
    
- [検索] ボタンを使用すると、リスト内の個々 のユーザーを検索します。
    
- 1 つ以上のユーザーを選択して、次の操作します。
    
  - そのユーザーに別の既定の番号を選択します。
    
  - **[なし]**に、プロバイダーを変更することによって、ユーザーの電話会議オフにします。
    
  - ユーザーに**電話会議**のライセンスが割り当てられている場合、電話会議プロバイダーとして Microsoft に切り替えます。
    
  - 許可/禁止匿名ユーザーが選択されているユーザーの電話会議を有効にします。
