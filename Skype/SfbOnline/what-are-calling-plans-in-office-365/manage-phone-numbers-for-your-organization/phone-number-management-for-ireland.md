# <a name="phone-number-management-for-ireland"></a>アイルランド用の電話番号の管理

取得と skype for Business Online アイルランドの電話番号を管理する情報を検索するのにには、次の表を使用します。 
  
詳細については、[組織の管理の電話番号](manage-phone-numbers-for-your-organization.md)を参照してください。
  
|**どのようなするすればよいですか。**|**正しい形式をダウンロードします。**|
|:-----|:-----|
|新しいユーザーの電話番号を取得します。  <br/> |アイルランドでは、Business 管理センターの Skype を使用して、組織の電話番号を取得できません。新しい電話番号の依頼を送信する必要があります。ダウンロード、[アイルランド (地理的な番号) の電話番号の新しい要求 (v.3.0)(en.us)](../../downloads/new-number-request-forms/new-phone-number-request-for-ireland-(geographic-numbers)-(v.3.0)-(en.us).pdf)フォームで、入力して、[お問い合わせに送信](mailto:ptneu@microsoft.com)します。<br/> |
|お問い合わせに転送する電話番号を取得する必要があります。  <br/> |これらの形式のいずれかをダウンロードするには。 <br/> [文字の承認 (山) アイルランド (地理的な番号) の (v.3.0)(en.us)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-ireland-(geographic-numbers)-(v.1.0)-(en.us).pdf) <br/> [文字の承認 (山) アイルランド (非地理番号) の (v.1.0)(en.us)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-ireland-(non-geographic-numbers)-(v.1.0)-(en.us).pdf) <br/>  [お問い合わせに送信](mailto:ptneu@microsoft.com)して入力します。  <br/> |
|国内電話ディレクトリ内の電話番号を公開/非公開にする必要があります。  <br/> |[送信](mailto:ptneu@microsoft.com)発行発行する電話番号のリストをします。 <br/> |
|サービス番号をユーザーの数を変更する必要があります。  <br/> |[送信](mailto:ptneu@microsoft.com )サービスの数値に変換するユーザーの数値のリストをします。 <br/> |
