# <a name="why-am-i-having-problems-entering-uk-address-information"></a>英国の入力で問題が発生するのはなぜですか 住所情報

ここでは、英国 (英国) で住所または住所を正しく検証するときに問題が発生する理由をいくつか示します。
  
## <a name="whats-wrong-with-the-company-name"></a>会社名に問題がある場合

会社名の入力に問題がある場合は、次の点を確認してください。
  
- 文字、数字、および次の特殊文字を使用できます。 ' &amp; " _ 
    
    > [!CAUTION]
    > 英国 (英国) では、会社名に + 記号を付けることはできません。 
  
- 80文字以下にする必要があります。
    
- 特殊文字で始めることはできません。
    
## <a name="whats-wrong-with-the-house-number"></a>番地に問題がある場合

番地の入力に問題がある場合は、次の点を確認してください。
  
- 文字、数字、および次の特殊文字を使用できます:-/[]
    
- 数字で始める必要があります。
    
- 20文字以下にする必要があります。
    
- 大文字と小文字のどちらでもかまいません。
    
## <a name="whats-wrong-with-the-house-name"></a>家の名前に問題がある場合

ハウス名の入力に問題がある場合は、次の点を確認してください。
  
- 文字、数字、および次の特殊文字を使用できます。 ' &amp; " _
    
- 50文字以下にする必要があります。
    
- 特殊文字で始めることはできません。
    
## <a name="whats-wrong-with-the-street-name"></a>ストリート名に問題がある場合

住所の入力に問題がある場合は、次の点を確認してください。
  
- 文字、数字、および次の特殊文字を使用できます。 ' &amp; " _ 
    
- 50文字以下にする必要があります。
    
- 特殊文字で始めることはできません。 
    
## <a name="whats-wrong-with-the-city-or-town"></a>市区町村や市町村に問題がありますか?

市町村名の入力に問題がある場合は、次の点を確認してください。
  
- 文字、数字、および次の特殊文字を使用できます。 ' &amp; " _
    
- 50文字以下にする必要があります。
    
- 特殊文字で始めることはできません。 
    
## <a name="whats-wrong-with-the-postal-code"></a>郵便番号に問題がある場合

郵便番号の入力に問題がある場合は、次の点を確認してください。
  
- 文字、数字、スペースのみを指定できます。
    
- 数字やスペースなど、8文字以下の文字を使用する必要があります。
    
- 英国 **(英国) を使用**している場合は、次の形式のいずれかで郵便番号を入力する**必要があり**ます。ここでは、文字 a = 文字 a-z、# sign = 0-9 からの番号、_ (アンダースコア) = 空白スペースを指定する必要があります。 いくつかの例を次に示します。
    
|**表示**|**例**|
|:-----|:-----|
|AA # A_ # AA  <br/> |EC3B 2BD  <br/> |
|# A_ # AA  <br/> |E1W 1AP  <br/> |
|# _ # AA  <br/> |F1 2AG  <br/> |
|# #_ # AA  <br/> |C44 8HT  <br/> |
|AA # _ # AA  <br/> |DR3 6HG  <br/> |
|AA # #_ # AA  <br/> |FN35 1TH  <br/> |

> [!NOTE]
> さらに追加で電話番号が必要な場合は、「[一般法人向け Office 365 のサポートへのお問い合わせ - 管理者向けヘルプ](https://support.office.com/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b)」をご覧ください。
