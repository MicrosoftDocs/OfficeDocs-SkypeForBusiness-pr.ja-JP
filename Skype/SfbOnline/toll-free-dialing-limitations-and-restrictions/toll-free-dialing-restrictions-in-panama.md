# <a name="toll-free-dialing-restrictions-in-panama"></a>パナマでのフリー ダイヤルの制限

[別の国または地域を検索する](../what-are-calling-plans-in-office-365/toll-free-dialing-limitations-and-restrictions.md)には、フリー ダイヤルの制限事項と制限事項の詳細についてはまたはです。


|**詳細**|**説明**|
|:-----|:-----|
|呼び出し、利用可能なフリー ダイヤルですか。  <br/> |あり  <br/> |
|ダイヤル形式  <br/> | 00-800-XXX-XXXX <br/> |
|国/地域の ISO コード  <br/> |PA  <br/> |
|対象となる島/地域  <br/> |該当しない  <br/> |
|対象外の島/地域  <br/> |該当しない  <br/> |
|有線/固定回線の状態  <br/> |C&amp;W からのみ。  <br/> |
|携帯電話ネットワークの状態  <br/> |なし  <br/> |
|コメント、制限、アクセスの問題など  <br/> |お客様は、フリー ダイヤル番号の前に 011 (国際長距離パナマ コード) をダイヤルする必要があります。  <br/> |
