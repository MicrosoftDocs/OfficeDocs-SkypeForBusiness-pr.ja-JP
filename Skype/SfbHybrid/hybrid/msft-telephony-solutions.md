# <a name="microsoft-telephony-solutions"></a>Microsoft テレフォニー ソリューション

Microsoft は、Microsoft クラウドで Teams への移行を開始する場合に、いくつかのオプションをサポートしています。 この記事では、組織内のユーザーに適切な Microsoft テレフォニー ソリューション (クラウドまたは エンタープライズ VoIP オンプレミスの電話システム) と、組織が公衆交換電話網 (PSTN) に接続する方法を決定するのに役立ちます。

この記事は、関連する技術図と共に使用する必要があります。これは、組織に対して正しい決定を下す視覚的な支援を提供します。

## <a name="private-branch-exchange-pbx-options"></a>プライベート ブランチ Exchange (PBX) オプション

### <a name="phone-system-microsoft-365-or-office-365"></a>電話システム (Microsoft 365 または Office 365)
  
電話システムは、Microsoft Teams または Skype for Business Online を使用して、Microsoft 365 または Office 365 クラウドで通話制御およびプライベート ブランチ Exchange (PBX) 機能を有効にする Microsoft のテクノロジです。

電話システムは、Teams または Skype for Business Online クライアントおよび認定デバイスで動作します。 電話システムを使用すると、既存の PBX システムを、Microsoft 365 または Office 365 から直接配信され、会社のクラウド生産性エクスペリエンスに緊密に統合された一連の機能に置き換けることができます。 電話システムを公衆交換電話網 (PSTN) に接続するには、Microsoft の通話プランまたは独自のテレフォニー キャリアを選択できます。

詳細については [、「What is Phone System in Microsoft 365 or Office 365」を参照してください](/MicrosoftTeams/what-is-phone-system-in-office-365)。

### <a name="enterprise-voice-skype-for-business-server"></a>エンタープライズ VoIP (Skype for Business Server)

エンタープライズ VoIPは、オンプレミスの Skype for Business Server で通話制御とプライベート ブランチ Exchange (PBX) 機能を有効にする Microsoft のテクノロジです。 この Skype for Business オプションは、独自のテレフォニー キャリアを使用して公衆交換電話網にのみ接続できます。

詳細については [、「Plan for エンタープライズ VoIP Skype for Business Server」を参照してください](../../SfbServer/plan-your-deployment/enterprise-voice-solution/enterprise-voice.md?bc=%2fSkypeForBusiness%2fbreadcrumb%2ftoc.json&toc=%2fSkypeForBusiness%2ftoc.json)。

## <a name="connection-to-the-public-switched-telephone-network-pstn-options"></a>公衆交換電話網 (PSTN) オプションへの接続

次の方法で公衆交換電話網 (PSTN) に接続できます。

- Microsoft 365 または microsoft 365 または Office 365 での Microsoft 通話プランの使用 
- 独自のテレフォニー キャリアの接続

### <a name="calling-plan-microsoft-365-or-office-365"></a>通話プラン (Microsoft 365 または Office 365)

このオプションは、Microsoft 365 または Office 365 電話システムを公衆交換電話網 (PSTN) に接続して、世界中の固定電話や携帯電話への通話を有効にします。 通話プランでは、Microsoft は PSTN 通信事業者です。

詳細については [、「Calling Plans for Microsoft 365 or Office 365」を参照してください](/MicrosoftTeams/calling-plans-for-office-365)。

### <a name="connect-your-own-telephony-carrier-microsoft-365-or-office-365-and-skype-for-business-on-premises"></a>独自のテレフォニー キャリアを接続する (Microsoft 365 または Office 365 および Skype for Business オンプレミス)

このオプションは、Microsoft 365 の電話システムまたは Office 365 または skype for Business オンプレミスの エンタープライズ VoIP システムをテレフォニー ネットワークに接続します。 このオプションには、サポートされているセッション ボーダー コントローラー (SBC) が必要です。 場合によっては、このオプションでは、オンプレミスに展開された追加の Microsoft ソフトウェアが必要になる場合があります。

## <a name="which-solution-is-right-for-your-organization"></a>組織に最適なソリューションは何ですか?

クラウド内ソリューション、接続専用キャリア ソリューション、またはオール イン ザ クラウドとサード パーティのキャリアの組み合わせを選択できます。

- 通話プラン付き電話システム (すべてクラウド内)

- ダイレクト ルーティングによる独自のキャリアを持つ電話システム

- Skype for Business Server または Cloud Connector Edition 経由で独自のキャリアを持つ電話システム

- エンタープライズ VoIP Skype for Business Server で独自のキャリアを使用する場合

選択するソリューションは、次のような現在および将来のニーズによって異なります。

- オンプレミスの展開で提供される機能を保持する場合は、必要に応じて保持します。
- ユーザーに展開するクライアント。
- クラウドにユーザーを移動する場合の計画。
- サードパーティの PBX や他のテレフォニー機器と相互運用する必要があるかどうか。

組織に最適なソリューションを決定するには、次の質問を検討してください。

- 既存の Skype for Business Server 展開がありますか?
- ユーザーは、オンプレミスの Skype for Business、Skype for Business Online のクラウド、または両方に保存されていますか? 
- オンプレミスのユーザーをクラウドに移動しますか?
- Microsoft の PSTN 通話プランは地域で利用できますか?
- 現在のテレフォニー キャリアを維持する必要がありますか?  たとえば、既存の契約のために現在のキャリアを維持する必要がありますか?
- 既存のオンプレミスレガシ PBX を必要とするか、保持する必要がありますか?
- 現在の従来の PBX は、ビジネスにとって重要な固有の機能を提供していますか?
- 電話システムで現在提供されていない機能が必要なユーザー、またはすべてのユーザーが必要ですか?

次の点に注意してください。

- 複雑な環境向けソリューションを設計する必要がある場合や、複数ステップの移行を管理する必要がある場合は、4 つのオプションすべてが互いに共に存在する可能性があります。
- Skype for Business Server またはクラウド コネクタ エディション経由で独自のキャリアを持つ電話システムは、Skype for Business Server またはクラウド コネクタでのみ展開できます。 Skype for Business Server とクラウド コネクタの共同存在は、1 つの会社ではサポートされていません。

## <a name="phone-system-with-calling-plan"></a>通話プランが設定された電話システム。


通話プランを使用した電話システムは、次の図に示すように、Teams または Skype for Business Online ユーザーのクラウド内のオール イン ザ クラウド オプションです。

![通話プランが設定された電話システム。](../../sfbserver2019/media/msft-telephony-solutions-1.png)


- Microsoft Phone System に国内通話プランまたは国際通話プランが追加され、世界中の電話に通話できます (ライセンスを受け取るサービス のレベルに応じて)。 
- PSTN 通話プランは Microsoft 365 または Office 365 から動作しますので、このオプションでは、オンプレミス展開の展開やメンテナンスは必要ではありません。
- お客様は、サード パーティ製 PBX、アナログ デバイス、および SBC でサポートされている他のサード パーティのテレフォニー 機器との相互運用性を確保するために、サポートされている SBC をダイレクト ルーティング経由で接続できます。

| インフラストラクチャの要件                   | 必須|
| :----------------------------------------------------| ---------:|
| Microsoft 365 または microsoft 365 または Office 365 への接続が必要 | はい |
| 世界中で利用可能*                            | いいえ  |
| サポートされているセッション ボーダー コントローラー (SBC) の展開と保守が必要 | いいえ |
| サードパーティの通信事業者との契約が必要      | いいえ   |
| Skype for Business Server または Cloud Connector Edition の展開と保守が必要 | いいえ |

\* 通話プランが利用可能な国の詳細については、「電話会議と通話プランの国と地域の可用性」 [を参照してください](/MicrosoftTeams/country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans)。


次の質問に「はい」と答える場合は、これが適切なソリューションです。

- 通話プランは地域で利用できます。
- 現在の PSTN 通信事業者を保持する必要はない。
- 公衆交換電話網 (PSTN) への Microsoft が管理するアクセスを使用する場合。
- セッション ボーダー コントローラーを自分で管理する必要はない。
- Teams や Skype for Business Online には、組織で必要なすべての機能があります。

詳細については [、「What is Phone System in Microsoft 365 and Office 365」](/MicrosoftTeams/what-is-phone-system-in-office-365) および [「Microsoft 365](/MicrosoftTeams/calling-plans-for-office-365)または Office 365 の通話プラン」を参照してください。

## <a name="phone-system-with-own-carrier-via-direct-routing"></a>ダイレクト ルーティングによる独自のキャリアを持つ電話システム

このオプションは、クラウド内の Microsoft Phone System に、Teams ユーザー用の事実上すべてのテレフォニー キャリアを提供します。

![ダイレクト ルーティングを介して通信事業者と電話システム](../../sfbserver2019/media/msft-telephony-solutions-2.png)

- 追加のオンプレミス ソフトウェアを必要とせずに、サポートされている独自の SBC を Microsoft Phone System に直接接続します。  
- Microsoft Phone System で、事実上すべてのテレフォニー キャリアを使用します。
- 顧客または通信事業者またはパートナーによって構成および管理できます (通信事業者またはパートナーがこのオプションを提供するか確認してください)。
- サードパーティの PBX やアナログ デバイスなど、テレフォニー機器と Microsoft Phone System の間の相互運用性を構成します。

| インフラストラクチャの要件                   | 必須|
| :----------------------------------------------------| ---------:|
| Microsoft 365 または microsoft 365 または Office 365 への接続が必要 | はい |
| 世界中で利用可能                            | はい  |
| サポートされているセッション ボーダー コントローラー (SBC) の展開と保守が必要 | はい |
| サードパーティの通信事業者との契約が必要*      | はい   |
| Skype for Business Server または Cloud Connector Edition の展開と保守が必要 | いいえ |

\* 通話プランを使用して電話システムに接続しているユーザーに、サードパーティの PBX、アナログ デバイス、その他のテレフォニー機器への接続を提供するオプションとして展開されていない限り。

次の質問に「はい」と答える場合は、これが適切なソリューションです。

- 電話システムによって Teams を使用する場合。
- 現在の PSTN 通信事業者を維持する必要がある。
- ルーティングを混在したい場合は、通話プランを経由して通話を行う場合があります。一部は通信事業者経由で行います
- サードパーティ製の PBX や、オーバーヘッド ページ、アナログ デバイスなどの機器と相互運用する必要があります。
- Teams には、組織で必要なすべての機能があります。

詳細については [、「What is Phone System in Microsoft 365 and Office 365」](/MicrosoftTeams/what-is-phone-system-in-office-365) および [「Plan Direct Routing 」を参照してください](/MicrosoftTeams/direct-routing-plan)。


## <a name="phone-system-with-own-carrier-via-skype-for-business-server-or-cloud-connector-edition"></a>Skype for Business Server または Cloud Connector Edition 経由で独自のキャリアを持つ電話システム

> [!Important]
> Skype for Business Online は 2021 年 7 月 31 日に廃止され、その後サービスにアクセスできなくなりました。  さらに、Skype for Business Server または Cloud Connector Edition と Skype for Business Online を介したオンプレミス環境間の PSTN 接続はサポートされなくなりました。  直接ルーティングを使用してオンプレミスのテレフォニー ネットワークを Teams に接続する方法 [について説明します](/MicrosoftTeams/direct-routing-landing-page)。

このオプションは、クラウド内の Microsoft Phone System に、Skype for Business Online ユーザー用のオンプレミステレフォニー ネットワークへの接続を提供します。

![Skype for Business Server またはクラウド コネクタ エディション経由で通信事業者と電話システム](../../sfbserver2019/media/msft-telephony-solutions-3.png)

 - オンプレミスに展開された Skype for Business Server または Skype for Business Cloud Connector Edition を介して、サポートされている独自の SBC を Microsoft Phone System に接続します。 
- Microsoft Phone System で、事実上すべてのテレフォニー キャリアを使用します。 
- Skype for Business Server オンプレミスが既にある場合は、それを利用できます。 展開しない場合は、より軽量なバージョン (Cloud Connector Edition) を展開できます。


| インフラストラクチャの要件                   | 必須|
| :----------------------------------------------------| ---------:|
| Microsoft 365 または microsoft 365 または Office 365 への接続が必要 | はい |
| 世界中で利用可能                            | はい  |
| サポートされているセッション ボーダー コントローラー (SBC) の展開と保守が必要 | はい |
| サードパーティの通信事業者との契約が必要      | はい   |
| Skype for Business Server または Cloud Connector Edition の展開と保守が必要 | はい |



次の質問に「はい」と答える場合は、これが適切なソリューションです。

- ユーザーに Skype for Business Online を使用する場合。
- PSTN 通話プランは地域では使用できません。
- 現在の PSTN 通信事業者を維持する必要がある。

詳細については、「What is Phone [System in Microsoft 365 and Office 365](/MicrosoftTeams/what-is-phone-system-in-office-365)」、Skype [for Business Server 2019、Plan](../../SfBServer2019/skype-for-business-server-2019.yml) [for Skype for Business Cloud Connector Edition](../../SfbServer/skype-for-business-hybrid-solutions/plan-your-phone-system-cloud-pbx-solution/plan-skype-for-business-cloud-connector-edition.md)を参照してください。

推奨事項: ビジネス条件が変わると、たとえば、PSTN キャリアを保持する必要がなくなりました。オプション 1 または 2 を使用して Microsoft Teams に移動して以下を行う必要はありません。
- メンテナンス コストを最小限に抑える
- Microsoft がリリースした最新の機能にアクセスする

- Microsoft Phone System で、事実上すべてのテレフォニー キャリアを使用します。 
- 展開と保守を行う最も複雑なオプション。

| インフラストラクチャの要件                   | 必須|

| Skype for Business Server の展開と保守が必要 | はい |
