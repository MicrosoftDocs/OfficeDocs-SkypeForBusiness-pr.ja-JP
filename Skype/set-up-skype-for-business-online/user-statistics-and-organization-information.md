# ユーザーの統計情報と組織の情報

> [!IMPORTANT]
> この記事は機械翻訳されています。機械翻訳についての「免責事項」をお読みください。 
  
Skype for Business ダッシュボードを使用すると、自社向けにサービスを設定する方法がすぐにわかります。まだ Skype for Business を構成していない場合、具体的な手順については、「[Skype for Business Online のセットアップ](set-up-skype-for-business-online.md)」を参照してください。
  
## Skype for Business Online のユーザーの統計情報を変更する方法

|||
|:-----|:-----|
|**プロパティ** <br/> |**変更方法** <br/> |
|Skype for Business Online ユーザー  <br/> |Skype for Business アプリを使用するユーザーを追加したり、使用を禁止するには、「[一般法人向け Office 365 ライセンスの割り当てまたは解除方法](https://support.office.com/article/997596b5-4173-4627-b915-36abac6786dc)」または「[Remove licenses from users in Office 365 for business](https://support.office.com/article/9b497c85-d0a4-4735-80fa-d3565bc05bd1)」をご覧ください。  <br/> ビジネスがユーザーの操作の一覧については、 [Office 365 から元従業員を削除する](https://support.office.com/article/44d96212-4d90-4027-9aa9-a95eddb367d1)を参照してください。  <br/> |
|ダイヤルイン会議が有効なユーザー  <br/> |ダイヤルイン会議の参加者を有効にするのには、 [Skype for Business および Microsoft Teams の電話会議のセットアップ](../audio-conferencing-in-office-365/set-up-audio-conferencing-for-skype-for-business-and-microsoft-teams.md)を参照してください。  <br/> |
|クラウドのユーザー  <br/> |これは、Skype for Business Online アカウントを持つ Office 365 ユーザーの数です。  <br/> ユーザーを Office 365 サブスクリプションに追加するには (つまり、ユーザー用に Office 365 アカウントを作成するには)、「[Office 365 にユーザーを個別に、またはまとめて追加する - 管理者向けヘルプ](https://support.office.com/article/1970f7d6-03b5-442f-b385-5880b9c256ec)」を参照してください。  <br/> |
|オンラインに存在して同期されているユーザー  <br/> |これは、オンプレミスの Active Directory と同期されている Skype for Business Online ユーザーの数です。  <br/> オンプレミスの Active Directory を設定するには、[Office 365 のディレクトリ同期をセットアップする](https://support.office.com/article/1b3b5318-6977-42ed-b5c7-96fa74b08846)」を参照してください。  <br/> |
   
> [!NOTE]
> [利用不可] と表示されている場合、情報が存在しないか、エラーが発生したことを示します。 
  
## 組織についての情報を変更する方法

|||
|:-----|:-----|
|**プロパティ** <br/> |**変更方法** <br/> |
|組織名  <br/> |これは、Office 365 にサインアップしたときに入力した名前です。これを変更するには、「[組織の住所およびその他のプロファイル情報を変更する](https://support.office.com/article/a36e5a52-4df2-479e-bb97-9e67b8483e10)」を参照してください。  <br/> |
|組織 ID  <br/> |組織 ID は変更できません。  <br/> |
|組織の作成日時  <br/> |この値は変更できません。  <br/> |
|社内の Active Directory 同期  <br/> |これは、内部設置型であるSkype for Business Online組織に、組織が同期しているかどうか。  <br/> オンプレミスの Active Directory を設定するには、[Office 365 のディレクトリ同期をセットアップする](https://support.office.com/article/1b3b5318-6977-42ed-b5c7-96fa74b08846)」を参照してください。  <br/> |
|組織のドメイン  <br/> |これらは、Office 365 を最初に設定したときに登録したドメインです。  <br/> ドメインを追加するには、 [Office 365 にドメインを追加する](https://support.office.com/article/2d2fa996-b760-411d-a5cc-190d63f13207)を参照してください。  <br/> ドメインを削除するには、「[Office 365 からドメインを削除する](https://support.office.com/article/f09696b2-8c29-4588-a08b-b333da19810c)」を参照してください。  <br/> |
