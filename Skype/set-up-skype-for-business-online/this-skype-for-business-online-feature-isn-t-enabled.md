# この Skype for Business Online の機能は有効でない

> [!IMPORTANT]
> この記事は機械翻訳されています。機械翻訳についての「免責事項」をお読みください。 
  
申し訳ありませんが、現在のユーザーにはこの機能を使用するライセンスがないようです。
  
|**考えられる原因**|**推奨される措置**|
|:-----|:-----|
|使用可能なライセンスがある場合は、このユーザーの機能をオンにしていません。  <br/> |**Office 365 管理センター**に移動 > **アクティブなユーザー**がこのユーザーのライセンスを取得します。  <br/> |
|ユーザーは、この機能が利用できない場所にします。  <br/> |可能であれば、 **Office 365 管理センター**に移動 > **アクティブなユーザー**機能が利用可能な場所にこのユーザーに割り当てるとします。  <br/> |
|この機能が、あなたの組織に対して有効になっていません。  <br/> |[ **Office 365 管理センター**]、[ **Skype for Business**] の順に選んで、組織全体に対してこの機能を有効にしてください。  <br/> |
