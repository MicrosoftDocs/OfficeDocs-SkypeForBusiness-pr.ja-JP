# <a name="administering-servers-after-migration"></a>移行後のサーバーの管理

一般に、管理対象サーバーのバージョンに対応する管理ツールを使用する必要があります。 管理ツールは、同じコンピューター上の異なるバージョンをインストールできません。
  
> [!IMPORTANT]
> ビジネス サーバー プールのパイロット、Skype が配置されると、2019 リソースを管理するためにトポロジ ビルダーまたはコントロール パネルを使用できません。 2019 ツールを使用すると、現在および以前のバージョンのすべてのリソースを管理する必要があります。
