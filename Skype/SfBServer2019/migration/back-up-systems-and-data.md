# <a name="back-up-systems-and-data"></a>システムとデータのバックアップ

移行を開始する前に、システム全体のバックアップを実行し、既存のシステム (各プールに保存されているユーザー アカウントのインベントリを含む) を文書化して、必要になった場合にロールバックできます。 データ、設定、およびシステムのバックアップと復元には、複数のツールおよびプログラムを使用できます。

詳細と手順については、「バージョンのバックアップと復元の手順」を参照してください。
