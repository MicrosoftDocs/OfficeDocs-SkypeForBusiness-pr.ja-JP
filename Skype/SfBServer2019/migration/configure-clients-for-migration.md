# <a name="configure-clients-for-migration"></a>移行用のクライアントの構成

このトピックでは、2019 年に移行する前に実行する推奨されるクライアントSkype for Business Serverします。 これらの構成の変更は、Lync Server 2013 または Skype for Business Server 2015 (移行するバージョン) で行う必要 **があります**。
  
### <a name="to-configure-clients-before-migration"></a>移行前にクライアントを構成するには

1. 既存のインストールの最新のサーバー、クライアント、デバイスの更新プログラム (修正プログラム) を展開します。
    
2. 以前のバージョンのクライアント Skype for Business Serverを使用して、最新の更新プログラムがインストールされているクライアントのみを許可します。
