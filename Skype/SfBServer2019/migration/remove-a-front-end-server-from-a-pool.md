# <a name="remove-a-front-end-server-from-a-pool"></a>プールからフロント エンド サーバーを削除します。

フロント エンド サーバーは、スタンドアロンのコンピューターとして存在できません。 プール内に 1 台のコンピューターだけがある場合でも、フロント エンド プールとして定義しなければなりません。
  
このトピックでは、既存のフロント エンド プールから個々 のフロント エンド サーバーを削除するプロセスについて説明します。 フロント エンド サーバーは、プール内の最後のサーバーまたはプールを完全に削除する場合は、[削除のフロント エンド プールまたは Standard Edition サーバー](remove-front-end-pool-or-standard-edition-server.md)を参照してください。 フロント エンド プールを削除する前に、個々 のフロント エンド サーバーを削除する必要はありません。 プールを削除すると、各フロント エンド サーバーを削除します。
  
### <a name="to-remove-a-front-end-server-from-a-pool"></a>プールからフロント エンド サーバーを削除するのには

1. ビジネス 2019 フロント エンド サーバーの Skype、トポロジ ビルダーを開きます。
    
2. 従来のインストール] ノードに移動します。
    
3. **エンタープライズ エディションのフロント エンド プール**を展開し、削除を削除するには、フロント エンド サーバーを右クリックし、[**削除**] をクリックするとフロント エンド サーバーとフロント エンド プールを展開します。
