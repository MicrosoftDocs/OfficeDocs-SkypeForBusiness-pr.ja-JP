# <a name="run-best-practices-analyzer"></a>ベスト プラクティス アナライザーを実行します。

ベスト プラクティス アナライザー ツールは、展開から構成情報を収集し、マイクロソフトのベスト プラクティスに従って構成が設定されているかどうかを決定します。 ツールは、Microsoft.NET Framework 3.5 Service Pack 1 を実行しているクライアント コンピューター上、または従来のインストールを実行するサーバー上に直接インストールできます。 インストールおよびクライアント コンピューターでこのツールを実行することをお勧めします。 管理ツールもインストールしてくださいローカル クライアント コンピューターにするため、ベスト プラクティス アナライザーは、データの完全なセットを収集できます。 
  
ベスト プラクティス アナライザーをダウンロードするには、Microsoft ダウンロード センターから[https://go.microsoft.com/fwlink/p/?linkid=246173](https://go.microsoft.com/fwlink/p/?linkid=246173)。
