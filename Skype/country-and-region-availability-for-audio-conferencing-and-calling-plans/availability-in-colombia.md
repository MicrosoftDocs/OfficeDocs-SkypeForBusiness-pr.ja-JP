# <a name="availability-in-colombia"></a>コロンビアでの利用可能性

コロンビアで利用できる Skype for Business と Microsoft Teams の Office 365 サービスを以下に示します。

|**機能**|**詳細**|
|:-----|:-----|
|電話会議は購入できますか?  <br/> |はい  <br/> |

|折り返し通話機能を使用するダイヤルアウト機能  <br/> |利用不可  <br/> |
