# <a name="skype-for-business-server-2015-management-tools"></a>Skype for Business Server 2015 管理ツール
 
**概要:** Skype for Business Server 2015 のサービス管理ツールについて説明します。
  
Skype for Business Server 2015 communications software (旧 Lync Server) には、エンタープライズレベルのコラボレーション要件をサポートするインスタントメッセージング (IM)、プレゼンス、会議、およびテレフォニーのソリューションが用意されています。 これらのサービスは、柔軟で強力なツールで管理できます。
