# <a name="skype-for-business-server-2015-management-tools"></a>Skype for Business Server 2015 管理ツール
 
**概要:** 2015 年のサービス管理ツールSkype for Business Server。
  
Skype for Business Server 2015 コミュニケーション ソフトウェア (旧 Lync Server) は、エンタープライズ レベルのコラボレーション要件をサポートするインスタント メッセージング (IM)、プレゼンス、会議、テレフォニー ソリューションを提供します。 これらのサービスを管理するためのツールは、柔軟性と強力な両方です。
