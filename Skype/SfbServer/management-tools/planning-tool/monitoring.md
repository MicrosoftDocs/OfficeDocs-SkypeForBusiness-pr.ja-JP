# <a name="monitoring-planning-tool"></a>監視 (計画ツール)

Skype for Business Server 2015 では、監視を使用して、ユーザーが関与する通信セッションに関する使用状況情報と QoE (Quality of Experience) データを収集します。

監視サーバーの計画の詳細については、「監視の計画」 [を参照してください](https://technet.microsoft.com/library/26cead5a-183c-42f1-a4b0-0e8d61c6159d.aspx)。
