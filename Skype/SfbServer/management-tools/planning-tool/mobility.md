# <a name="mobility-planning-tool"></a>Mobility (Planning Tool)
 
Skype for Business Server 2015 のモビリティ機能は、モバイル デバイスでの Skype for Business 機能をサポートしています。
  
モビリティ機能は Lync Server 2013 と同じです。 モビリティの環境の準備の詳細については [、「Plan for Mobility for Skype for Business Server」](../../plan-your-deployment/mobility.md)を参照してください。
