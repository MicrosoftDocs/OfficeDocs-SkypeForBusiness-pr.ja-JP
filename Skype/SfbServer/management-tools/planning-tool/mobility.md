# <a name="mobility-planning-tool"></a>Mobility (Planning Tool)
 
Skype for Business Server 2015 のモバイル機能は、モバイルデバイスでの Skype for Business の機能をサポートしています。
  
モバイル機能は、Lync Server 2013 と同じです。 モバイル環境の準備について詳しくは、「 [Skype For Business Server のモビリティの計画](../../plan-your-deployment/mobility.md)」をご覧ください。
