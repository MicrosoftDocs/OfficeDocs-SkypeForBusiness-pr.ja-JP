# <a name="start-services"></a>サービスの開始
 
展開されたサーバーの役割でサービスを開始するには、**手順 4**で [**実行**] をクリックします。 ローカルの管理者グループまたは同等のメンバーとしてログオンしている必要があります。 
  
> [!NOTE]
> **開始サービス**が実行されている場合は、[実行] を**もう一度**クリックしてコマンドを再実行できます。
