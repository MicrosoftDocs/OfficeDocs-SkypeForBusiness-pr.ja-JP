# <a name="view-certificate"></a>証明書の表示
 
[**証明書の表示**] ページでは、ローカルの証明書ストアの有効な証明書を選択するためのウィンドウが表示されます。証明書を選択して、[**証明書の詳細の表示**] をクリックします。証明書が新しいウィンドウに表示されます。[**証明書**] ウィンドウで、証明書の全般情報を表示できます。[**詳細**] タブを選択すると、サブジェクト名、サブジェクトの別名など、証明書の詳細情報が表示されます。[証明のパス] には、証明書の "信頼チェーン" のパスが表示されます。証明書情報の確認が完了したら、[**OK**] をクリックします。
