# <a name="setup-or-remove-skype-for-business-server-components-intro"></a>Skype for Business Server コンポーネントのセットアップまたは削除 (開始)
 
Skype for Business Server 2015 コンポーネントのインストール、ライセンス認証、または非アクティブ化の準備をしています。 実行されるプロセスは、中央管理ストアに公開されている定義の現在の構成によって定義されます。 タスクを正常に完了するには、ローカル管理者グループのメンバーとしてログインしており、少なくとも Active Directory ドメインサービスを読み取ることができる必要があります。
  
[**次へ**] をクリックして、[**コマンドを実行しています**] 画面に進みます。
