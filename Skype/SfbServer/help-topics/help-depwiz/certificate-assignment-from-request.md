# <a name="certificate-assignment-from-request"></a>証明書の割り当て (要求から)
 
[ **証明書の割り当** て] ページは、役割サーバーで要求された使用法に証明書を割り当てるプロセスを開始します。 [**証明書の詳細の表示**] をクリックして証明書を表示します。 [全般] タブ **、[** 詳細 **] タブ、** または [認定パス] タブ **を選択** できます。[ **全般]** タブには、発行元証明機関やプライベート キーの状態など、証明書に関する基本情報が表示されます。 [詳細] タブには、サブジェクト名、サブジェクトの別名など、証明書の詳細な情報が表示されます。 [ **証明書パス]** タブには、証明書から発行機関または中間証明機関 (存在する場合) からルート証明機関への証明書の "信頼チェーン" が表示されます。
  
[**次へ**] をクリックして続行します。
