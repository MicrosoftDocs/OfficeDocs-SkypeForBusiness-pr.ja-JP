# <a name="certificate-assignment-from-request"></a>証明書の割り当て (要求から)
 
[ **証明書の割り** 当て] ページでは、役割サーバーで要求された使用法に証明書を割り当てるプロセスを開始します。 [**証明書の詳細の表示**] をクリックして証明書を表示します。 [全般] タブ **、[** 詳細] タブ、または **[証明** のパス] タブ **を選択** できます。[ **全般** ] タブには、発行元の証明機関や、プライベート キーの状態など、証明書に関する基本情報が表示されます。 [詳細] タブには、サブジェクト名、サブジェクトの別名など、証明書の詳細な情報が表示されます。 [ **証明書のパス** ] タブには、証明書から、発行元または中間証明機関 (存在する場合) からルート証明機関への証明書 "信頼チェーン" が表示されます。
  
[**次へ**] をクリックして続行します。
