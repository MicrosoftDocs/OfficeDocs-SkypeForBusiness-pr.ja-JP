# <a name="certificate-request-geography"></a>証明書要求 (地理)
 
[**地理情報**] ページでは、証明書の要求に関する追加情報を要求します。 このページで、[**国**] を選択し、テキストボックスに**都道府県/州**と**市区町村**を入力します。 略語は使用しないことに注意してください。都道府県と市区町村/地域の完全な名前を指定する必要があります。
