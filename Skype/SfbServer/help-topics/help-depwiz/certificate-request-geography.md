# <a name="certificate-request-geography"></a>証明書要求 (地理)
 
[**地理情報**] ページでは、証明書要求についての追加情報の入力を要求されます。このページでは、[**国**] を選択し、テキスト ボックスに [**都道府県**] と [**市区町村**] を入力します。都道府県名と市区町村名は、略称を使用せず、完全な名称を入力してください。
