# <a name="certificate-assignment-summary"></a>証明書の割り当て (概要)
 
証明書割り当てが終了する前に、割り当てられる証明書の詳細を示した **[証明書割り当ての概要]** が表示されます。 情報を確認して、サブジェクト名とサブジェクトの代替名が正しいか確認します。 証明書の概要の確認が終了したら、**[次へ]** をクリックします。
