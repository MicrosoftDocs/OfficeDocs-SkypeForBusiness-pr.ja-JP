# <a name="certificate-assignment-summary"></a>証明書の割り当て (概要)
 
証明書の割り当てが完了する前に、割り当てられている証明書の詳細を一覧表示する**証明書の割り当ての概要**が表示されます。 情報を確認して、サブジェクト名と件名の代替名が正しいことを確認します。 証明書の概要を確認したら、[**次へ**] をクリックします。
