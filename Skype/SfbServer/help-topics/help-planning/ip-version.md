# <a name="ip-version-planning-tool"></a>IP バージョン (計画ツール)
 
Skype for Business Server 2015 で IP アドレスを構成する場合は、3 つのオプションがあります。 Skype for Business Server 2015 を構成して、IP バージョン 4 (IPv4)、IP バージョン 6 (IPv6)、または両方の組み合わせ (デュアル スタックと呼ばれる) のみをサポートできます。
  
IP インフラストラクチャの選択の詳細については [、「Plan for IPv6 in Skype for Business」を参照してください](../../plan-your-deployment/network-requirements/ipv6.md)。
