# <a name="skype-for-business-server-ip-version-planning-tool"></a>Skype for Business Server IP バージョン計画ツール
 
2015 年に IP アドレスを構成する場合は、3 Skype for Business Serverがあります。 2015 Skype for Business Server を構成して、IP バージョン 4 (IPv4)、IP バージョン 6 (IPv6) のみ、または両方の組み合わせ (デュアル スタックと呼ばれる) のみをサポートできます。
  
IP インフラストラクチャの選択の詳細については、「[Plan for IPv6 in Skype for Business」を参照してください](../../plan-your-deployment/network-requirements/ipv6.md)。
