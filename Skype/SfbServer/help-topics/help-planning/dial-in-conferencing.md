# <a name="dial-in-conferencing-planning-tool"></a>ダイヤルイン会議 (計画ツール)
 
Skype for Business Server 2015 ダイヤルイン会議を使用すると、サードパーティの電話会議プロバイダーを使用することなく、公衆交換電話網 (PSTN) 電話を使用して、Skype for Business Server 2015 会議の音声部分に参加できます。
  
組織でダイヤルイン会議を有効にする計画の詳細については、以下のトピックとリンク先のトピックを参照してください。 
  
- [Skype for Business Server 2015 での会議の計画](../../plan-your-deployment/conferencing/conferencing.md)
