# <a name="mobility-planning-tool"></a>Mobility (Planning Tool)
 
Skype for Business Server 2015 のモバイル機能は、モバイルデバイスでの Skype for Business の機能をサポートしています。
  
モバイル機能は、Lync Server 2013 と同じです。 モバイル環境の準備について詳しくは、「 [Lync Server 2013 でのモビリティの計画](https://technet.microsoft.com/en-us/library/hh689981%28v=ocs.15%29.aspx)」をご覧ください。
