# <a name="skype-for-business-server-mobility-planning-tool"></a>Skype for Business Server: モビリティ計画ツール
 
2015 年 2015 Skype for Business Serverモビリティ機能は、モバイル Skype for Business機能をサポートします。
  
モビリティ機能は、Lync Server 2013 と同じです。 モビリティのための環境の準備の詳細については [、「Lync Server 2013 での](/previous-versions/office/lync-server-2013/lync-server-2013-planning-for-mobility)モビリティの計画」を参照してください。
