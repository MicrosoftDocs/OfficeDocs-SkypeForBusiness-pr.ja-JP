# <a name="move-all-users"></a>すべてのユーザーの移動

[**ユーザーの移動**] ダイアログ ボックスでは、展開内のプール間でユーザー アカウントを移動する場合の、移動元プールと移動先プールを選択できます。

ユーザーの移動の詳細については、「[Move Users to Another Pool](/previous-versions/office/lync-server-2013/lync-server-2013-move-users-to-another-pool)」を参照してください。
