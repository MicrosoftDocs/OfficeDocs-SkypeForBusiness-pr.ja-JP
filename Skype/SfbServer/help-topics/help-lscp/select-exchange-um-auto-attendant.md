# <a name="select-exchange-um-auto-attendant"></a>Exchange UM 自動応答の選択
 
**[EXCHANGE Um 自動応答の選択**] ダイアログボックスを使って、未割り当ての番号の範囲への着信呼び出しを処理する Exchange UM 自動応答を選びます。
  
検索ボックスに、使用する Exchange UM 自動応答の名前のすべてまたは一部を入力します。 利用可能なすべての Exchange UM 自動応答を返すには、検索ボックスを空のままにして、検索アイコンをクリックします。
  
検索によって返された Exchange UM 自動応答の一覧で、割り当てられていない番号の範囲への着信通話を処理するために使用する自動応答を選択します。
