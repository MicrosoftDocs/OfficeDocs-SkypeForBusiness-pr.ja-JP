# <a name="select-exchange-um-auto-attendant"></a>Exchange UM 自動応答の選択
 
[**Exchange UM 自動応答の選択**] ダイアログ ボックスを使用して、	割り当てられていない番号の範囲への着信通話を処理する Exchange UM 自動応答を選択します。
  
検索ボックスに、使用する Exchange UM 自動応答の名前の一部またはすべてを入力します。使用可能な Exchange UM 自動応答をすべて返すには、検索ボックスを空白にしたままで検索アイコンをクリックします。
  
検索によって返される Exchange UM 自動応答の一覧で、割り当てられていない番号の範囲への着信通話を処理するのに使用する Exchange UM 自動応答を選択します。
