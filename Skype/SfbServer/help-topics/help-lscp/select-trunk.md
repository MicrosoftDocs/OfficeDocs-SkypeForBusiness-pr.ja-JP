# <a name="select-trunk"></a>トランクの選択
 
[**トランクの選択**] ダイアログ ボックスを使用して、使用可能な既存のトランクを検索します。
  
> [!NOTE]
> 使用可能なすべてのトランクを取得するには、検索ボックスに何も入力せずに検索アイコンをクリックします。 
  
Skype for Business Server コントロール パネルを使用して実行できるさまざまな手順の詳細については [、「Skype for Business Server 2015](../../manage/manage.md)の管理」を参照してください。
