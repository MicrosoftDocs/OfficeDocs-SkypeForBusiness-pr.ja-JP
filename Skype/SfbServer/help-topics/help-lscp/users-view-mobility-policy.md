# <a name="users-view-mobility-policy"></a>ユーザー: モビリティ ポリシーの表示

[**表示-モビリティポリシー** ] ダイアログに、選択したモバイルポリシーのプロパティが表示されます。

モバイル機能ポリシーの割り当ての詳細については、「[ユーザーごとのモビリティポリシーの割り当て](https://technet.microsoft.com/library/d8bf997f-4bc7-48d3-973b-323505f55e9d.aspx)と[モビリティポリシーの構成](https://technet.microsoft.com/library/595536e0-9bb3-49a3-8d13-1a77351ebc62.aspx)」を参照してください。
