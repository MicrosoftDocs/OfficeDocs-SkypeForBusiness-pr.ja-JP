# <a name="users-view-mobility-policy"></a>ユーザー: モビリティ ポリシーの表示

[**表示 - モビリティ ポリシー**] ダイアログには、選択中のモビリティ ポリシーのプロパティが表示されます。

モビリティ ポリシーの割り当ての詳細については、「[Assign a Per-User Mobility Policy](/previous-versions/office/lync-server-2013/lync-server-2013-assign-a-per-user-mobility-policy)」および「[Configuring Mobility Policy](/previous-versions/office/lync-server-2013/lync-server-2013-configuring-mobility-policy)」を参照してください。
