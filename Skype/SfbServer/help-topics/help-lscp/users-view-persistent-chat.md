# <a name="users-view-persistent-chat"></a>ユーザー: 常設チャットの表示

**CsPersistentChatAdministrator**ロール (永続的なチャット管理者) のメンバーには、他のユーザーにし、自らをチャット ルームの管理者の権限およびアクセス許可を与えることができます。

永続的なチャットの管理者は、次に行うことができます。

- 部屋のカテゴリとチャット ルームを作成します。

- すべてのカテゴリおよびチャット ルームのメンバーシップを設定します。

- すべてのカテゴリおよびチャット ルームのすべての設定を管理します。

- 有効にするか、永続的なチャット サーバーのポリシーを無効にします。

- 設定し、永続的なチャット サーバー プールの構成設定を管理します。

詳細については、展開に関するドキュメントの[永続的なチャット管理者を追加する](https://technet.microsoft.com/library/c107eb20-4e58-4463-b4f9-63fb5b1d9534.aspx)を参照してください。
