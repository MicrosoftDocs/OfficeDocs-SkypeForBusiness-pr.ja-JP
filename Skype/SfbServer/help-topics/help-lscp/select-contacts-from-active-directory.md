# <a name="select-contacts-from-active-directory"></a>Active Directory からの連絡先の選択
 
ビジネス用の Skype を有効にする連絡先を検索するのにには、 **Active Directory からの選択**] ダイアログ ボックスを使用します。 検索ボックスに、検索する連絡先の SIP アドレスまたは表示名を入力します。 使用可能な連絡先をすべて取得する場合は、検索ボックスに何も入力しないでください。
  
詳細ビジネス サーバーのコントロール パネルの Skype を使用して実行できるさまざまな手順については、[ビジネス サーバー 2015 の Skype の管理](../../manage/manage.md)を参照してください。
