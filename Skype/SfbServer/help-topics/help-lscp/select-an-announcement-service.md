# <a name="select-an-announcement-service"></a>アナウンス サービスの選択
 
**[アナウンスメントサービスの選択**] ダイアログボックスを使って、未割り当ての番号の範囲への着信通話を処理するアナウンスメントサービスを選びます。
  
検索ボックスに、使用するお知らせサービスの名前のすべてまたは一部を入力します。 利用可能なすべてのお知らせサービスを取得するには、検索ボックスを空のままにして、検索アイコンをクリックします。
  
検索によって返されるお知らせサービスの一覧で、未割り当て番号の範囲への着信通話の処理に使用するサービスを選択します。
