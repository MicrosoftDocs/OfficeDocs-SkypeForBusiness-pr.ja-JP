# <a name="select-an-announcement-service"></a>お知らせサービスの選択
 
未使用の番号範囲に対する着信を処理するアナウンス サービスを選択するには、[**アナウンス サービスの選択**] ダイアログ ボックスを使用します。
  
検索ボックスに、使用するアナウンス サービスの名前の一部またはすべてを入力します。使用可能なアナウンス サービスをすべて取得するには、検索ボックスを空白にしたままで検索アイコンをクリックします。
  
検索によって返されたアナウンス サービスの一覧で、未使用の番号範囲への着信の処理に使用するサービスを選択します。
