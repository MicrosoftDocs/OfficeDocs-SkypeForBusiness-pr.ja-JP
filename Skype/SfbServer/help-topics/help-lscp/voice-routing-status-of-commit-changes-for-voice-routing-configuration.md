# <a name="voice-routing-status-of-commit-changes-for-voice-routing-configuration"></a>音声ルーティング: 音声ルーティング構成のコミット変更の状態
 
変更が完了しなかった場合は、このダイアログ ボックスのメッセージを確認し、音声ルーティング構成を変更してから再試行してください。
  
コントロール パネルを使用して実行できるさまざまな手順の詳細については、「Skype for Business Server [2015」](../../manage/manage.md)を参照Skype for Business Serverしてください。
