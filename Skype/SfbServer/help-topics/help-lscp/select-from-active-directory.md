# <a name="select-from-active-directory"></a>Active Directory からの選択
 
[Active **Directory から選択] ダイアログ** ボックスを使用して、Skype for Business Server を有効にするユーザーを検索します。 検索ボックスに、検索するユーザーの SIP アドレスまたは表示名を入力します。 使用可能なユーザーすべてを取得する場合は、検索ボックスに何も入力しないでください。
  
Skype for Business Server コントロール パネルを使用して実行できるさまざまな手順の詳細については [、「Skype for Business Server 2015](../../manage/manage.md)の管理」を参照してください。
