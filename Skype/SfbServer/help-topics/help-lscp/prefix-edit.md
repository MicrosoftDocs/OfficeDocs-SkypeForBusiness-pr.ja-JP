# <a name="prefix-edit"></a>プレフィックスの編集
 
プレフィックスを定義するかを含め、このルートを処理する必要があります番号の接頭番号を追加、数字を開始して、必要な場合は + です。 たとえば、+123 で始まるすべての番号を照合することができます。
