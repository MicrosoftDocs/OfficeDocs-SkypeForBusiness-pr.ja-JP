# <a name="prefix-edit"></a>プレフィックスの編集
 
プレフィックスまたは開始桁の数字を定義するには、このルートで処理する必要がある番号プレフィックスを追加します (必要に応じて + を含めます)。たとえば、+123 で始まるすべての番号を一致させることができます。
  
コントロール パネルを使用して実行できるさまざまな手順の詳細については、「Skype for Business Server [2015」](../../manage/manage.md)を参照Skype for Business Serverしてください。
