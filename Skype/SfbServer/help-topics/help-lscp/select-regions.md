# <a name="select-regions"></a>地域の選択
 
[地域 **の選択] ダイアログ ボックス** を使用して、環境で使用可能な地域を検索します。 既存の地域を使用するには、一覧で地域をクリックして [**OK**] をクリックします。
  
> [!NOTE]
> 使用可能な地域すべてを取得するには、検索ボックスに何も入力せずに検索アイコンをクリックします。
