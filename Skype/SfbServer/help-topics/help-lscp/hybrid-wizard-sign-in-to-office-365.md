# <a name="hybrid-wizard-sign-in-to-microsoft-365-or-office-365"></a>ハイブリッド ウィザード: Microsoft 365 または Office 365 にサインインする

Skype for Business Server Hybrid Manageability ウィザードを使用して、Skype for Business Online を使用したハイブリッド環境の構成に役立ちます。

## <a name="sign-in-to-microsoft-365-or-office-365"></a>Microsoft 365 または Office 365 にサインインする

Microsoft 365 または Office 365 組織のユーザー名とパスワードを入力して、Microsoft 365 または Office 365 組織にオンプレミス展開にログインして接続します。 Microsoft 365 または Office 365 にサインアップするときに、onmicrosoft ドメイン用に作成された管理者アカウントを使用する必要があります (たとえば、admin@contoso.onmicrosoft.com)。

ハイブリッド用に環境を構成する方法の詳細については、「ハイブリッド展開の [概要」を参照してください](/previous-versions/office/lync-server-2013/lync-server-2013-overview-of-hybrid-deployments)。
