# <a name="save-voice-routing-test-case-information"></a>音声ルーティングのテスト ケース情報の保存
 
[**ボイスルーティングテストケース情報の保存**] ダイアログボックスを使用して、atemporary ボイスルーティングテストケースを保存できます。 [**ボイスルーティングテストケース情報を保存**する] で、このテストケースの名前を [**名前**] に入力し、[ **OK]** をクリックします。 
  
> [!NOTE]
> 一時的なテストケースを保存しても、全体管理ストアに保存されるわけではありません。 代わりに、[**ボイスルーティング**] ページの [**テストボイスルーティング**] タブからテストケースを利用できるようになります。 テストケースは、**未確定**の状態で表示されます。 テストケースを中央管理ストアに保存するには、[**音声ルーティング**] ページ内の任意の**アクション**メニューから [**すべてコミット**] をクリックして、音声構成をコミットする必要があります。
  
Skype for Business Server コントロールパネルを使用して実行できるさまざまな手順の詳細については、「Skype for business [server 2015 を管理](../../manage/manage.md)する」を参照してください。
