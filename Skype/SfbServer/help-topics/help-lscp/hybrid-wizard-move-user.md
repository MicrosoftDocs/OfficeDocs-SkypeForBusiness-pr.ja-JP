# <a name="hybrid-wizard-move-user"></a>ハイブリッド ウィザード: ユーザーの移動

Skype のオンライン ビジネスのホームに設置プールでユーザー アカウントがホーム サーバーを移動するのにには、ユーザーの移動ウィザードを使用します。 詳細については、「[Use the Hybrid Manageability wizard](https://technet.microsoft.com/library/d777f79b-a740-4aba-a9e2-c91f0315b6f8.aspx)」を参照してください。
