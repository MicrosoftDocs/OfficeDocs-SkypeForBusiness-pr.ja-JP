# <a name="hybrid-manageability-wizard"></a>ハイブリッド管理性ウィザード

ハイブリッド管理ウィザードを使用して、ハイブリッド展開をオンラインでセットアップするために必要な構成タスクの一部を実行Skype for Businessできます。 ハイブリッド用に展開を構成する方法の詳細については、「ハイブリッド管理ウィザードを使用して環境を構成 [する」を参照してください](https://technet.microsoft.com/library/d777f79b-a740-4aba-a9e2-c91f0315b6f8.aspx)。
