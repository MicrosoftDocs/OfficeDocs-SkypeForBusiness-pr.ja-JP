# <a name="web-conferencing-planning-tool"></a>Web Conferencing (Planning Tool)
 
Skype for Business Server web 会議には、ドキュメントのグループ作業、アプリケーション共有、デスクトップ共有が含まれます。
  
> [!NOTE]
> Skype for Business Server では、Office Web Apps と Office Web Apps サーバーを使って、PowerPoint プレゼンテーションの共有とレンダリングを処理します。 Web 会議機能は、Microsoft Lync 2013 とまったく同じように動作します。 Office Web Apps サーバーのインストールと構成の詳細については、「 [Office Web Apps サーバーおよび Lync server 2013 との統合を構成する](https://technet.microsoft.com/en-us/library/jj204792%28v=ocs.15%29.aspx)」を参照してください。
