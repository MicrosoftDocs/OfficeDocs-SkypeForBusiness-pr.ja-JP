# <a name="voice-infrastructure-planning-tool"></a>音声インフラストラクチャ (計画ツール)
 
Skype for Business Serverは、SIP トランクを使用したり、PSTN ゲートウェイに SIP 接続を直接接続したり、PBX への直接 SIP 接続を使用して、信頼性の高いスケーラブルな PSTN 接続を提供します。 詳細については、「Plan [for PSTN connectivity in Skype for Business Server」 を参照してください](../../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity-0.md)。
