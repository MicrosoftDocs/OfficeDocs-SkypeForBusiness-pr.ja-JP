# <a name="voice-infrastructure-planning-tool"></a>音声インフラストラクチャ (計画ツール)
 
Skype for Business Server は、SIP トランク、PSTN ゲートウェイへの直接 SIP 接続、または PBX への直接 SIP 接続を使用して、信頼性が高くスケーラブルな PSTN 接続を提供します。 詳細については [、「Skype for Business Server での PSTN 接続の計画」を参照してください](../../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity-0.md)。
