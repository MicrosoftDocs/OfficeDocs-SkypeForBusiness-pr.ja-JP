# <a name="monitoring-planning-tool"></a>(プランニング ・ ツール) の監視
 
ビジネス サーバーの Skype は、の監視は使用状況に関する情報と、ユーザーに関連する通信セッションに関するエクスペリエンスの品質 (QoE) データを収集するために使用されます。
  
サーバーの監視の計画の詳細については、[監視の計画](http://technet.microsoft.com/library/26cead5a-183c-42f1-a4b0-0e8d61c6159d.aspx)を参照してください。
