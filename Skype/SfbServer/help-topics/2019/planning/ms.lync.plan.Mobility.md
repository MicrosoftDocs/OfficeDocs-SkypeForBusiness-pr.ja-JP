# <a name="mobility-planning-tool"></a>モビリティ (計画ツール)
 
Skype for Business Server のモビリティ機能は、モバイル デバイス上の Skype for Business 機能をサポートします。
  
モビリティ機能は、Skype for Business Server の Lync Server 2013 と同じです。 モビリティのための環境の準備の詳細については [、「Lync Server 2013 での](/previous-versions/office/lync-server-2013/lync-server-2013-planning-for-mobility)モビリティの計画」を参照してください。
