# <a name="mobility-planning-tool"></a>移動 (計画ツール)
 
ビジネス サーバーの Skype のモビリティ機能では、モバイル デバイスでのビジネス機能のため Skype をサポートしています。
  
データ移動機能は、Lync Server 2013 のようにサーバーをビジネス用の Skype で同じです。 移動用に環境を準備する方法の詳細については、 [Lync Server 2013 での移動を計画する](https://technet.microsoft.com/en-us/library/hh689981%28v=ocs.15%29.aspx)を参照してください。
