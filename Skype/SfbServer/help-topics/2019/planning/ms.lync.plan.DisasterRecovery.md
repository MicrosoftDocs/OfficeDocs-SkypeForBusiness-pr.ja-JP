# <a name="disaster-recovery-planning-tool"></a>災害復旧 (計画ツール)
 
Skype ビジネス サーバーの障害を含むペアのフロント エンド プールを有効にすると回復手段は 2 つのデータ ・ センター内にあります。
  
災害復旧のため、環境を準備する方法の詳細は、[高可用性とビジネスのサーバー用の Skype での災害復旧の計画](../../../plan-your-deployment/high-availability-and-disaster-recovery/high-availability-and-disaster-recovery.md)を参照してください。
