# <a name="load-balancer-planning-tool"></a>ロード バランサー (計画ツール)
 
フロント エンド プール、ディレクター プール、またはエッジ サーバーのプールがあれば、これらのプールの負荷分散を展開する必要があります。 ロード バランシングによって、プール内のすべてのサーバーにトラフィックが分散されます。 詳細については、以下を参照してください。
  
- [負荷分散のビジネス用の Skype の要件](../../plan-your-deployment/network-requirements/load-balancing.md)
    
- [ビジネス サーバー 2015 の Skype の DNS の要件](../../plan-your-deployment/network-requirements/dns.md)
