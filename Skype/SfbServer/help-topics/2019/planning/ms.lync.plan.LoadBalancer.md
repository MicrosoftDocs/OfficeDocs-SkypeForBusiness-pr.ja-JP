# <a name="load-balancer-planning-tool"></a>ロード バランサー (計画ツール)
 
フロントエンド プール、ディレクター プール、またはエッジ サーバー プールがある場合は、これらのプールの負荷分散を展開する必要があります。 ロード バランシングによって、プール内のすべてのサーバーにトラフィックが分散されます。 詳細については、以下を参照してください。
