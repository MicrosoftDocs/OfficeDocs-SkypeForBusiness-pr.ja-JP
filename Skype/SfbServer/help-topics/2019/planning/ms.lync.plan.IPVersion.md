# <a name="ip-version-planning-tool"></a>IP Version (Planning Tool)
 
Skype for Business Server で IP アドレスを構成する際には、次の3つのオプションがあります。 Skype for Business Server では、IP バージョン 4 (IPv4)、IP バージョン 6 (IPv6)、またはその両方の組み合わせ (デュアルスタックとも呼ばれます) のみをサポートするように構成することができます。
  
IP インフラストラクチャの選択の詳細については、「 [Skype For business での IPv6 の計画](../../../plan-your-deployment/network-requirements/ipv6.md)」を参照してください。
