# <a name="move-all-users"></a>すべてのユーザーの移動

[**ユーザーの移動**] ダイアログ ボックスでは、展開内のプール間でユーザー アカウントを移動する場合の、移動元プールと移動先プールを選択できます。

ユーザーの移動の詳細については、「[Move Users to Another Pool](https://technet.microsoft.com/library/e7b4968c-0e9d-4d56-b5f1-9edf0f7206f8.aspx)」を参照してください。
