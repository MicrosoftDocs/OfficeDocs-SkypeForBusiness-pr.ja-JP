# <a name="select-trunk"></a>トランクの選択
 
[**トランクの選択**] ダイアログ ボックスを使用して、使用可能な既存のトランクを検索します。
  
> [!NOTE]
> 使用可能なすべてのトランクを取得するには、検索ボックスに何も入力せずに検索アイコンをクリックします。
