# <a name="select-trunk"></a>トランクの選択
 
**[トランクの選択**] ダイアログボックスを使用して、使用できる既存のトランクを検索します。
  
> [!NOTE]
> 使用可能なすべての trunks を取得するには、[検索] ボックスに何も入力せずに、[検索] アイコンをクリックします。
