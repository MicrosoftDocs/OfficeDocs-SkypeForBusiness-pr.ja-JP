# <a name="prefix-edit"></a>プレフィックスの編集
 
プレフィックスまたは開始番号を定義するには、必要に応じて、このルートが処理する番号のプレフィックスを追加します。 たとえば、+ 123 で始まるすべての番号に一致させることができます。
