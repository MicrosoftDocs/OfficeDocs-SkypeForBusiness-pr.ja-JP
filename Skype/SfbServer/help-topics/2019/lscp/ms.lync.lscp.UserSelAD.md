# <a name="select-from-active-directory"></a>Active Directory からの選択
 
[Active **Directory から選択] ダイアログ** ボックスを使用して、ユーザーに対して有効にするユーザーをSkype for Business Server。 検索ボックスに、検索するユーザーの SIP アドレスまたは表示名を入力します。 使用可能なユーザーすべてを取得する場合は、検索ボックスに何も入力しないでください。
