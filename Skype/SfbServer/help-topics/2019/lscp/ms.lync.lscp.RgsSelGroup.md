# <a name="select-groups"></a>グループを選択します。
 
応答グループ キューは、エージェントが呼び出しに応答するまでの応答グループへの呼び出しを保持します。 キューには、エージェント グループが割り当てられます。 作成または、キューを編集するときは、キューに割り当てる 1 つまたは複数のエージェント グループを選択するのに**グループの選択**] ダイアログ ボックスを使用します。
  
## <a name="ui-reference"></a>UI リファレンス

次の一覧は、コントロールと、[**グループの選択**] ダイアログ ボックス内のフィールドについて説明します。
  
- エージェント グループの名前の検索ボックスを検索します。 名前のすべてまたは一部を入力します。 検索ボックスは、組織のすべてのエージェント グループを表示するのには、空のままにします。 
    
- **名**返されたエージェント グループの名前を表示します。
    
- **参加ポリシー**エージェント グループに対して定義されている参加ポリシーが表示されます: 公式または非公式です。
    
- **ルーティングの方法**エージェント グループに対して定義されているルーティングの種類を表示します。
    
キューの操作についての詳細は、操作マニュアルの[応答グループ キューの管理](http://technet.microsoft.com/library/1e91720c-ab67-4dfb-b30c-0ef2a8012310.aspx)を参照してください。
