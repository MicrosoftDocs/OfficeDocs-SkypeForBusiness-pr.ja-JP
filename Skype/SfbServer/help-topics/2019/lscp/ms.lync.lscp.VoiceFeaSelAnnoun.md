# <a name="select-an-announcement-service"></a>アナウンス サービスの選択
 
**お知らせサービスを選択**] ダイアログ ボックスを使用すると、割り当てられていない番号の範囲への着信呼び出しを処理する、お知らせサービスを選択できます。
  
[検索] ボックスで、使用するお知らせサービスの名前のすべてまたは一部を入力します。 使用可能なすべてのお知らせサービスを返すには、[検索] ボックスを空白のままにして検索アイコンをクリックします。
  
検索によって返されるお知らせサービスの一覧で、割り当てられていない番号の範囲への着信呼び出しを処理するために使用するサービスを選択します。
