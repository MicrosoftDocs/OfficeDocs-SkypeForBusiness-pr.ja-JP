# <a name="select-time-out-queue"></a>タイムアウト キューの選択
 
呼び出しがタイムアウトしたときに転送されるキューを選択するのには、[**キューのタイムアウトを選択**] ダイアログ ボックスを使用します。
  
[検索] ボックスで、使用するキューの名前のすべてまたは一部を入力します。 使用可能なすべてのキューを取得するには、[検索] ボックスを空白のままし、検索アイコンをクリックします。
  
検索によって返されるキューの一覧では、タイムアウトの呼び出しに使用するキューを選択します。
