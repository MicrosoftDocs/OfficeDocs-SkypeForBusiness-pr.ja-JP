# <a name="select-hyperlink-prefix"></a>ハイパーリンクのプレフィックスの選択
 
**[ハイパーリンクプレフィックスの選択**] ダイアログボックスを使用して、ブロックするハイパーリンクの種類を選択するか、警告メッセージを送信するか、または (選択して警告メッセージをブロックまたは送信する)、または [**ハイパーリンクプレフィックス**] の下のカテゴリからの拡張機能のチェックボックスをオフ (許可) に設定します。 矢印をクリックしてカテゴリを展開し、そのカテゴリのすべてのハイパーリンクプレフィックスを表示します。 カテゴリのすべてのハイパーリンクのプレフィックスを選択する場合は、カテゴリ名を選択します。 
  
ハイパーリンクプレフィックスの一覧に表示されているハイパーリンクのプレフィックスが表示されない場合**は、[****リストに新しいハイパーリンクプレフィックスを追加**] の下のテキストボックスに拡張子を入力して、[**追加**] をクリックします。 
  
[選択した**ハイパーリンクのプレフィックス**] ボックスの一覧に目的のハイパーリンクのプレフィックスをすべて追加したら、[ **OK]** をクリックします。
