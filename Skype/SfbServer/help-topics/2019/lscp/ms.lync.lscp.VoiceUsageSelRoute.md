# <a name="select-routes"></a>ルートの選択
 
[ルート **の選択] ダイアログ ボックス** を使用して、使用できる既存のボイス ルートを検索します。
  
> [!NOTE]
> 使用可能なすべてのボイス ルートを取得するには、検索ボックスに何も入力しないで、検索アイコンをクリックします。
