# <a name="select-routes"></a>ルートの選択
 
[ルート **の選択] ダイアログ** ボックスを使用して、使用できる既存の音声ルートを検索します。
  
> [!NOTE]
> 使用可能なすべての音声ルートを取得するには、検索ボックスに何も入力しないし、検索アイコンをクリックします。
