# <a name="conferencing-policy-create-new-or-edit-existing"></a>会議ポリシー: 新規作成または現在の形式のままで編集

会議ポリシーにより、会議 (ミーティングとも呼ばれます) でユーザーが使用できる機能が定義されます。

## <a name="ui-reference"></a>UI リファレンス

次の一覧に、このページのフィールドを示します。

- **スコープ**作成または変更する会議ポリシーの範囲 (グローバル、サイト、またはユーザー) を識別します。

- **名前**各会議ポリシーには名前が必要です。 グローバルおよびサイトの会議のポリシーには既定で名前が付けられており、名前を変更することはできません。 ユーザー会議ポリシーの場合は、ユーザーまたはユーザーのグループを識別するわかりやすい名前を使用します。

    > [!NOTE]
    > この名前は、会議ポリシーの保存後には変更できません。

- **説明**このフィールドは省略可能です。 会議ポリシーに関する追加情報を提供する場合に使用します。

- **開催者ポリシー**このセクションの設定は、会議を開催するユーザーに適用されます。 設定が選択されている場合、ユーザーは指定された特性を持つ会議を整理できます。 設定が選択されていない場合、ユーザーはこの特性で会議を開催することはできません。 この設定では、ユーザーが他の会議の参加者としてアクセスできるかどうかは判断されません。

    セクションを開いたり閉じたりするには、ラベルの横にある上矢印および下矢印をクリックします。

- **会議の最大サイズ**会議で許可されるユーザーの最大数。 既定で、会議の最大サイズは 250 です。

- **参加者に匿名ユーザーの招待を許可する**このチェックボックスをオンにすると、ユーザーは匿名ユーザーを会議に招待することができます。 匿名ユーザーとは、組織の Active Directory ドメインサービスの資格情報を持っておらず、そのため、認証されていないユーザーのことです。

- **レコーディング**参加者が会議を記録できるかどうかを指定します。 **[なし**] または [**レコーディングを有効にする**] オプション。

- **フェデレーションおよび匿名の参加者が記録できるように**する外部および未認証の参加者が会議を記録することを許可するには、このチェックボックスをオンにします。

- **音声/ビデオ**参加者が音声とビデオを使用できるかどうかを指定します。

  - **なし**オーディオとビデオの使用を禁止するには、このオプションを選択します。

  - **IP オーディオを有効にする**このチェックボックスをオンにすると、オーディオは使用できますが、ビデオは使用できません。

  - **IP オーディオ/ビデオを有効にする**オーディオとビデオの両方を許可するには、このオプションを選択します。

- **PSTN ダイヤルイン会議を有効にする**[**音声/ビデオ**] で音声を有効にした場合、ユーザーが公衆交換電話網 (PSTN) を使って会議にダイヤルインできるようにするには、このチェックボックスをオンにします。

- **匿名の参加者にダイヤルアウトを許可する**ユーザーが会議にダイヤルインできるようにし、ダイヤルアウトものを使用して、認証されていない (匿名) ユーザーが会議に参加することを許可する場合は、このチェックボックスをオンにします。 会議サーバーはダイヤルアウト番号を使用してユーザーを呼び出し、ユーザーはその電話に応答し、会議に参加します。

- **参加者がエンタープライズ voip に対して有効になっていないダイヤルアウトを許可する**[**オーディオ/ビデオ**] で音声を有効にした場合、エンタープライズ voip が有効になっていないユーザーがダイヤルアウトものを使用して会議に参加できるようにするには、このチェックボックスをオンにします。 ダイヤルアウトものを使用すると、会議サーバーによってユーザーが電話を発信し、ユーザーは電話に応答して電話会議に参加します。

- **複数のビデオストリームを許可する**[**音声/ビデオ**] でビデオを有効にしている場合は、このチェックボックスをオンにすると、ユーザーはギャラリービューのビデオを使って会議を開催できます。 このチェックボックスをオンにすると、ユーザーが複数のビデオストリームを送信する会議を整理することができます。 このチェックボックスをオフにした場合、ユーザーは1つのビデオストリームを送信する会議のみを整理できます。

    > [!NOTE]
    > このオプションによって、会議でサポートされるビデオ ストリームの種類が決まります。参加者が複数のビデオ ストリームを受信できるかどうかは、このオプションで決まりません。[**複数のビデオ ストリームでの参加を参加者に許可する**] オプションによって、参加者が複数のビデオ ストリームを受信できるかどうかが決まります。

- **データの共同作業**会議でデータの共同作業を可能にするかどうかを指定します。 **[なし**] または [**データの共同作業を有効にする**] オプション。

    次の設定はデータのグループ作業に適用されます。

  - **フェデレーションおよび匿名の参加者がコンテンツをダウンロードできるように**するデータの共同作業を許可する場合は、このチェックボックスをオンにすると、外部および未認証のユーザーが、スライドや配布資料などのコンテンツを会議からダウンロードできるようになります。

  - **参加者にファイルの送信を許可する**データの共同作業を許可する場合は、このチェックボックスをオンにすると、会議中にすべての参加者にファイルを転送できます。

  - **注釈を有効にする**データの共同作業を許可する場合は、このチェックボックスをオンにして、会議中に共有されたコンテンツに対して参加者が画面上の注釈を設定できるようにします。

  - **PowerPoint の注釈を有効にする**注釈を許可する場合は、このチェックボックスをオンにすると、参加者は会議中に PowerPoint スライドのコメントを共有することができます。

  - **投票を有効にする**データの共同作業を許可する場合は、このチェックボックスをオンにすると、参加者は会議中に投票を保留できます。

- **アプリケーション共有**会議でアプリケーション共有を許可するかどうかを指定します。 [**アプリケーション共有を無効にする**] または [**アプリケーション共有を有効に**する] オプションがあります。

    次の設定はアプリケーション共有に適用されます。

  - **参加者に制御を許可する**アプリケーション共有を許可する場合は、このチェックボックスをオンにすると、参加者は会議中に共有されているアプリケーションやデスクトップの管理を行うことができます。

  - **フェデレーションおよび匿名の参加者による制御を許可する**アプリケーション共有を許可する場合は、このチェックボックスをオンにすると、外部および未認証の参加者が、会議中に共有されているアプリケーションまたはデスクトップの管理を行うことができます。

- **参加者ポリシー**このセクションの設定は、電話会議または2パーティセッションの参加者としてユーザーに適用されます。 次の設定はユーザーごとの設定であるため、会議または2パーティのセッションの1人のユーザーが、同じ電話会議または2パーティセッションの他のユーザーとは異なる機能を持つ場合があります。

    セクションを開いたり閉じたりするには、ラベルの横にある上矢印および下矢印をクリックします。

- [**アプリケーションとデスクトップの共有を有効にする**] を選択すると、会議または 2 者間セッションへの参加中にユーザーにアプリケーションまたはデスクトップの共有が許可されます。[**アプリケーションとデスクトップの共有を無効にする**] を選択すると、会議または 2 者間セッションへの参加中にユーザーにアプリケーションまたはデスクトップの共有が禁止されます。

- **ピアツーピアファイル送信を有効にする**このチェックボックスをオンにすると、会議または2パーティーのセッション中に、ユーザー間のファイル転送 (つまり、すべての参加者を伴わないファイル送信) が許可されます。

- **ピアツーピアの録音を有効にする**ユーザーが2パーティーのセッションを記録できるようにするには、このチェックボックスをオンにします。

- **複数のビデオストリームで参加者が参加できるよう**にするこのチェックボックスをオンにすると、参加者は、許可されている会議でギャラリーの表示ビデオを受信することができます。 このオプションが選択されていない場合、参加者は、会議で許可されている内容に関係なく、1つのビデオストリームのみを受け取ることができます。

    > [!NOTE]
    > [**複数のビデオ ストリームを許可する**] は、会議で複数のビデオ ストリームを許可するかどうかを指定します。

会議の機能の詳細については、「計画」のドキュメントの「[Overview of Conferencing](https://technet.microsoft.com/library/5bb90e69-3d4f-4d59-a1ee-2550de84439f.aspx)」を参照してください。会議ポリシーの使用の詳細については、「操作」のドキュメントの「[Conferencing Policies](https://technet.microsoft.com/library/8f92eb7c-ee66-4df6-a726-4bff93b122cb.aspx)」を参照してください。
