# <a name="save-voice-routing-test-case-information"></a>音声ルーティングのテスト ケース情報の保存
 
[音声ルーティング テスト ケース情報 **の** 保存] ダイアログ ボックスを使用して、一時音声ルーティング テスト ケースを保存できます。 [**音声ルーティング テスト ケース情報の保存**] の [**名前**] に、このテスト ケースに付ける名前を入力して [**OK**] をクリックします。 
  
> [!NOTE]
> 一時的なテスト ケースを保存しても、サーバーの全体管理ストアに保存されるのではありません。 代わりに、[**音声のルーティング**] ページの [**音声ルーティングのテスト**] タブからテスト ケースを使用できるようにします。 テスト ケースは、[**コミットされていません**] 状態で表示されます。 テスト ケースをサーバーの全体管理ストアに保存するには、[音声ルーティング] ページの[アクション]メニューの [すべてコミット] をクリックして音声構成 **をコミットする必要** があります。
