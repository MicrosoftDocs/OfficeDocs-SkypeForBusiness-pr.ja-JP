# <a name="select-exchange-um-auto-attendant"></a>Exchange UM の自動応答を選択します。
 
Exchange UM 自動応答未使用の番号の範囲への着信呼び出しを処理するを選択するのに**を選択して Exchange UM 自動応答**] ダイアログ ボックスを使用します。
  
[検索] ボックスで、Exchange UM 自動応答を使用するの名前のすべてまたは一部を入力します。 すべて利用可能な Exchange UM 自動応答を返すには、[検索] ボックスを空白のままにして検索アイコンをクリックします。
  
Exchange UM 自動アテンダントの検索によって返されるのリストで、割り当てられていない番号の範囲への着信呼び出しを処理するために使用する自動応答を選択します。
