# <a name="select-pstn-usage-record"></a>PSTN 使用法レコードの選択
 
**[Pstn 使用状況レコードの選択**] ダイアログボックスを使用して、既存の公衆交換電話網 (PSTN) 使用状況レコードを検索します。
  
> [!NOTE]
> 利用可能なすべての PSTN 利用状況レコードを取得するには、[検索] ボックスに何も入力せずに、[検索] アイコンをクリックします。
