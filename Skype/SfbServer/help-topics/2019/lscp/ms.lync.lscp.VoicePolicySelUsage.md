# <a name="select-pstn-usage-record"></a>PSTN 使用法レコードの選択
 
[**PSTN 使用法レコードの選択**] ダイアログ ボックスを使用して、使用可能な既存の公衆交換電話網 (PSTN) 使用法レコードを検索します。
  
> [!NOTE]
> 使用可能な PSTN 使用法レコードすべてを取得するには、検索ボックスに何も入力せずに検索アイコンをクリックします。
