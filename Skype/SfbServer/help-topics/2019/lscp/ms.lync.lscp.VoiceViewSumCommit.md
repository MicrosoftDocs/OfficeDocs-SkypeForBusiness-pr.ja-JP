# <a name="uncommitted-voice-configuration-settings-commit"></a>コミットされていない音声構成設定: コミット
 
[コミット **されていない音声構成**] ダイアログ 設定を使用して、行ったすべての変更を確認できます。 ダイアログ ボックス内をスクロールして未確定の変更をすべて確認するか、矢印をクリックして別のセクションを展開または折りたたみます。 終了したら、[**確定**] をクリックしてそれらの変更をすべて確定するか、[**キャンセル**] をクリックして戻り、変更点を修正します。
  
> [!IMPORTANT]
> [**確定**] をクリックすると、[**未確定の音声構成設定**] ダイアログ ボックスの一覧にある **すべての** 変更が確定されます。
