# <a name="location-policy-create-new-or-edit-existing"></a>場所のポリシー: 新規作成または現在の形式のままで編集

場所ポリシーを構成すると、拡張 9-1-1 (E9-1-1) を有効にするかどうか、E9-1-1 の使用方法、およびユーザーと連絡先での場所情報の使用方法を指定することができます。
