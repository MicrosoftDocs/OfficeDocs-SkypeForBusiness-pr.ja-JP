# <a name="select-normalization-rules"></a>正規化ルールの選択
 
正規化ルールでは、.NET Frameworkを使用して、ダイヤルされた電話番号を E.164 形式に変換します。 各ダイヤル プランには、正規化ルールを 1 つ以上割り当てる必要があります。
  
既存の正規化ルールを使用するには、一覧でルールをクリックして [**OK**] をクリックします。
