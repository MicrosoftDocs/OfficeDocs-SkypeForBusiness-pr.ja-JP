# <a name="pin-set-successfully"></a>PIN 設定の成功
 
メールを使ってユーザーに PIN を送信するには、[**電子メールアプリケーションを開く] をクリックして、新しい pin をユーザーに送信**します。 PIN をメールメッセージにコピーするには、[ **pin の表示**] を選択します。
