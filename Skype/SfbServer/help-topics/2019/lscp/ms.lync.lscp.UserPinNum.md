# <a name="pin-set-successfully"></a>PIN 設定の成功
 
電子メールを使用してユーザーに PIN を送信するには、[**電子メール アプリケーションを開いて新しい PIN をユーザーに送信する**] をクリックします。PIN を電子メール メッセージにコピーするには、[**PIN の表示**] を選択します。
