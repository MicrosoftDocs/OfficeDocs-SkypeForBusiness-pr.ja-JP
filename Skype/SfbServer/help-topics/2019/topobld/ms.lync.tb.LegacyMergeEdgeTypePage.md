# <a name="legacy-merge-edge-type"></a>従来版のマージ エッジの種類
 
**[単一エッジ サーバー]** または **[負荷分散された統合エッジ]** サーバーの場合、**[次へ]** をクリックして構成情報を指定します。 拡張エッジ **では、** 各サーバーの構成を統合エッジ構成に変更してからマージする必要があります。
