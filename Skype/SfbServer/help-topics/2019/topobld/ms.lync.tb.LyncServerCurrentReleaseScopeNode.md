# <a name="skype-for-business-server-create-new-expander"></a>Skype for Business Server の新しい展開の作成
 
トポロジ ビルダーで新しいサーバーの役割を作成するには **、Skype for Business Server** を右クリックし、[新規] を選択して、作成するサーバーの役割を選択します。
