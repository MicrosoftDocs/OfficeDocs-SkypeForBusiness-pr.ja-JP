# <a name="skype-for-business-server-create-new-expander"></a>Skype for Business Server の新しい展開コントロールの作成
 
トポロジビルダーで新しいサーバーロールを作成するには、[ **Skype For Business server**] を右クリックし、[**新規**] を選択して、作成するサーバーの役割を選びます。
