# <a name="add-vis-fqdn"></a>VIS の FQDN の追加
 
トポロジ ビルダーで新しい VIS またはビデオ ゲートウェイを定義するために必要な手順は「オンライン[ビジネス サーバーの Skype で VIS プールを作成します](../../../deploy/deploy-video-interop-server/create-a-vis-pool.md)。
  
[ビジネス サーバーの Skype で VIS のサーバーの役割の展開](../../../deploy/deploy-video-interop-server/deploy-the-vis-server-role.md)でオンライン Windows PowerShell を使用して、サーバー上のビデオの相互運用機能のサーバーの役割を割り当てるために必要な手順を説明します。
  
[ビジネス サーバーの Skype でビデオの相互運用機能のサーバーの構成](../../../deploy/deploy-video-interop-server/configure-the-vis.md)にオンライン、Windows PowerShell を使用して、新しいビデオの相互運用機能サーバーの構成手順を説明します。
  
 [CUCM ビジネス サーバーの Skype での相互運用のために構成](../../../deploy/deploy-video-interop-server/configure-cucm-for-interoperation.md)にオンライン、VIS と相互運用のための Cisco GUI で設定する必要があるビデオ ゲートウェイの設定を説明します。
  
 [Skype ビジネス サーバーとの相互運用のため、VTC の構成](../../../deploy/deploy-video-interop-server/configure-a-vtc-for-interoperation.md)でオンライン、ビデオ ゲートウェイ、VIS との相互運用のための Cisco の GUI で設定する必要がある VTC の設定を説明します。
