# <a name="legacy-merge-results"></a>従来版のマージの結果

従来の展開と既存の展開の統合を開始するには、[Skype for Business Server] を **クリックします**。 トポロジ ビルダー結合ウィザードの実行の詳細については、「移行」のドキュメントの「[Merge Using Topology Builder Merge Wizard](/previous-versions/office/lync-server-2013/merge-using-topology-builder-merge-wizard)」を参照してください。
