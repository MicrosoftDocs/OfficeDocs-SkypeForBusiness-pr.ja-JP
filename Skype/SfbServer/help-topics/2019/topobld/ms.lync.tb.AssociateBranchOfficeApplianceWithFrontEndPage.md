# <a name="associate-branch-office-appliance-with-front-end"></a>ブランチ オフィス アプライアンスをフロント エンドと関連付ける
 
存続可能ブランチ アプライアンスまたは存続可能ブランチ サーバーをフロントエンド プールまたは Standard Edition サーバーに関連付けるには、[フロントエンド プールの選択] ページで、フロントエンド プールまたは Standard Edition サーバーをフロントエンド プールドロップダウン リストから選択します。 [**次へ**] をクリックして続行します。
