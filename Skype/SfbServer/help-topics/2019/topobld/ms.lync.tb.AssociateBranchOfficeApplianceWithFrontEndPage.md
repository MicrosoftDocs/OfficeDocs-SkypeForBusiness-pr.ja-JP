# <a name="associate-branch-office-appliance-with-front-end"></a>ブランチ オフィス アプライアンスとフロントエンドの関連付け
 
Survivable Branch Appliance または Survivable Branch Server をフロントエンドプールまたは Standard Edition サーバーに関連付けるには、 **[フロント**エンドプールの選択] ページで、 **[フロントエンドプール] ボックス**の一覧から [フロントエンドプール] または [standard edition サーバー] を選びます。 [**次へ**] をクリックして続行します。
