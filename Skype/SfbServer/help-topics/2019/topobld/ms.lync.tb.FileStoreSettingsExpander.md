# <a name="file-store-settings-expander"></a>ファイル ストアの設定の拡張
 
定義済みのファイル ストアのプロパティを編集します。
  
ファイル共有の名前の新しい共有の名前を定義できます。 ファイル サーバーの完全修飾ドメイン名 (FQDN) のプロパティで指定されたサーバー上の共有を作成する必要があります。
  
> [!NOTE]
> ファイル サーバーの FQDN プロパティで指定されているサーバーの FQDN を変更することはできません。 別のサーバー上のファイル ストアのファイル共有を配置する場合は、新しいファイル ストアのエントリを作成します。
