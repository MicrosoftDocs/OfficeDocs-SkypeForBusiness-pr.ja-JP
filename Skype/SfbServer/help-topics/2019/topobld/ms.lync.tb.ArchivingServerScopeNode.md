# <a name="archiving-server-scope-node"></a>アーカイブ サーバーのスコープ ノード
 
トポロジ ビルダーでは、コンソール ツリーでプールを右クリックし、[プロパティの編集] を選択して、アーカイブを実行しているサーバーに関連するタスクを **実行できます**。 [**プロパティの編集**] ページで、アーカイブを実行しているサーバーの完全修飾ドメイン名 (FQDN)、アーカイブ SQL Server データベース、およびアーカイブ ファイル ストアを変更できます。
