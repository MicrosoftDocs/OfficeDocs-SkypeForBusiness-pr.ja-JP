トポロジ ビルダーでは、コンソール ツリーで、プールを右クリックし、アーカイブを実行するサーバーに関連するタスクを実行することができ、**プロパティの編集**] を選択します。 [**プロパティの編集**] ページで、アーカイブを実行しているサーバーの完全修飾ドメイン名 (FQDN)、アーカイブ SQL Server データベース、およびアーカイブ ファイル ストアを変更できます。
