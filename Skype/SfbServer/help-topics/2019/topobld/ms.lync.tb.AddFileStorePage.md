# <a name="add-file-store"></a>ファイル ストアを追加します。
 
新しいファイル ストアを定義するには、次のように指定します。
  
- 新しいファイル ストアをホストするサーバーの完全修飾ドメイン名 (FQDN) です。 サーバーの FQDN は、トポロジを公開する前に、ドメイン ネーム システム (DNS) 内にすることがあります。
    
- ホスト サーバー上のファイル共有の名前です。 トポロジを公開する前に、ファイル共有が存在する必要があります。
