# <a name="legacy-merge-next-hop"></a>従来版のマージの次ホップ
 
**次ホップの FQDN**は、従来のエッジ サーバーが内部トラフィックをルーティングする次ホップのサーバーを識別します。 このエッジ サーバーが内部トラフィックをルーティングする次ホップのサーバーの完全修飾ドメイン名 (FQDN) を入力し、**フロント エンド プール**または**ディレクター**であるかどうかを選択します。
