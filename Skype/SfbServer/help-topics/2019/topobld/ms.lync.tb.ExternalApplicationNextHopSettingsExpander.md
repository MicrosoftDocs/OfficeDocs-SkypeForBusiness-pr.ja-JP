# <a name="external-application-next-hop-settings-expander"></a>外部アプリケーションの次ホップ設定エキスパンダー
 
ドロップダウン リストから定義済みの Enterprise Edition フロントエンド プールまたは Standard Edition フロントエンド サーバーを選択して、信頼済みアプリケーション サーバー プールの次ホップ サーバーを指定できます。 ディレクターまたはディレクター プールは、信頼済みアプリケーション サーバーの次ホップとしては有効な選択肢ではないため、リストに表示されません。 **[OK] を** クリックして変更を受け入れて保存します。 変更を破棄してプロパティ ページを終了するには、**[キャンセル]** をクリックします。
