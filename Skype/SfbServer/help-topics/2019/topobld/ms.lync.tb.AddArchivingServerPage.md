# <a name="add-archiving-server"></a>アーカイブ サーバーの追加
 
アーカイブ サーバーをトポロジ設計に展開するには、アーカイブ サーバーを展開するサーバーの完全修飾ドメイン名 (FQDN) を指定する必要があります。 Skype for Business Server のアーカイブとインストールを含むトポロジを公開する前に、アーカイブを展開するためのすべての前提条件を完了しておく必要があります。
