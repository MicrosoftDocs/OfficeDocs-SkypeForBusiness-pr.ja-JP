# <a name="add-archiving-server"></a>アーカイブ サーバーの追加
 
アーカイブ サーバーをトポロジ設計に展開するには、アーカイブ サーバーを展開するサーバーの完全修飾ドメイン名 (FQDN) を指定する必要があります。 アーカイブとインストールを含むトポロジを公開する前に、Skype for Business Server展開のすべての前提条件を完了している必要があります。
