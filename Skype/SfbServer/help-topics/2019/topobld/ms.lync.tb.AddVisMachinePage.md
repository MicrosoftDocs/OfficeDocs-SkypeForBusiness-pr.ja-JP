Windows PowerShell を使用して、サーバー上のビデオの相互運用機能のサーバーの役割を割り当てるために必要な手順をオンライン[ビジネス サーバー 2015 の Skype で VIS のサーバーの役割の展開](../../deploy/deploy-video-interop-server/deploy-the-vis-server-role.md)で説明します。
