# <a name="add-director-features"></a>ディレクターの機能の追加

ディレクター プールの監視サーバーを追加する場合は、**監視 (CDR および QoE 指標)** を選択します。

ディレクター プールを追加する方法の詳細については、[ディレクターのシナリオ](https://technet.microsoft.com/library/d2cf384a-0860-4779-80ce-cba2543be322.aspx)と[、ディレクターの設定](https://technet.microsoft.com/library/408b76f7-6fdd-4e50-8a3e-e87db12c1394.aspx)を参照してください。
