# <a name="monitoring-scope-node"></a>監視のスコープ ノード
 
トポロジ ビルダーでは、監視を実行している個々のサーバーに関連するタスクを実行するには、コンソール ツリーで監視を実行しているサーバーを右クリックし、ツールバーの [アクション] をクリックするか、[操作] ウィンドウでタスクをクリックし、監視を実行しているサーバーの次のいずれかのオプションをクリックします。
  
- **プロパティの編集**。 監視を **実行している** サーバーの [プロパティの編集] ページを開き、監視データベースを実行しているサーバーの完全修飾ドメイン名 (FQDN) をSQL Serverします。
    
- **Delete** 監視を実行しているサーバーをトポロジと監視データベースからSQL Serverします。
    
このトピックで既に述べた、監視を実行しているサーバーのオプションに加えて、監視ノードからは、コンソール ツリーの他のノードからも使用可能な全般的なオプションを選択できます。これには、次のオプションが含まれます。
  
- **トポロジ。** このオプションを使用すると、新しいトポロジの定義、既存のトポロジの開き、現在のトポロジのダウンロード、トポロジのコピーの保存、トポロジの発行、データベースのインストール、展開の削除を行います。
    
- **ヘルプ**。このオプションを使用して、アクティブ ノードのヘルプにアクセスできます。
