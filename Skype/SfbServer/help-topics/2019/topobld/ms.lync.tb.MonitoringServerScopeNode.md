# <a name="monitoring-scope-node"></a>監視のスコープ ノード
 
トポロジ ビルダーでは、コンソール ツリーで監視を実行しているサーバーを右クリックするか、ツール バーの [アクション] をクリックするか、[操作] ウィンドウでタスクをクリックして、監視を実行しているサーバーに対して次のいずれかのオプションをクリックすることで、監視を実行している個々のサーバーに関連するタスクを実行できます。
  
- **プロパティの編集**。 監視を **実行** しているサーバーの [プロパティの編集] ページを開きます。このページでは、監視データベースまたは監視データベースを実行しているサーバーの完全修飾ドメイン名 (FQDN) SQL Serverできます。
    
- **Delete** 監視を実行しているサーバーをトポロジおよび監視データベースからSQL Serverします。
    
このトピックで既に述べた、監視を実行しているサーバーのオプションに加えて、監視ノードからは、コンソール ツリーの他のノードからも使用可能な全般的なオプションを選択できます。これには、次のオプションが含まれます。
  
- **トポロジ 。** このオプションを使用して、新しいトポロジの定義、既存のトポロジを開く、現在のトポロジのダウンロード、トポロジのコピーの保存、トポロジの公開、データベースのインストール、または展開の削除を行います。
    
- **ヘルプ**。このオプションを使用して、アクティブ ノードのヘルプにアクセスできます。
