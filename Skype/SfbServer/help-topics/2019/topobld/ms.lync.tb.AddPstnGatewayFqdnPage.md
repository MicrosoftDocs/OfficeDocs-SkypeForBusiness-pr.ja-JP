# <a name="define-the-pstn-gateway-fqdn"></a>PSTN ゲートウェイの FQDN の定義

公衆交換電話網 (PSTN) ゲートウェイは、PSTN または構内交換 (機 PBX) には、エンタープライズ VoIP コンポーネントを接続するのには、完全修飾ドメイン名 (FQDN) を指定します。

詳細については、 [Skype のビジネス サーバーの PSTN 接続のコンポーネント](../../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity.md)を参照してください。
