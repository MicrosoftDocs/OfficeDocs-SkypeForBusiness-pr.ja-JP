# <a name="define-the-pstn-gateway-fqdn"></a>PSTN ゲートウェイ FQDN の定義

エンタープライズ VoIP コンポーネントを PSTN または PBX (Private Branch Exchange) に接続する公衆交換電話網 (PSTN) ゲートウェイの完全修飾ドメイン名 (FQDN) を指定します。

詳細については、Skype for Business Server の PSTN 接続 [コンポーネントを参照してください。](../../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity.md)
