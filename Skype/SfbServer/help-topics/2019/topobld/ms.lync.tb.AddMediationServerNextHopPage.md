# <a name="add-mediation-server-nexthop"></a>仲介サーバーの次ホップの追加
 
定義する仲介サーバーの次のホップを定義するのには、Enterprise Edition サーバーまたは Standard Edition サーバー、仲介サーバーの次ホップとして機能する**次ホップ プール**のドロップ ダウン リストからを選択します。 [**次へ**] をクリックして続行します。
