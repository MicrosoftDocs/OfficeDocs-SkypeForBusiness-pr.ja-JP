# <a name="add-mediation-server-nexthop"></a>仲介サーバーの次ホップの追加
 
定義する仲介サーバーの次ホップを定義するには、[次ホップ プール] ドロップダウン リストから、仲介サーバーの次ホップとして機能する Enterprise Edition サーバーまたは Standard Edition サーバーを選択します。 [**次へ**] をクリックして続行します。
