# <a name="add-nexthop-trusted-application-server"></a>次ホップ信頼済みアプリケーション サーバーの追加
 
信頼されたアプリケーション サーバーまたはプールの次ホップの宛先を設定するには、次の操作を行います。
  
**次ホップ プール**ボックスの一覧で Enterprise Edition プールまたは標準エディション フロント エンド サーバーとしては、次のホップを使用する信頼されたアプリケーション プールを選択します。
  
信頼されたアプリケーション プールの作成を完了する**完了**をクリックします。
