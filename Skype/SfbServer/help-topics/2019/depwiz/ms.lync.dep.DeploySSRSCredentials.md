# <a name="sql-server-reporting-services-credentials"></a>SQL Server Reporting Services (資格情報)
 
監視データベースにアクセスするための SQL Server Reporting Services (SSRS) の資格情報を指定するには、次の操作を行います。
  
[**資格情報の指定**] ページで、監視データベース (QoEMetrics および LcsCDR) にアクセスするために使用するユーザー資格情報を指定します。 
  
> [!NOTE]
> 詳細については、次の「レポート データ ソース (SSRS) の資格情報と接続情報を指定する」を参照してください [https://go.microsoft.com/fwlink/p/?LinkId=263431](https://go.microsoft.com/fwlink/p/?LinkId=263431) 。 
  
監視を実行しているサーバーの Reporting Services レポートの展開の詳細については、「Skype for Business Server での監視レポートの [インストール」を参照してください](../../../deploy/deploy-monitoring/install-monitoring-reports.md)。
