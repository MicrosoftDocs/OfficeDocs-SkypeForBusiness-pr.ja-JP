# <a name="view-certificate"></a>証明書の表示
 
[**証明書の表示**] ページには、ローカル証明書ストアで有効な証明書を選択できるウィンドウがあります。 証明書を選択し、[**証明書の詳細の表示**] をクリックします。 証明書が新しいウィンドウに表示されます。 [**証明書**] ウィンドウで、一般的な証明書情報を表示できます。 [**詳細**] タブをクリックして、証明書に関する詳細情報を表示します。たとえば、サブジェクト名やサブジェクトの代替名などです。 証明のパスには、証明書の "信頼チェーン" のパスが表示されます。 証明書情報の確認が完了したら、[ **OK]** をクリックします。
