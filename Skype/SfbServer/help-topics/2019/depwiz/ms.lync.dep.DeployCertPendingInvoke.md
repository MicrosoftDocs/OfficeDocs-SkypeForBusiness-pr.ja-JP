# <a name="pending-certificate-invoke"></a>保留中の証明書 (起動)
 
**[実行コマンド]** ページには、保留中の証明書を取得するコマンドの状態が表示されます。タスクが完了すると、**[保留中の証明書の取得**] ログ ファイルが表示可能になります。**[ログの表示]** をクリックして、処理およびタスクの完了状態を確認します。**[次へ]** をクリックして続行します。
  
> [!IMPORTANT]
> 証明書がまだ証明機関の管理者によって承認されていない場合は、[保留中の要求の取得] ログ ファイルに保留中の要求があることが示されます。証明機関の管理者が要求を拒否した場合は、障害があることがログ ファイルに示され、要求が拒否され、要求を拒否した管理者のユーザー名を示したエラーが報告されます。
