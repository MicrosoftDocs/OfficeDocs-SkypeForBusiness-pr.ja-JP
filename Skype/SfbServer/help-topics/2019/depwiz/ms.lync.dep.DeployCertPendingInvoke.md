# <a name="pending-certificate-invoke"></a>保留中の証明書 (起動)
 
[**コマンドの実行**] ページには、保留中の証明書を取得しているコマンドの状態が表示されます。 タスクが完了すると、[**保留中の証明書の取得**] ログファイルが利用できるようになります。 [**ログの表示**] をクリックして、処理とタスクの完了状態を確認します。 [**次へ**] をクリックして続行します。
  
> [!IMPORTANT]
> 証明書が証明機関の管理者によって承認されていない場合、[保留中の要求の取得] ログファイルには、保留中の要求があることが示されます。 証明機関の管理者が要求を拒否した場合、ログファイルにエラーが表示され、要求が拒否されたことを示すエラーと、要求を拒否したユーザーの名前が報告されます。
