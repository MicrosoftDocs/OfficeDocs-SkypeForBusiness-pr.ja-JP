# <a name="setup-or-remove-skype-for-business-server-components-intro"></a>Skype for Business Server コンポーネントのセットアップまたは削除 (開始)
 
Skype for Business Server コンポーネントをインストールおよびアクティブ化または非アクティブ化してアンインストールする準備をしています。 実行されるプロセスは、中央管理ストアに公開されている定義の現在の構成によって定義されます。 タスクを正常に完了するには、ローカルの Administrators グループのメンバーとしてログインし、少なくとも Active Directory ドメイン サービスを読み取る必要があります。
  
**[次へ]** をクリックして、**[コマンドの実行]** 画面に進みます。
