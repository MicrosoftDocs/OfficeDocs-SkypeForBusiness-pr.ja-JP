# <a name="certificate-request-certificate-signing-request-generated"></a>証明書要求 (生成された証明書の署名要求)
 
証明書 **のオフライン要求** が完了すると、[証明書要求ファイル] ページが表示されます。 証明書要求を表示し、それをオフライン証明機関 (CA) またはパブリック CA に送信する準備をするには、[表示] ボタン **をクリック** します。 証明書要求は 、既定でWindows メモ帳に表示されます。 この画面からエンコードされた証明書要求をコピーして貼り付けるか、選択したオフライン CA またはパブリック CA からの指示に従って証明書要求の処理を完了できます。 [完了 **] をクリック** して続行します。
