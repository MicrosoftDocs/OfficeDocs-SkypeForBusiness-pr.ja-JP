# <a name="certificate-request-certificate-signing-request"></a>証明書要求 (証明書の署名要求)
 
オフライン要求を作成するには、まず証明書署名要求 (CSR) を作成する必要があります。 [**ファイル名**] テキストボックスに、CSR のパスとファイル名を入力します。 [**参照**] をクリックして、CSR を作成する場所を選択できます。 [**次へ**] をクリックして続行します。
  
前の画面に戻るには、[**戻る**] をクリックします。 システムを変更せずに終了するには、[**キャンセル**] をクリックします。
  
> [!IMPORTANT]
> この手順で作成したファイルは、公共の証明機関 (CA)、内部のオフライン CA、または CA 管理者に送信されて処理されます。 ファイルの場所と名前をメモしておくと、証明書を取得して CA に提出することができます。 既定の場所は、証明書ウィザードを実行しているユーザーのドキュメントディレクトリです。
