オフライン要求を作成するには、まず証明書署名要求 (CSR) を作成する必要があります。 **ファイル名**] テキスト ボックスでは、CSR のパスとファイル名を入力します。 CSR を作成する場所を選択する**参照**ををクリックすることができます。 [**次へ**] をクリックして続行します。
  
**戻る**前の画面に戻る] をクリックします。 システムを変更せずに終了する**キャンセル**をクリックします。
  
> [!IMPORTANT]
> この手順で作成されるファイルは、パブリック証明機関 (CA)、内部のオフライン CA に送信または処理のため、CA の管理者に送信します。 それを取得して、CSR を CA に送信できるようにファイルの場所と名前に注意してください。 既定の場所は、証明書ウィザードを実行するユーザーのドキュメント ディレクトリです。
