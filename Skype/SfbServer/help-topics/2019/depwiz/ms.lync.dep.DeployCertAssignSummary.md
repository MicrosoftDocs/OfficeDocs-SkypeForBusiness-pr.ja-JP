# <a name="certificate-assignment-summary"></a>証明書の割り当て (概要)
 
証明書の割り当てが完了したら、前に割り当てられる証明書の詳細が一覧表示される**証明書の割り当てのサマリー**が表示されます。 サブジェクト名とサブジェクト代替名が正しいことを確認する情報を確認します。 証明書の概要を確認したら、「**次へ**」をクリックしてします。
