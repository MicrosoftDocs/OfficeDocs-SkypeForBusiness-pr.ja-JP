# <a name="startservices-intro"></a>サービスの開始 (開始)
 
展開している Skype for Business Server の役割サーバーでサービスを開始するには、[はじめに] 画面を確認します。 Skype for Business サーバーを起動するために発行されたコマンドが、サービスの開始を待機していないことに注意してください。 これは、サービスを開始するための最適な方法であり、Windows Services Microsoft 管理コンソール (MMC) を使用してサービスの実際の状態を監視する必要があります。 [**次へ**] をクリックして [**サービスの開始**] タスクを開始します。
