# <a name="startservices-intro"></a>サービスの開始 (開始)
 
ビジネスのサーバーの役割を持つサーバーを展開することで Skype のサービスを開始するには、最初の画面を確認します。 ビジネス サーバーは、Skype を起動して実行されたコマンドは待たない、サービスを起動することを確認します。 これは、サービスを開始するための最適な方法であり、Windows Services Microsoft 管理コンソール (MMC) を使用してサービスの実際の状態を監視する必要があります。 [**次へ**] をクリックして [**サービスの開始**] タスクを開始します。
