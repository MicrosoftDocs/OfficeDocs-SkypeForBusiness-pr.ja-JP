# <a name="prepare-current-forest-universal-group-location"></a>現在のフォレストの準備 (ユニバーサル グループの場所)
 
フォレストでユニバーサルグループを作成する必要がある場所を指定する必要があります。 [**ユニバーサルグループの場所**] ページでは、ログオンしているコンピューターが既定としてのメンバーであるドメインを使用します。 既定の**ローカルドメイン**以外のドメインでユニバーサルグループを作成する場合は、[**ドメインの FQDN** ] を選択し、テキストボックスに完全修飾ドメイン名 (FQDN) を入力します。 代替ドメインを指定することは、ユニバーサルグループを作成する場所以外のドメインからフォレストの準備タスクを実行しているときに役立ちます。 [**次へ**] をクリックして続行します。
