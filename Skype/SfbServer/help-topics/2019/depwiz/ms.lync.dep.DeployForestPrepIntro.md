# <a name="prepare-current-forrest-intro"></a>現在のフォレストの準備 (開始)
 
[**フォレストの準備**] タスクを開始するには、[**次へ**] をクリックします。 **フォレストの準備**タスクを実行すると、グローバル設定を作成し、ビジネス サーバーの Skype で使用するためのユニバーサル グループを作成します。
