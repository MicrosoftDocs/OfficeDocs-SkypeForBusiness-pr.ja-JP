# <a name="prepare-current-forrest-intro"></a>現在のフォレストの準備 (開始)
 
**[フォレストの準備]** タスクを開始するには、**[次へ]** をクリックします。 フォレストの **準備タスクを実行** すると、グローバル設定が作成され、ユーザーが使用するユニバーサル Skype for Business Server。
