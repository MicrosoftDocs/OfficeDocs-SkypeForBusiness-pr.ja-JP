# <a name="pending-certificate-returned"></a>保留中の証明書 (返される)
 
**オンライン証明書要求の状態**] ページでは、証明書の現在のステータスを返します。 証明書が証明機関の管理者の承認待ち状態のままである場合などですが、されていない場合は、まだ承認されて、状態は、証明書がまだ処理されていることを示します。
  
保留中の要求が承認され、証明書の取得が成功した場合は、状態ことを示す証明書をローカル証明書ストアに割り当てられているステータス ウィンドウは、証明書の拇印も表示されます。
  
証明機関の管理者が証明書の要求を拒否した場合、ステータス ウィンドウは (証明機関が管理している)、指定された ID を持つ保留中の証明書の要求が拒否されたことを報告します。
  
保留中の証明書の処理を完了する**完了**をクリックします。
