# <a name="pending-certificate-returned"></a>保留中の証明書 (結果)
 
[**オンライン証明書の要求の状態**] ページでは、証明書の現在の状態を返します。 証明書がまだ承認されていない場合は、証明機関管理者によって保留中の承認状態になっている場合、その状態は、証明書がまだ処理中であることを示します。
  
保留中の要求が承認されていて、証明書の取得に成功した場合、状態はローカル証明書ストアに証明書が割り当てられていることを示し、[状態] ウィンドウには証明書の拇印も表示されます。
  
証明機関管理者が証明書の要求を拒否した場合、[状態] ウィンドウには、特定の ID (証明機関によって管理される) の保留中の証明書の要求が拒否されたことが通知されます。
  
[**完了**] をクリックして、保留中の証明書処理を完了します。
