# <a name="setup-or-remove-skype-for-business-server-components-invoke"></a>Skype for Business Server コンポーネントのセットアップまたは削除 (起動)
 
**コマンドの実行**には、サーバーの役割を展開するのには、実行するタスクの実行の概要が表示されます。 タスクが完了した後を展開し、アクティブまたは非アクティブし、アンインストール、概要が表示されます。 さらに、ボックスの一覧で選択コマンドは、実行時に作成されるログ ファイルのいずれか。 、表示するログ ファイルを選択するドロップダウン リストを使用し、[**ログの表示**] をクリックします。
  
終了後の概要とログのファイルを確認するには、**完了**] をクリックします。
