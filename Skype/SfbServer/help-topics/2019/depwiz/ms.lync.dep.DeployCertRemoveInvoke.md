# <a name="remove-certificate-invoke"></a>証明書の削除 (起動)
 
[**実行コマンド**] ページには、証明書を削除するための詳細が表示されます。 証明書の削除からサマリー情報を確認します。 [証明書の削除] ログファイルがドロップダウンリストに表示されていることを確認し、[**ログの表示**] をクリックして、証明書の削除の詳細を確認します。 概要とログファイルの確認が完了したら、[**完了**] をクリックします。
