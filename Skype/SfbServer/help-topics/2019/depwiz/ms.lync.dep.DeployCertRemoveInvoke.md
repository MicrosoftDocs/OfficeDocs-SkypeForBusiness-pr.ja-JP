# <a name="remove-certificate-invoke"></a>証明書の削除 (起動)
 
[**コマンドを実行しています**] ページには、証明書の削除の詳細が表示されます。証明書削除の概要情報を確認します。ドロップダウン リストに [証明書の削除] ログ ファイルが表示されていることを確認し、[**ログの表示**] をクリックして証明書削除の詳細を確認します。概要とログ ファイルの確認が終了したら、[**完了**] をクリックします。
