# <a name="lync-server-2013-create-new-expander"></a>Lync Server 2013 新規作成エキスパンダー
 
トポロジ ビルダーの Lync Server 2013 ノードで新しいサーバー ロールを作成するには、**Lync Server 2013** を右クリックし、[新規] を選択してから、作成するサーバー の役割を選択します。
