# <a name="remove-deployment-introduction"></a>展開の削除の概要
 
**展開の削除ウィザード**は、展開全体を使用停止にする場合のみ実行します。このウィザードによって、1 つのフロントエンド サーバーで構成される空のトポロジが作成されます。このウィザードを実行して空のトポロジを公開すると、元のトポロジに戻すことができなくなります。
  
> [!IMPORTANT]
> [**次へ**] をクリックする前に、ウィザードの最初のページに指定されているように、非アクティブなすべてのコンピューターを削除し、統合コミュニケーションからすべてのユーザーとエンドポイントを無効にし、すべての会議ディレクトリを削除していることを確認してください。
