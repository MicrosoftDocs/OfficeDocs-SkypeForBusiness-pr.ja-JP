# <a name="remove-deployment-introduction"></a>展開の削除の概要
 
展開全体の使用を停止する場合にのみ、**展開の削除ウィザード**を実行します。 このウィザードは、1つのフロントエンドサーバーで構成される空のトポロジを作成します。 このウィザードを実行して空のトポロジを公開した後は、元のトポロジに戻すことはできません。
  
> [!IMPORTANT]
> [**次へ**] をクリックする前に、非アクティブなコンピューターを削除し、統合された通信からすべてのユーザーとエンドポイントを無効にして、ウィザードの最初のページで指定したとおりにすべての会議ディレクトリを削除してください。
