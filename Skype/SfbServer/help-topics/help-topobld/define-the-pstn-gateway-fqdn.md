# <a name="define-the-pstn-gateway-fqdn"></a>PSTN ゲートウェイの FQDN の定義

エンタープライズボイスコンポーネントを PSTN またはプライベートブランチ exchange (PBX) に接続する、公衆交換電話網 (PSTN) ゲートウェイの完全修飾ドメイン名 (FQDN) を指定します。

詳細については、「 [Skype For Business Server の PSTN 接続コンポーネント 2015](../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity.md) 」を参照してください。
