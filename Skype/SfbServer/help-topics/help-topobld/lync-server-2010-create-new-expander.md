# <a name="lync-server-2010-create-new-expander"></a>Lync Server 2010 の新規作成エキスパンダー
 
[Lync server **2010**] を右クリックし、[**新規**作成] を選択して、作成するサーバーの役割を選ぶと、トポロジビルダーの lync server 2010 ノードで新しいサーバーの役割を作成できます。
