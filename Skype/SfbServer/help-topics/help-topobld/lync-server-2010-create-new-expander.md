# <a name="lync-server-2010-create-new-expander"></a>Lync Server 2010 新規作成エキスパンダー
 
Lync Server 2010 を右クリックし、[新規] を選択し、作成するサーバーの役割を選択して、トポロジ ビルダーの **Lync Server 2010** ノードに新しいサーバー の役割を作成できます。
