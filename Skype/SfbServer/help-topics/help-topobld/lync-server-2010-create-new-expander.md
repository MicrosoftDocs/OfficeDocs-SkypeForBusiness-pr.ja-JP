# <a name="lync-server-2010-create-new-expander"></a>Lync Server 2010 新規作成エキスパンダー
 
Lync Server 2010 を右クリックし、[新規] を選択して、作成するサーバーの役割を選択すると、トポロジ ビルダーの **Lync Server 2010** ノードで新しいサーバーの役割を作成できます。
