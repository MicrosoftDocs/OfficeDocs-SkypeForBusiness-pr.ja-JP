# <a name="lync-server-2010-create-new-expander"></a>Lync Server 2010 新規作成エキスパンダー
 
トポロジ ビルダーの Lync Server 2010 ノードで新しいサーバー の役割を作成するには、**Lync Server 2010** を右クリックし、[新規] を選択してから、作成するサーバーの役割を選択します。
