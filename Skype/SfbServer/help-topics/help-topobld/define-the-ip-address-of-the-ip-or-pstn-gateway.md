# <a name="define-the-ip-address-of-the-ip-or-pstn-gateway"></a>IP または PSTN ゲートウェイの IP アドレスの定義

IP または電話交換電話網 (PSTN) ゲートウェイの IPv4 アドレスまたは IPv6 アドレスを定義するには、[ **Ipv4 を有効に**する] または [ **ipv6 を有効**にする] を選択します。 [指定した**ip アドレスをすべて使用**する] または [**サービスの使用を制限する] を選択した Ip アドレスに制限**することができます。

[**設定済み IP アドレスをすべて使用**する] を選択すると、ゲートウェイで構成されているすべてのアドレスが使用されます。 必要に応じて、[**サービスの利用制限を選択した ip アドレスに制限**] を選択して、 **PSTN IP アドレス**の入力を有効にします。 このゲートウェイに割り当てられている IPv4 アドレスまたは IPv6 アドレスを入力します。
