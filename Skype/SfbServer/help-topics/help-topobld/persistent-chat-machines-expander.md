# <a name="persistent-chat-machines-expander"></a>常設チャット コンピューターの展開
 
展開済みの常設チャットサーバーまたは常設チャットサーバープールをアクティブ化または非アクティブ化するには、[ **コンピューターの状態の設定**] にリストされているサーバーまたはプールを使用します。 一覧にあるサーバーまたはプールを選択し、[**アクティブにする**] をクリックすると、サーバーまたはプールはアクティブに設定されます。
  
一覧でサーバーまたはプールを選択し、[ **非アクティブ**にする] をクリックして、展開済みの常設チャットサーバーまたは常設チャットサーバープールを非アクティブに設定します。 ただし、少なくとも 1 台のサーバーがアクティブに設定されている必要があります。
  
 [**OK**]: ダイアログでの変更を受け入れて確定します。
  
 [**キャンセル**]: 変更を破棄してダイアログを閉じます。
  
 [**ヘルプ**]: このヘルプ画面を表示します。
