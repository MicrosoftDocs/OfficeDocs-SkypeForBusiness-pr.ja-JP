# <a name="persistent-chat-machines-expander"></a>常設チャット コンピューター エキスパンダー
 
コンピューターの状態設定に記載されているサーバーまたはプールを使用して、展開された常設チャット サーバーまたは常設チャット サーバー プールをアクティブ化または **非アクティブ化します**。 一覧にあるサーバーまたはプールを選択し、[**アクティブにする**] をクリックすると、サーバーまたはプールはアクティブに設定されます。
  
一覧でサーバーまたはプールを選択し、[非アクティブにする] をクリックして、展開された常設チャット サーバーまたは常設チャット サーバー プールを非アクティブ **に設定します**。 ただし、少なくとも 1 台のサーバーがアクティブに設定されている必要があります。
  
 [**OK**]: ダイアログでの変更を受け入れて確定します。
  
 [**キャンセル**]: 変更を破棄してダイアログを閉じます。
  
 [**ヘルプ**]: このヘルプ画面を表示します。
