# <a name="associate-director-with-monitoring-page-for-lync-server-2010"></a>ディレクターが Lync Server 2010 のページを監視することを関連付ける
 
監視サーバーを構成し、それのディレクターまたはディレクターにプール。
  
**サーバーの監視**の下で監視サーバーを一覧から選択します。
  
サーバーの監視がないと、サーバーの監視をすぐに作成する、[**新規**] をクリックします。
  
 **戻る**: 前のダイアログに戻ります。
  
 **完了**: 変更をコミットし、ウィザードを完了します。
  
 **キャンセル**: 変更を破棄し、構成をキャンセルします。
  
 **ヘルプ**: このトピックなどの情報のヘルプを表示
