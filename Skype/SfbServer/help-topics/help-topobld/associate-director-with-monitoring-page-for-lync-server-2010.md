# <a name="associate-director-with-monitoring-page-for-lync-server-2010"></a>Lync Server 2010 用のディレクターと監視の関連付けページ
 
監視サーバーを構成し、それをディレクターまたはディレクタープールに関連付けるには、次の操作を行います。
  
[ **Monitoring server** ] で、監視サーバーの一覧から選びます。
  
監視サーバーがなく、今すぐ監視サーバーを作成する場合は、[**新規**] をクリックします。
  
 **戻る**: 前のダイアログに戻ります。
  
 **[完了**]: 変更をコミットしてウィザードを完了します。
  
 **[キャンセル]**: 変更を破棄して構成を取り消します。
  
 [**ヘルプ**]: このトピックなどのヘルプ情報を表示します。
