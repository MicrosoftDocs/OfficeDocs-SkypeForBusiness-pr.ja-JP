# <a name="associate-director-with-monitoring-page-for-lync-server-2010"></a>Lync Server 2010 用のディレクターと監視の関連付けページ
 
監視サーバーを構成し、ディレクター またはディレクター プールに関連付けるには、次の手順を実行します。
  
[監視 **サーバー] の** 一覧から監視サーバーを選択します。
  
監視サーバーがない場合に監視サーバーを今すぐ作成する場合は、[新規] を **クリックします**。
  
 [**戻る**]: 前のダイアログに戻ります。
  
 [**完了**]: 変更を確定して、ウィザードを完了します。
  
 [**キャンセル**]: 変化を破棄し、構成をキャンセルします。
  
 [**ヘルプ**]: このトピックのようなヘルプ情報を表示します。
