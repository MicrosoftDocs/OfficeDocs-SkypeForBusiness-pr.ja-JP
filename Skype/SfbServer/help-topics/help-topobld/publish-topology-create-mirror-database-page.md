# <a name="publish-topology-create-mirror-database-page"></a>トポロジの公開でのミラー データベースの作成ページ
 
このドキュメントは暫定版であり、変更される可能性があります。空白のトピックがプレースホルダーとして含まれています。
  
トポロジ ビルダーは、定義されたトポロジ ドキュメント内のすべての構成設定を解析します。 ミラー 化されたデータベースを構成SQL Server、ミラー ストア、プライマリ ストア、ファイル共有パス、およびデータベース パスを表示する画面が表示されます。 ミラー化されたデータベースとファイル共有パスを作成するには、データベース ミラー ストアに関連付けられているチェック ボックスをオンにします。 データベースを作成して定義されたデータベースに展開しない場合は、データベース ミラーのチェック ボックスをオフSQL Server。
  
 この時点でこれらのデータベースの 1 つまたはすべてをインストールしない場合は、インストールしないデータベースをクリアします。 後で、トポロジ ビルダーの **ほとんどのノード** アイテムを右クリックして、[データベースのインストール] を使用します。
