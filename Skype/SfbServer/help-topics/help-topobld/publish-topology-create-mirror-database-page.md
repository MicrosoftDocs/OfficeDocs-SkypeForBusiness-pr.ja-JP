# <a name="publish-topology-create-mirror-database-page"></a>トポロジの公開でのミラー データベースの作成ページ
 
このドキュメントは暫定版であり、変更される可能性があります。 空白のトピックがプレースホルダーとして含まれています。
  
トポロジ ビルダーは、定義したトポロジ ドキュメント内のすべての構成設定を解析します。 ミラー 化された SQL Server データベースを構成した場合は、ミラー ストア、プライマリ ストア、ファイル共有パス、およびデータベース パスを表示する画面が表示されます。 ミラー 化されたデータベースとファイル共有パスを作成するには、データベース ミラー ストアに関連付けられているチェック ボックスをオンにします。 データベースを作成して定義済みのデータベース に展開しない場合は、データベース ミラーのチェック ボックスをSQL Server。
  
 現時点でこれらのデータベースの 1 つまたはすべてはインストールしない場合は、インストールしないデータベースをクリアします。 後で、トポロジ ビルダーのほとんどのノード項目を右クリックして、[データベースのインストール] を使用します。
