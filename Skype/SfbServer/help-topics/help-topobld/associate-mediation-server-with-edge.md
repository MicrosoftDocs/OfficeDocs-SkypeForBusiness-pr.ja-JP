# <a name="associate-mediation-server-with-edge"></a>仲介サーバーとエッジの関連付け
 
仲介サーバーを定義するには、サーバーをフロント エンド サーバーと、エッジ サーバーまたはプールに仲介サーバーを関連付けるための設定を指定します。 エッジ サーバーまたはエッジ サーバー プールに仲介サーバーまたはプールを関連付けるには、次の操作を行います。
  
1. 」ドロップ ダウン リストからエッジ プールまたは仲介サーバーに関連付けるエッジ サーバーを選択します。
    
2. [**次へ**] をクリックして続行します。
    
> [!NOTE]
> まだ未定義の場合、エッジ サーバーまたはエッジ サーバー プールの展開で、ドロップ ダウン リストのエントリは含まれません。 **次へ**を続行し、展開に、エッジ サーバーまたはプールを追加した後は、仲介サーバーの設定を編集をクリックすることができます。
  
詳細については、 [Skype のビジネス サーバー 2015 でトポロジ ビルダーでの仲介サーバーの展開](../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md)を参照してください。
