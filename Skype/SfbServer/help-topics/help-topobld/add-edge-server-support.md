# <a name="add-edge-server-support"></a>エッジ サーバーのサポートを追加します。

初期のトポロジを展開するときに、外部ユーザー アクセスまたは後でのサポートを追加することができます。 既存のトポロジにエッジ サーバーを追加する方法の詳細は、エッジ サーバーの展開に関するドキュメントの「[エッジ トポロジ定義](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx)を参照してください。
