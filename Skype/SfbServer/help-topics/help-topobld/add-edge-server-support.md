# <a name="add-edge-server-support"></a>エッジ サーバーのサポートの追加

最初のトポロジを展開するとき、または後で、外部ユーザーアクセスのサポートを追加することができます。 既存のトポロジにエッジ サーバーを追加する方法の詳細については、「エッジ サーバーの展開」のドキュメントの「[Define Your Edge Topology](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx)」を参照してください。
