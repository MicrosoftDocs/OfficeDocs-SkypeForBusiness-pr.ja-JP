# <a name="install-database-create-mirror-database-page"></a>データベースのインストールでのミラー データベースの作成ページ
 
このドキュメントは暫定版であり、変更される可能性があります。 空白のトピックがプレースホルダーとして含まれています。
  
公開プロセス中に SQL Server ミラー データベースをインストールしないことを選択した場合は、トポロジ ビルダーの任意のノードを右クリックして 、[データベースのインストール] を使用してインストールします。 インストールするミラー ストアを選択します。 詳細設定を定義するには、[**設定**] をクリックします。
