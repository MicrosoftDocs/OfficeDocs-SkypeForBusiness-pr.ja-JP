このドキュメントは暫定版であり、変更される可能性があります。 空白のトピックがプレースホルダーとして含まれています。
  
公開プロセス中に SQL Server ミラーデータベースをインストールしないことを選択した場合は、トポロジビルダーの任意のノードで、[**データベースのインストール**] をクリックしてインストールします。 インストールするミラーストアを選択します。 [**設定**] をクリックして、詳細設定を定義します。
