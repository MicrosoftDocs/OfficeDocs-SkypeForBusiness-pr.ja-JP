このドキュメントは暫定版であり、変更される可能性があります。 空白のトピックがプレースホルダーとして含まれています。
  
発行プロセス中に SQL Server ミラーデータベースをインストールしないことを選択した場合は、トポロジビルダーの任意のノードで、[ **データベースのインストール** ] を使用してインストールします。 インストールするミラー ストアを選択します。 詳細設定を定義するには、[**設定**] をクリックします。
