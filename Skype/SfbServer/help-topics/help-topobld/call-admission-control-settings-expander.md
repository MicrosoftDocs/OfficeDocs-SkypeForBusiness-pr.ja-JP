# <a name="call-admission-control-settings-expander"></a>通話受付管理設定エキスパンダー
 
通話受付管理 (CAC) は、使用可能な帯域幅に基づいてオーディオおよびビデオ伝送に制限を課すことができる地域、サイト、およびサブネットで構成されるネットワークです。 CAC のネットワークを構成したら、帯域幅制限を適用するために CAC を有効にする必要があります。 
  
> [!NOTE]
> コントロールパネルまたは管理シェルコマンドレットを使用して CAC を有効にすることもできます。 
  
サイトの [**プロパティの編集**] ダイアログボックスの [**通話受付管理の設定**] セクションでは、次の設定を変更できます。
  
- **通話受付管理を有効にする** CAC を有効にするには、この設定を選択します。 ネットワーク全体で CAC を無効にするには、この設定をオフにします。 CAC を有効にするには、CAC 用にネットワークを構成しておく必要があります。 詳細については、「展開」のドキュメントの「 [deploy call 受付管理 In Skype For Business Server 2015](../../deploy/deploy-enterprise-voice/deploy-call-admission-control.md) 」を参照してください。
    
- **通話受付管理を実行するフロントエンドプール** CAC を有効にした場合、それを実行するプールを変更できます。 ドロップダウンリストからプールを選択します。
