# <a name="add-mediation-server-nexthop"></a>仲介サーバーの次ホップを追加する
 
定義する仲介サーバーの次ホップを定義するには、[ **次ホッププール** ] ドロップダウンリストから、仲介サーバーの次ホップとして機能する Enterprise Edition サーバーまたは Standard Edition サーバーを選択します。 [**次へ**] をクリックして続行します。
