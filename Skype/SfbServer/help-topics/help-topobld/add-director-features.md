# <a name="add-director-features"></a>ディレクターの機能の追加

監督プールに監視サーバーを含める場合は、[ **monitoring (CDR および QoE メトリック)**] を選びます。

ディレクタープールの追加の詳細については、「[ディレクターのシナリオ](https://technet.microsoft.com/library/d2cf384a-0860-4779-80ce-cba2543be322.aspx)と[ディレクターのセットアップ](https://technet.microsoft.com/library/408b76f7-6fdd-4e50-8a3e-e87db12c1394.aspx)」を参照してください。
