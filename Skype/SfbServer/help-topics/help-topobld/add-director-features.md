# <a name="add-director-features"></a>ディレクター機能の追加

ディレクター プールに監視サーバーを含める場合は、監視 **(CDR および QoE 指標) を選択します**。

ディレクター プールの追加の詳細については、「 [ディレクター](https://technet.microsoft.com/library/d2cf384a-0860-4779-80ce-cba2543be322.aspx) のシナリオ」および「ディレクターの設定 [」を参照してください](https://technet.microsoft.com/library/408b76f7-6fdd-4e50-8a3e-e87db12c1394.aspx)。
