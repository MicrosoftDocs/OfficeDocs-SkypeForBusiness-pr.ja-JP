# <a name="associate-branch-office-appliance-with-front-end"></a>ブランチ オフィス アプライアンスをフロント エンドと関連付ける
 
存続可能ブランチ アプライアンスまたは存続可能ブランチ サーバーをフロントエンド プールまたは Standard Edition サーバーに関連付けるには、[フロント エンド プールの選択]  ページで、[フロントエンド プール] ドロップダウン リストからフロントエンド プールまたは Standard Edition サーバーを選択します。 [**次へ**] をクリックして続行します。
