# <a name="associate-branch-office-appliance-with-front-end"></a>ブランチ オフィス アプライアンスとフロントエンドの関連付け
 
Survivable Branch Appliance または Survivable Branch Server をフロントエンドプールまたは Standard Edition サーバーに関連付けるには、 **[フロント**エンドプールの選択] ページで、フロントエンドプールまたは標準エディションのサーバーを**フロントエンドプール**から選びます。ドロップダウンリスト。 [**次へ**] をクリックして続行します。
