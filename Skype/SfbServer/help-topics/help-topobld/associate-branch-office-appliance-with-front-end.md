# <a name="associate-branch-office-appliance-with-front-end"></a>ブランチ オフィス アプライアンスをフロントエンドと関連付ける
 
存続可能 Branch Appliance または存続可能ブランチサーバーをフロントエンドプールまたは Standard Edition サーバーに関連付けるには、[フロントエンド **プールの選択** ] ページで、フロントエンドプールまたは standard edition サーバーを [ **フロント** エンドプール] ボックスの一覧から選択します。 [**次へ**] をクリックして続行します。
