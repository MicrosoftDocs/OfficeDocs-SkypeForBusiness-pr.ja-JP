# <a name="legacy-merge-summary-page"></a>従来版のマージの概要ページ

Communications Server 2007 R2 Microsoft Office Lync Server 2013 への移行では、トポロジ ビルダーの機能を使用して既存のトポロジをマージします。 概要画面では、lync Server 2013 トポロジ ビルダー構成への Office Communications Server 2007 R2 の統合の構成中に行ったすべてのアクションと選択が指定されます。

この概要を確認し、選択した内容が適切に定義されているかどうかを判断します。概要に示されている選択内容に問題がなければ、マージを続行します。
