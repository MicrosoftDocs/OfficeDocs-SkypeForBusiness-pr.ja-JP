# <a name="legacy-merge-results"></a>従来のマージの結果

ビジネス サーバーの Skype での従来の配置をマージするには、**次**の手順をクリックします。 トポロジ ビルダー結合ウィザードの実行の詳細については、「移行」のドキュメントの「[Merge Using Topology Builder Merge Wizard](https://technet.microsoft.com/library/c3f3c425-dab6-4dcd-bf0e-d7fde05f2ebf.aspx)」を参照してください。
