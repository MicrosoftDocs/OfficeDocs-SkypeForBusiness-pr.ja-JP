# <a name="publish-topology-next-step"></a>トポロジの公開の次の手順

トポロジの発行が完了すると、**発行ウィザード**の [完了] ページが表示されます。

トポロジの公開プロセス中に手順を完了すると、実行されたタスクに関する情報を含むログが作成されます。 特定の手順に関連するログファイルを表示するには、ログファイルを確認する必要がある手順を選択し、[**ログの表示**] をクリックします。

> [!NOTE]
> [トポロジの発行プロセスの実行中に、ログファイルが関連付けられることはありません。 手順を選択して**ログを表示**できる場合は、この手順でログファイルが作成されます。 [**ログの表示**] ボタンが使用できない (淡色表示されている) 場合、この手順でログファイルは作成されませんでした。

トポロジを完了するために推奨される次の手順を確認するには、 **[ここをクリックして to do リストを開き**ます] リンクをクリックします。 このリンクをクリックして開くテキストファイルでは、ローカルセットアップを実行する必要があるサーバーと、展開を完了するために必要な追加の手順が示されます。 記載されている各タスクについて、展開ドキュメントの該当するセクションを参照してください。 詳細については、「 [Lync Server 2013 Preview の展開](https://technet.microsoft.com/library/b76795a4-4e71-4c70-a5c0-d1197fa8028c.aspx)」を参照してください。また、to do リストに記載されているサーバーの役割ごとに、関連するセクションを参照してください。

[**完了**] をクリックする

詳細については、「[トポロジを発行する](https://technet.microsoft.com/library/3b5a744b-b3a8-4538-a55e-e2e4f72dff47.aspx)」を参照してください。
