# <a name="install-mirror-database-option-page"></a>ミラー データベースのインストール オプション ページ
 
以下のものを定義することにより、[**ミラー データベースの設定**] を構成します。
  
- [ファイル **共有へのパス]** を入力して、ミラーリングするデータベースのSQL Serverの場所を定義します。
    
    > [!NOTE]
    > プライマリ インスタンスSQL Server (名前付きインスタンスまたは既定のインスタンス) には、ここで定義したファイル共有への書き込みアクセス許可が必要です。 ミラー インスタンスSQL Server (名前付きインスタンスまたは既定のインスタンス) は、同じファイル共有に対する読み取りアクセス許可を持っている必要があります。 
  
  [**OK**]: ダイアログでの変更を受け入れて確定します。
  
  [**キャンセル**]: 変更を破棄してダイアログを閉じます。
  
  [**ヘルプ**]: このヘルプ画面を表示します。
