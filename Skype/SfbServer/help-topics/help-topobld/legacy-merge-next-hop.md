# <a name="legacy-merge-next-hop"></a>従来版のマージの次ホップ
 
**次ホップ FQDN**は、従来のエッジサーバーが内部トラフィックをルーティングする次ホップサーバーを識別します。 このエッジサーバーが内部トラフィックをルーティングする次ホップサーバーの完全修飾ドメイン名 (FQDN) を入力し、**フロントエンドプール**と**ディレクター**のどちらであるかを選択します。
