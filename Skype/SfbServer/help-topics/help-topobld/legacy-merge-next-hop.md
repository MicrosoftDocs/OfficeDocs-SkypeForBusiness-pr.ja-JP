# <a name="legacy-merge-next-hop"></a>従来版のマージの次ホップ
 
[**次ホップ FQDN**] では、従来のエッジ サーバーが内部トラフィックをルーティングする次ホップ サーバーを指定します。このエッジ サーバーによる内部トラフィックのルーティング先となる次ホップ サーバーの完全修飾ドメイン名 (FQDN) を入力し、それが [**フロントエンド プール**] と [**ディレクター**] のどちらであるかを選択します。
