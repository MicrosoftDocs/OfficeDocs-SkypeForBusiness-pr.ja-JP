# <a name="publish-topology"></a>トポロジの公開
 
**[トポロジの公開]** ページは、公開プロセスを開始すると表示されます。処理を進める前に、一覧表示されるタスクを読んで、それらのタスクが完了していることを確認します。処理を進める準備ができたら、[**次へ**] をクリックします。公開プロセスを終了する必要がある場合は、[**キャンセル**] をクリックします。
