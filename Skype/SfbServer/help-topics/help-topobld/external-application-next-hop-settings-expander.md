# <a name="external-application-next-hop-settings-expander"></a>外部アプリケーションの次ホップ設定エキスパンダー
 
信頼済みアプリケーションサーバープールの次ホップサーバーを指定するには、定義済みの Enterprise Edition フロントエンドプールまたは Standard Edition フロントエンドサーバーをドロップダウンリストから選択します。 ディレクターまたはディレクター プールは、信頼済みアプリケーション サーバーの次ホップとしては有効な選択肢ではないため、リストに表示されません。 [ **OK]** をクリックして変更を受け入れ、保存します。 変更を破棄してプロパティ ページを終了するには、**[キャンセル]** をクリックします。
