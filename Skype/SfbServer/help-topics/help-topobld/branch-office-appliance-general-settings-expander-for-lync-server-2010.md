# <a name="branch-office-appliance-general-settings-expander-for-lync-server-2010"></a>Lync Server 2010 用のブランチ オフィス アプライアンス全般設定エキスパンダー
 
存続可能ブランチ アプライアンスまたは存続可能ブランチ サーバーのプロパティを編集するには、[全般] **で、次** の構成を行います。
  
- **FQDN**: 存続可能ブランチ アプライアンスまたは存続可能ブランチ サーバーの完全修飾ドメイン名を入力します。
    
    **すべての構成済み IP アドレスは** 、すべての目的で存続可能ブランチ アプライアンスまたは存続可能ブランチ サーバーで構成された IP アドレスを使用します。
    
    [**選択された IP アドレスのみにサービスの使用を制限する**] PSTN で使用するサーバーおよび IP アドレスを定義する別のアドレスを構成します。
    
    [**プライマリ IP アドレス**]: PSTN 関連の機能を除く、すべての目的のために IP アドレスを定義および構成します。
    
    [**PSTN の IP アドレス**]: 公衆交換電話網 (PSTN) 機能に関連付けられた IP アドレス。
    
- 関連付 **けを構成して** 、他のサーバーの役割が構成され、存続可能ブランチ アプライアンスまたは存続可能ブランチ サーバーに関連付けられている必要があります。
    
    **アーカイブ サーバーの関連付け** 一覧から、存続可能ブランチ アプライアンスまたは存続可能ブランチ サーバーに関連付けるアーカイブ サーバーを選択します。
    
    この **存続可能** ブランチ アプライアンスまたは存続可能ブランチ サーバーに関連付けるアーカイブ サーバーを作成していない場合は、[新規] をクリックします。
    
    **監視サーバーの関連付け** 存続可能ブランチ アプライアンスまたは存続可能ブランチ サーバーに関連付ける監視サーバーを一覧から選択します。
    
    この **存続可能** ブランチ アプライアンスまたは存続可能ブランチ サーバーに関連付ける監視サーバーを作成していない場合は、[新規] をクリックします。
    
    **エッジ プールの関連付け (メディア コンポーネントの場合)** 一覧から、存続可能ブランチ アプライアンスまたは存続可能ブランチ サーバーに関連付けるエッジ サーバーまたはエッジ プールを選択します。
    
  [**OK**]: ダイアログでの変更を受け入れて確定します。
  
  [**キャンセル**]: 変更を破棄してダイアログを閉じます。
  
  [**ヘルプ**]: このヘルプ画面を表示します。
