# <a name="add-director-association-for-lync-server-2010"></a>Lync Server 2010 のディレクターの関連付けを追加します。
 
定義する、ディレクターまたはディレクターのプールがあるかどうかがサーバーの監視に関連付けられていること。 ディレクターまたはディレクターに監視サーバーを関連付けるにはチェック ボックスをオンにプールされます。
  
> [!TIP]
> ディレクターまたはディレクター、監視サーバーに関連付けるには、このチェック ボックスを選択する場合は、後で作成することを指定しないをまだ作成して、監視サーバー データベース (および省略可能なインスタンス) 場合、プールです。
