# <a name="add-monitoring-server"></a>監視サーバーの追加

監視サーバーをトポロジ設計に組み込むには、監視サーバーを展開するサーバーの完全修飾ドメイン名 (FQDN) を指定する必要があります。 監視サーバーを含むトポロジを公開し、サーバーに Skype for Business Server をインストールする前に、監視を展開するための前提条件を満たす必要があります。 これらの前提条件の詳細については、「展開」のドキュメントの「[Deploying Monitoring](/previous-versions/office/lync-server-2013/lync-server-2013-deploying-monitoring)」を参照してください。
