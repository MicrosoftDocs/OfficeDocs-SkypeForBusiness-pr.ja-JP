# <a name="video-interop-server-general-settings"></a>ビデオ相互運用サーバーの全般設定
 
トポロジ ビルダーで新しい VIS またはビデオ ゲートウェイを定義するために必要な手順については、「Create [a VIS pool in the VIS pool in Skype for Business Server 2015」を参照](../../deploy/deploy-video-interop-server/create-a-vis-pool.md)してください。
  
[2015](../../deploy/deploy-video-interop-server/deploy-the-vis-server-role.md) 年の VIS サーバーの役割の展開で、Windows PowerShell を使用してサーバーにビデオ相互運用サーバーの役割を割り当てるSkype for Business Server。
  
2015 年のビデオ相互運用サーバーの構成手順については、「Windows PowerShell [Skype for Business Server 2015](../../deploy/deploy-video-interop-server/configure-the-vis.md) でビデオ相互運用サーバーを構成する」を参照してください。
  
 VIS との相互運用のために Cisco GUI で設定する必要があるビデオ ゲートウェイの設定については、「Configure [CUCM for Interoperation for interoperation with Skype for Business Server 2015」を参照](../../deploy/deploy-video-interop-server/configure-cucm-for-interoperation.md)してください。
  
 ビデオ ゲートウェイと VIS との相互運用のために Cisco GUI で設定する必要がある VTC の設定については、「Configure [a VTC for Interoperation for interoperation with Skype for Business Server 2015」をオンラインで](../../deploy/deploy-video-interop-server/configure-a-vtc-for-interoperation.md)説明します。
