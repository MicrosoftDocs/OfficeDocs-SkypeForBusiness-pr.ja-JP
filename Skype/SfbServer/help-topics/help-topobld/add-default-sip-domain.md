# <a name="add-default-sip-domain"></a>既定の SIP ドメインを追加する
 
展開でサポートするプライマリ セッション開始プロトコル (SIP) ドメインを指定します。 プライマリ SIP ドメインは、Active Directory ドメインサービスドメインと同じであったり、異なる場合があります。 たとえば、Active Directory ドメインが fabrikam.net の場合に、プライマリ SIP ドメインとして contoso.net などを指定できます。
