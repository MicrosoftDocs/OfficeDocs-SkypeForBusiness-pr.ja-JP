# <a name="lync-server-2013-scope-node"></a>Lync Server 2013 のスコープ ノード
 
トポロジビルダーで新しい VIS またはビデオゲートウェイを定義するために必要な手順については、オンラインの「 [Create a VIS pool In Skype For Business Server 2015](../../deploy/deploy-video-interop-server/create-a-vis-pool.md)」を参照してください。
  
Windows PowerShell を使用してサーバーにビデオ相互運用サーバーの役割を割り当てるために必要な手順については、「 [Deploy THE VIS server role In Skype For Business server 2015」](../../deploy/deploy-video-interop-server/deploy-the-vis-server-role.md)を参照してください。
  
Windows PowerShell を使用した新しいビデオ相互運用サーバーの構成手順については、オンラインの「 [Configure The Video Interop server In Skype For Business server 2015」](../../deploy/deploy-video-interop-server/configure-the-vis.md)を参照してください。
  
 VIS との相互運用のために Cisco の GUI で設定する必要があるビデオゲートウェイの設定については、「 [CONFIGURE CUCM For 相互運用と Skype For Business Server 2015](../../deploy/deploy-video-interop-server/configure-cucm-for-interoperation.md)」を参照してください。
  
 ビデオゲートウェイおよび VIS との相互運用のために Cisco の GUI で設定する必要がある VTC の設定については、オンラインの「 [Skype for Business Server 2015 との相互運用のための VTC の構成](../../deploy/deploy-video-interop-server/configure-a-vtc-for-interoperation.md)」を参照してください。
