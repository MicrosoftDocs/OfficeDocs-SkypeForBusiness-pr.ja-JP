# <a name="deploy-instant-messaging-and-presence-in-skype-for-business-server"></a>Business Server のインスタント メッセージングとプレゼンスでは、Skype を導入します。
 
**の概要:** Business Server のインスタント メッセージングとプレゼンスを Skype での展開の手順について説明します。
  
## <a name="deployment-steps-for-instant-messaging-and-presence"></a>インスタント メッセージングとプレゼンスの展開手順

インスタント メッセージングとプレゼンスの計画に関する記事では、展開を計画するための情報について段階的に説明します。 ビジネス サーバーの Skype をインストールする場合、インスタント メッセージングとプレゼンスが有効になります。 インスタント メッセージングとプレゼンスの計画に関する詳細については、[インスタント メッセージングとプレゼンス ビジネス サーバーの Skype での計画](../../plan-your-deployment/instant-messaging-and-presence.md)を参照してください。
  
正常にカスタマイズ可能な配置手順の一部について説明するここで、インスタント メッセージングとプレゼンスのシナリオを調整します。
