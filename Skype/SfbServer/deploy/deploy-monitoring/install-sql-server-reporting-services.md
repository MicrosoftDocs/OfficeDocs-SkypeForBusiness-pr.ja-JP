# <a name="install-sql-server-reporting-services-in-skype-for-business-server"></a>Skype for Business Server での SQL Server Reporting Services のインストール 
 
**概要:** Skype for Business Server で使用される SQL Server Reporting Services に関する情報を見つける方法について説明します。
  
Skype for Business Server は、レポートを表示および監視するために SQL Server Reporting Services (SSRS) を使用できます。 この機能を使用するには、Reporting Services がインストールされ、構成されている必要があります。
  
## <a name="install-sql-server-reporting-services"></a>SQL Server Reporting Services をインストールする

Skype for Business Server の監視レポートを使用する予定がある場合 (詳細については、このドキュメントの次の記事を参照してください)、最初に SQL Server Reporting Services をインストールする必要があります。Reporting Services は、Microsoft SQL Server をインストールするとき、または SQL Server をインストールした後で、いつでもインストールできます。 SQL Server をインストールしていない場合は、このドキュメントで前述した手順に従ってください。 SQL Server をインストールするときは、[機能の選択] ページで [Reporting Services] を選択していることを確認してください。 SQL Server Reporting Services がインストールされます。
  
SQL Server Reporting Services のインストール方法については、「 [Sql Server Reporting services (SSRS)](https://technet.microsoft.com/library/ms159106.aspx)」を参照してください。
  
Sql server が既にインストールされているが、SQL Server Reporting Services をインストールしていない場合は、SQL Server Reporting Services の適切な手順に従って、その機能を追加できます。
