# <a name="install-sql-server-reporting-services-in-skype-for-business-server-2015"></a>Skype のビジネス サーバー 2015 用の SQL Server レポート サービスをインストールします。
 
**の概要:**ビジネス サーバー 2015 の Skype を使って、[SQL Server レポート サービスに関する情報を検索する場所について説明します。
  
ビジネス サーバー 2015 の Skype は、表示およびレポートを監視するため、SQL Server レポート サービス (SSRS) を使用できます。 この機能を使用するには、Reporting Services をインストールして構成する必要があります。
  
## <a name="install-sql-server-reporting-services"></a>SQL Server Reporting Services のインストール

Skype を使用する場合は、ビジネス サーバー 2015 監視レポート (詳細についてはこのドキュメントの次の資料を参照してください) する必要があります最初をインストールする SQL Server レポート サービスです。レポート サービスは、同時に Microsoft SQL Server をインストールするまたは SQL Server をインストールした後は、いつでもインストールできます。 SQL Server がインストールされていない場合、指示に従って、このドキュメントの以前のバージョンで提供されています。 SQL Server をインストールする場合は、[機能の選択] ページで、レポート サービスを選択することを確認します。 SQL Server レポート サービスをインストールします。
  
SQL Server レポート サービスをインストールする方法については、 [SQL Server レポート サービス (SSRS)](https://technet.microsoft.com/en-us/library/ms159106.aspx)を参照してください。
  
SQL Server はインストールしたが、SQL Server Reporting Services はインストールしていない場合は、SQL Server Reporting Services に適した指示に従って、この機能を追加できます。
