# <a name="install-sql-server-reporting-services-in-skype-for-business-server"></a>Skype for Business Server SQL Server Reporting Services をインストールする 
 
**概要:** Skype for Business Server で使用される Reporting Services のSQL Serverを検索する場所について学習します。
  
Skype for Business Server では、SQL Server Reporting Services (SSRS) を使用してレポートを表示および監視できます。 この機能を使用するには、Reporting Services をインストールして構成する必要があります。
  
## <a name="install-sql-server-reporting-services"></a>SQL Server Reporting Services のインストール

Skype for Business Server 監視レポートを使用する場合は (詳細については、このドキュメントの次の記事を参照してください)、最初に Reporting Services SQL Serverインストールする必要があります。Reporting Services は、アプリケーションのインストールと同時に、またはMicrosoft SQL Server後にいつでもSQL Serverインストールできます。 インストールされていない場合SQL Server、このドキュメントで前に説明した手順に従います。 アプリケーションをインストールSQL Server、[機能の選択] ページで Reporting Services を選択します。 この場合、Reporting Services SQL Serverインストールされます。
  
Reporting Services をインストールする方法SQL Server Reporting [Services (SSRS)](https://technet.microsoft.com/library/ms159106.aspx)SQL Server参照してください。
  
既に Reporting Services をインストールSQL Server、SQL Server Reporting Services をインストールしていない場合は、Reporting Services の適切な手順に従って、その機能SQL Server追加できます。
