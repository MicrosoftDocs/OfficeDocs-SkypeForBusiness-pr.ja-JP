# <a name="optional-verify-response-group-deployment-in-skype-for-business"></a>(省略可能)ビジネス用の Skype で確認応答グループの展開
 
ビジネス サーバーのエンタープライズ VoIP の Skype で、応答グループの展開成功を確認します。
  
応答グループを構成した後は、期待どおりに、応答グループかどうかを確認する構成を確認する必要があります。 少なくとも、次の種類のユーザーを使用して以下のシナリオを検証してください。
  
 **ユーザー**
  
- Skype のビジネス上のホーム ユーザー
    
- 公衆交換電話網 (PSTN) を使用する外部ユーザー
    
- 担当者は、Skype のビジネス上のホーム
    
  **Scenarios**
  
- ビジネス ユーザーの Skype では、応答グループを呼び出します。
    
- 外部ユーザーが応答グループを呼び出す。
    
- エージェントが別の通話に応答しているときにユーザーが応答グループを呼び出し、キューに入る。
