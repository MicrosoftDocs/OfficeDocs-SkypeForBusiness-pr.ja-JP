Skype for Business Server 環境にエッジ サーバーまたはエッジ プールを展開する理由 組織の内部ネットワークにログインしていない外部ユーザーが内部ユーザーと対話できる必要がある場合に必要です。 これらの外部ユーザーは、認証された匿名リモート ユーザー、フェデレーション パートナー、または他のモバイル クライアントである可能性があります。
