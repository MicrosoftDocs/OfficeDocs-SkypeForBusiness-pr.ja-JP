# <a name="provisioning-of-skype-room-system-exchange-and-skype-accounts"></a>Skype Room System Exchange および Skype アカウントのプロビジョニング
 
以下のトピックでは、Skype Room System の Exchange および Skype アカウントをプロビジョニングする方法について説明します。 

> [!NOTE]
> Microsoft Teams ルームは、依存関係と展開手順が異なるさまざまな製品です。 Microsoft Teams ルームの詳細については、「Microsoft Teams ルームの[展開の概要](room-systems-v2.md)」を参照してください。
  
> [!NOTE]
> Skype Room System アカウントのプロビジョニングは、組織のトポロジの種類によって異なります。 Active Directory トポロジの詳細については、「 [Skype For Business Server 2015 の環境要件](../../plan-your-deployment/requirements-for-your-environment/environmental-requirements.md)」を参照してください。
