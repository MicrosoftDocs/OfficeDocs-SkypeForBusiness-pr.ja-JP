# <a name="hardwareversions-table-in-skype-for-business-server-2015"></a>Skype for Business Server 2015 のハードウェアバージョンの表
 
ハードウェアバージョンの表はサポートテーブルです。 各レコードには、1つのデバイスハードウェアバージョンに関する情報が格納されます。
  
|**列**|**データ型**|**キー/インデックス**|**詳細**|
|:-----|:-----|:-----|:-----|
|**VersionId** <br/> |int  <br/> |Primary  <br/> |このハードウェアバージョンを識別する一意の番号。  <br/> |
|**バージョン** <br/> |nvarchar(256)  <br/> | <br/> |ハードウェアのバージョン。  <br/> |
