# <a name="pools-table"></a>Pools テーブル
 
プールテーブルは、さまざまなプールに関する情報を格納するサポートテーブルです。 テーブル内の各レコードは、1つのプールを表します。
  
|**列**|**データ型**|**キー/インデックス**|**詳細**|
|:-----|:-----|:-----|:-----|
|**PoolId** <br/> |int  <br/> |Primary  <br/> |このプールを識別する一意の番号。  <br/> |
|**PoolFQDN** <br/> |nvarchar(256)  <br/> | <br/> |プールの FQDN。  <br/> |
