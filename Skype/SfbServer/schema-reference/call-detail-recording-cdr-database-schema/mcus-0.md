# <a name="mcus-view"></a>Mcu ビュー
 
Mcu のビューでは、会議セッションに参加した Mcu に関する情報を格納します。 このビューは、Microsoft Lync Server 2013 で導入されました。
  
|**列**|**データ型**|**詳細**|
|:-----|:-----|:-----|
|**McuId** <br/> |int  <br/> |MCU を識別する一意の番号です。  <br/> |
|**McuUri** <br/> |nvarchar(450)  <br/> |MCU の URI。  <br/> |
|**McuUriType** <br/> |nvarchar(256)  <br/> |MCU の URI の種類です。 詳細については、 [UriTypes テーブル](uritypes.md)を参照してください。 <br/> |
