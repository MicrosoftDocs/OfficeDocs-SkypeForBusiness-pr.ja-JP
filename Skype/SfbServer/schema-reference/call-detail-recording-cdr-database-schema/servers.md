# <a name="servers-table"></a>[サーバー] テーブル
 
[サーバー] テーブルは、さまざまなサーバーに関する情報を格納するサポート テーブルです。 テーブル内の各レコードは、1 つのサーバーを表します。
  
|**列**|**データ型**|**キー/インデックス**|**詳細**|
|:-----|:-----|:-----|:-----|
|**ServerId** <br/> |int  <br/> |Primary  <br/> |このサーバーを識別する一意の番号。  <br/> |
|**ServerFQDN** <br/> |nvarchar(256)  <br/> | <br/> |サーバー FQDN。  <br/> |
