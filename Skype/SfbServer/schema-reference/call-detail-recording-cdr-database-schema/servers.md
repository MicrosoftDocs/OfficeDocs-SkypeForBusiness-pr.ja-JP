# <a name="servers-table"></a>Servers テーブル
 
サーバーのテーブルは、さまざまなサーバーに関する情報を格納するサポート テーブルです。 テーブル内の各レコードは、1 つのサーバーを表します。
  
|**列**|**データ型**|**キーまたはインデックス**|**詳細**|
|:-----|:-----|:-----|:-----|
|**ServerId** <br/> |int  <br/> |Primary  <br/> |このサーバーを識別する一意の番号です。  <br/> |
|**ServerFQDN** <br/> |nvarchar(256)  <br/> | <br/> |サーバーの FQDN です。  <br/> |
