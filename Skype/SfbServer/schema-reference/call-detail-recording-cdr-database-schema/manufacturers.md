# <a name="manufacturers-table-in-skype-for-business-server-2015"></a>ビジネス サーバー 2015 の Skype の製造元テーブル
 
メーカーはテーブルをサポートします。 各レコードは、1 つのデバイス (机上電話) の製造元に関する情報を格納します。
  
|**列**|**データ型**|**キー/インデックス**|**詳細**|
|:-----|:-----|:-----|:-----|
|**ManufacturerId** <br/> |int  <br/> |Primary  <br/> |この製造元を識別する一意の番号です。  <br/> |
|**[製造元]** <br/> |nvarchar(256)  <br/> | <br/> |製造元の名前です。  <br/> |
