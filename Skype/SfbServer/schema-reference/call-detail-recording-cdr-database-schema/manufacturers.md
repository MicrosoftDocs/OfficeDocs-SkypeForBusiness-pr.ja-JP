# <a name="manufacturers-table-in-skype-for-business-server-2015"></a>Skype for Business Server 2015 の製造元テーブル
 
製造業者テーブルは、サポートされているテーブルです。 各レコードには、1つのデバイス (卓上電話) の製造元に関する情報が格納されます。
  
|**列**|**データ型**|**キー/インデックス**|**詳細**|
|:-----|:-----|:-----|:-----|
|**Manufacturerid]** <br/> |int  <br/> |Primary  <br/> |この製造元を特定する固有の番号。  <br/> |
|**[製造元]** <br/> |nvarchar(256)  <br/> | <br/> |メーカー名。  <br/> |
