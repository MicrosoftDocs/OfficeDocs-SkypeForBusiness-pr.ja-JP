# <a name="devicedriver-table"></a>DeviceDriver テーブル
 
早い段階はテーブルをサポートします。 デバイスのレンダリングまたは各レコードをいずれかのキャプチャ デバイスで使用されているドライバーを表します。
  
|**列**|**データ型**|**キー/インデックス**|**詳細**|
|:-----|:-----|:-----|:-----|
|**DeviceDriverKey** <br/> |int  <br/> |Primary  <br/> |このデバイス ドライバーのレコードを識別する一意の番号です。  <br/> |
|**DeviceDriver** <br/> |varchar(256)  <br/> |一意  <br/> |デバイス ドライバーの名前です。  <br/> |
