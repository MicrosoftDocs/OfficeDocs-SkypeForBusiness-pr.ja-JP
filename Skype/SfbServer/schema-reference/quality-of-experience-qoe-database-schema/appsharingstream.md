# <a name="appsharingstream-table"></a>AppSharingStream テーブル
 
AppSharingStream テーブルには、アプリケーション共有のストリームが使用するネットワークの QoE 測定値が含まれます。 この表は、Microsoft Lync Server 2013 で導入されました。
