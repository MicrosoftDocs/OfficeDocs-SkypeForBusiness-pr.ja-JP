# <a name="networkconfigurationsettings-view"></a>NetworkConfigurationSettings ビュー
 
NetworkConfigurationSettings ビューでは、ネットワーク構成に関する情報を格納します。 このビューは、Microsoft Lync Server 2013 で導入されました。
  
|**列**|**データ型**|**詳細**|
|:-----|:-----|:-----|
|SubnetIP  <br/> |int  <br/> |サブネットの IP アドレス (IPv4 のアドレスだけがサポートされます)。  <br/> |
|UserSiteName  <br/> |nvarchar (128)  <br/> |ユーザーのサイトの名前です。  <br/> |
|RegionName  <br/> |nvarchar (128)  <br/> |ユーザーの国/地域の名前です。  <br/> |
