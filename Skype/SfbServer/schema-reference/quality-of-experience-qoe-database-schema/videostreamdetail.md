# <a name="videostreamdetail-view"></a>VideoStreamDetail ビュー
 
VideoStreamDetail ビューには、データベース内の各ビデオ ストリームに関する情報が格納されています。 このビューは、Microsoft Lync Server 2013 で導入されました。
