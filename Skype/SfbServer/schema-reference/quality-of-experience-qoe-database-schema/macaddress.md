# <a name="macaddress-table"></a>MacAddress テーブル
 
MacAddress テーブルは補助的なテーブルです。個々のレコードが 1 つのソースを表します。
  
|**列**|**データ型**|**キー/インデックス**|**詳細**|
|:-----|:-----|:-----|:-----|
|**MacAddressKey** <br/> |int  <br/> |Primary  <br/> |MAC アドレスを識別する一意の番号。  <br/> |
|**MacAddress** <br/> |varchar(256)  <br/> |一意  <br/> |MAC アドレス文字列。  <br/> |
