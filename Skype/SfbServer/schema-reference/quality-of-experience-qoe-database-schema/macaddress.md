# <a name="macaddress-table"></a>MacAddress テーブル
 
MacAddress はテーブルをサポートします。 各レコードは、1 つのソースを表します。
  
|**列**|**データ型**|**キー/インデックス**|**詳細**|
|:-----|:-----|:-----|:-----|
|**MacAddressKey** <br/> |int  <br/> |Primary  <br/> |Mac アドレスを識別する一意の番号です。  <br/> |
|**MacAddress** <br/> |varchar(256)  <br/> |一意  <br/> |Mac アドレスの文字列です。  <br/> |
