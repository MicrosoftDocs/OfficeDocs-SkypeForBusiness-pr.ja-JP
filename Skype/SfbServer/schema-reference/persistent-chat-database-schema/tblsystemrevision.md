# <a name="tblsystemrevision"></a>tblSystemRevision
 
tblSystemRevision には、複数の管理者クライアント間で一貫性を確保するために、tblAdminLock テーブルと共に使用されるリビジョン番号が含まれています。
  
**行**

|**列**|**種類**|**説明**|
|:-----|:-----|:-----|
|sysRevision  <br/> |bigint (null ではない)  <br/> |リビジョン番号。  <br/> |
