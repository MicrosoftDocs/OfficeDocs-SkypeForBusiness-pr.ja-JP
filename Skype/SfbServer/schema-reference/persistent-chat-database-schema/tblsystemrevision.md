# <a name="tblsystemrevision"></a>tblSystemRevision
 
tblSystemRevision には、tblAdminLock テーブルで複数の管理者クライアントの間で一貫性を達成するために使用されるリビジョン番号が含まれています。
  
**列**

|**列**|**型**|**説明**|
|:-----|:-----|:-----|
|sysRevision  <br/> |bigint 型の値、null でないです。  <br/> |リビジョン番号です。  <br/> |
