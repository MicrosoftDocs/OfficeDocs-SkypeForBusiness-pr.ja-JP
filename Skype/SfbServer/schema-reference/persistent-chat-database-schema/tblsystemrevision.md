# <a name="tblsystemrevision"></a>tblSystemRevision
 
tblSystemRevision には、複数の管理者クライアント間で一貫性を実現するために tblAdminLock テーブルと一緒に使用されるリビジョン番号が含まれる。
  
**列**

|**列**|**型**|**説明**|
|:-----|:-----|:-----|
|sysRevision  <br/> |NULL でない bigint  <br/> |改訂番号  <br/> |
