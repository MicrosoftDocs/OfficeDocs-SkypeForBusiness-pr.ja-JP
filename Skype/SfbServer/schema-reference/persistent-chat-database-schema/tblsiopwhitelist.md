# <a name="tblsiopwhitelist"></a>tblSiopWhiteList
 
tblSiopWhiteList は、ノードと関連付けることができる登録済みのアドインの一覧です。
  
**列**

|**列**|**タイプ**|**説明**|
|:-----|:-----|:-----|
|siopID  <br/> |GUID では、null でないです。  <br/> |アドインの GUID です。  <br/> |
|siopName  <br/> |nvarchar (50) では、null でないです。  <br/> |アドインの表示名です。  <br/> |
|siopUrl  <br/> |nvarchar (255)、null でないです。  <br/> |アドインの URL です。  <br/> |
   
**キー**

|**列**|**説明**|
|:-----|:-----|
|siopID  <br/> |プライマリ ・ キーです。  <br/> |
