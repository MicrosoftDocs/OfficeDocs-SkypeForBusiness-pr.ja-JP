<span data-ttu-id="318d7-105">エンタープライズ音声通話管理機能は、着信通話のルーティングと応答の方法を制御します。</span><span class="sxs-lookup"><span data-stu-id="318d7-105">Enterprise Voice call management features control how incoming calls are routed and answered.</span></span> <span data-ttu-id="318d7-106">Skype for Business Server には、次のような通話管理機能が用意されています。</span><span class="sxs-lookup"><span data-stu-id="318d7-106">Skype for Business Server provides the following call management features:</span></span>
