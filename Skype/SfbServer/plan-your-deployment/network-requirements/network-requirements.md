## <a name="managing-quality-of-service"></a><span data-ttu-id="af8c7-589">サービスの品質の管理</span><span class="sxs-lookup"><span data-stu-id="af8c7-589">Managing Quality of Service</span></span>

<span data-ttu-id="af8c7-591">サービスの品質 (QoS) は、一部の組織で使用されるネットワーク テクノロジで、音声およびビデオ通信に最適なエンド ユーザー エクスペリエンスを提供するのに役立ちます。</span><span class="sxs-lookup"><span data-stu-id="af8c7-591">Quality of Service (QoS) is a networking technology that is used in some organizations to help provide an optimal end-user experience for audio and video communications.</span></span> <span data-ttu-id="af8c7-592">QoS は、帯域幅が制限されているネットワークで最も頻繁に使用されます。多数のネットワーク パケットが、非常に少量の使用可能な帯域幅と競合する場合、QoS を使用すると、管理者は音声またはビデオ データを転送するパケットに高い優先順位を割り当てできます。</span><span class="sxs-lookup"><span data-stu-id="af8c7-592">QoS is most frequently used on networks where bandwidth is limited: with a large number of network packets competing for a fairly small amount of available bandwidth, QoS enables administrators to assign higher priorities to packets carrying audio or video data.</span></span> <span data-ttu-id="af8c7-593">これらのパケットの優先順位を高くすることで、音声およびビデオ通信は、ファイル転送、Web ブラウズ、データベース バックアップなどの関係するネットワーク セッションよりも、高速で中断の少ない処理で完了する可能性が高くなります。</span><span class="sxs-lookup"><span data-stu-id="af8c7-593">By giving these packets a higher priority, audio and video communications are likely to complete faster, and with less interruption, than network sessions involving things such as file transfers, web browsing, or database backups.</span></span> <span data-ttu-id="af8c7-594">これは、ファイル転送またはデータベース バックアップに使用されるネットワーク パケットに"ベスト ベスト ベスト" 優先順位が割り当てられるためです。</span><span class="sxs-lookup"><span data-stu-id="af8c7-594">That's because network packets used for file transfers or database backups are assigned a "best effort" priority.</span></span>
