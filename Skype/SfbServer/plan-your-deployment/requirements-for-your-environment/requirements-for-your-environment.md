# <a name="requirements-for-your-skype-for-business-2015-environment"></a>Skype for Business 2015 環境の要件
 
**概要:** Skype for Business Server 2015 のシステム要件と、ネットワークまたはインフラストラクチャに関する要件を扱ったトピックの概要を示します。Skype for Business Server 2015 をインストールおよび展開するために何かをインストールまたは構成する必要がある場合は、ここで説明します。
  
これは、Skype for Business Server 2015 の展開に関する要件に関する記事のランディングページです。 これには、サーバー要件と、ドメインとネットワーク構造、証明書を含む、準備する必要がある一般的な事項が含まれます。
  
## <a name="whats-in-this-topic"></a>このトピックの内容

「環境の要件」セクションの2つの記事では、組織のサーバー環境で Skype for Business Server 2015 への展開またはアップグレードを成功させるために必要なものとその両方を簡単に説明します。
  
### <a name="server-requirements"></a>サーバー要件

[Skype For Business server 2015 のサーバー要件](server-requirements.md)は、展開で使用する必要がある個別のサーバーでの準備に必要なものです。
  
### <a name="environmental-requirements"></a>環境要件

Skype for [Business server 2015 の環境要件](environmental-requirements.md)は、Skype For business server 2015 を展開する前に、インフラストラクチャで準備しておく必要があるさまざまな点です。
