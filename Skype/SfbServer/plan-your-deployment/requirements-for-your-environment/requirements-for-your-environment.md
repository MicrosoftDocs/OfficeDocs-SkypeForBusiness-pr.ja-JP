# <a name="requirements-for-your-skype-for-business-2015-environment"></a>Skype for Business 2015 環境の要件
 
**概要:** Skype for Business Server 2015 のシステム要件およびネットワーク要件またはインフラストラクチャ要件に関するトピックの概要。 Skype for Business Server 2015 のインストールと展開の前提条件として何かをインストールまたは構成する必要がある場合は、ここに表示されます。
  
これは、Skype for Business Server 2015 の展開に関する要件に関する記事のランディング ページです。 これには、サーバー要件と、ドメインとネットワーク構造、証明書など、準備する必要がある一般的な内容の両方が含まれます。
  
## <a name="whats-in-this-topic"></a>このトピックの内容

環境の要件に関するセクションの 2 つの記事では、先に進み、組織のサーバー環境で Skype for Business Server 2015 を正常に展開またはアップグレードする前に、必要な要件や構成内容の概要を示します。
  
### <a name="server-requirements"></a>サーバーの要件

[Skype for Business Server 2015](server-requirements.md) のサーバー要件は、展開で使用する予定の個々のサーバーで準備する必要がある要件です。
  
### <a name="environmental-requirements"></a>環境要件

[Skype for Business Server 2015](environmental-requirements.md) の環境要件は、Skype for Business Server 2015 を展開する前にインフラストラクチャで準備する必要がある広範な要件です。
