### <a name="skype-for-business-web-app-and-skype-meetings-app"></a>Skype ビジネス Web アプリケーションおよび Skype 会議アプリケーション

Skype 会議アプリケーションは、直接の 10 の Windows アプリケーションのみのインストール時にブラウザーを使用し、10 の Windows ネイティブ アプリケーションとして実行です。 フロント エンド サーバーから配布を有効にする特定のサーバー構成が必要です。 ビジネス Web アプリケーションの Skype は、のように、このクライアントはビジネス ユーザー向けに、Skype で作成されたミーティングに参加するのには通常のクライアントを持たない組織外からゲストも可能です。
