# <a name="synchronize-users-to-the-cloud-and-assign-licenses"></a>ユーザーとクラウドを同期してライセンスを割り当てる
 
Office 365 にユーザーを実際に移動するには、前に、まず、クラウドに同期される、ライセンスを割り当てることを確認します。 この操作には、Office 365 管理センターを使用します。
  
### <a name="to-confirm-that-a-user-is-synchronized-with-office-365"></a>ユーザーが Office 365 と同期していることを確認するのには

1. Office 365 の管理ページを開きます (https://portal.office.com)。
    
2. 左側のナビゲーション ウィンドウで、[**ユーザー**]、[**アクティブなユーザー**] の順にクリックします。
    
3. [**ユーザーの検索**] をクリックして、ユーザーの名前を入力します。
    
4. ユーザーが表示されており、そのステータスが [**Active Directory と同期済み**] であることを確認します。
    
    ユーザーがまだ同期されていない場合は、3 時間以内に次の自動同期が実行されます。 または、すぐに同期を強制する可能性があります。 詳細については、[ディレクトリ同期の強制](https://msdn.microsoft.com/en-us/library/azure/JJ151771.aspx)を参照してください。
    
## <a name="to-assign-the-license"></a>ライセンスを割り当てるには

Office 365 のライセンスを割り当てるには、[ユーザーの PBX をクラウドのライセンスを割り当てまたは割り当て解除](https://support.office.com/article/Assign-or-unassign-a-Cloud-PBX-license-for-a-user-36c6d5a6-5ea8-4c44-9f18-fea33d5a847e)を参照してください。
