# <a name="configure-and-manage-skype-for-business-cloud-connector-edition"></a>Skype for Business Cloud Connector エディションの構成と管理
 
Skype をビジネス クラウド コネクタ版、オンライン ビジネスの電話システムと Skype のインターネット電話サービスを Office 365 (クラウド PBX) で、設置型音声インフラストラクチャの統合を有効にするのには最小限の設置型のトポロジを構成する方法について説明します。 
  
開始する前に[ビジネス クラウド コネクタ ・ エディションの Skype の計画](plan-skype-for-business-cloud-connector-edition.md)の前提条件を確認する必要があります。
  
> [!IMPORTANT]
> このトピックの手順は Cloud Connector Edition 1.4.1 以降のみに適用されます。 アップグレードした場合いないまだクラウド コネクタ版 2.1 に、[クラウドのコネクタの新しいバージョンへのアップグレード](upgrade-to-a-new-version-of-cloud-connector.md)を参照してください。 インストール ファイルをダウンロードすることができます[https://aka.ms/CloudConnectorInstaller](https://aka.ms/CloudConnectorInstaller)。
