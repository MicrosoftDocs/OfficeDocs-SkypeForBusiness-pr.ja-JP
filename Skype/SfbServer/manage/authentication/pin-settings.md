# <a name="manage-pin-settings-in-skype-for-business-server"></a>Skype for Business Server で PIN 設定を管理する
 
**概要:** Skype for Business Server で PIN 設定を管理します。
  
Skype for Business Server で PIN を管理するには、以下のセクションの手順を使用します。
  
Skype for Business Server の PIN ポリシーは、Skype for Business Server コントロール パネルまたは Skype for Business Server 管理シェルから管理できます。 組織の PIN ポリシーを構成するには、次の手順を使用します。
  
- [Skype for Business Server で PIN ポリシー情報を表示する](view-pin-policy-information.md)
    
- [Skype for Business Server で新しい PIN ポリシーを作成する](create-a-new-pin-policy.md)
    
- [Skype for Business Server で既存の PIN ポリシーを変更する](modify-an-existing-pin-policy.md)
    
- [Skype for Business Server で PIN ポリシーを削除する](delete-a-pin-policy.md)
    
- [Skype for Business Server でユーザー単位の PIN ポリシーを割り当てる](assign-a-per-user-pin-policy.md)
    
次の手順を使用して、Skype for Business Server コントロール パネルの [ユーザー] セクションからユーザーのダイヤルイン会議 PIN を管理します。
  
- [Skype for Business Server でユーザーのダイヤルイン会議 PIN を設定する](set-a-user-s-dial-in-conferencing-pin.md)
    
- [Skype for Business Server でユーザー PIN 情報を表示する](view-user-pin-information.md)
    
- [Skype for Business Server でユーザー PIN をロックまたはロック解除する](lock-or-unlock-a-user-pin.md)
