## <a name="sign-in-to-skype-for-business-for-the-first-time"></a>初めてSkype for Businessにサインインする

サインイン情報は、通常、インストール時にSkype for Business構成されます。 ただし、クライアントを初めて使用Skype for Business、クライアントを手動で開始する必要がある場合があります。
  
### <a name="to-sign-in-for-the-first-time"></a>初めてサインインするには

1. 組織のネットワークにログオンします。
    
2. [すべての **プログラム**  >  **の開始] を**  >  **選択Skype for Business。**
    
    サインイン画面が表示されます。
    
    - サインイン アドレス ボックスが既に入力されている場合は、表示されているアドレスが正しいか確認します。
    
    - 正しく設定されていない場合、またはボックスが空の場合は、サインイン アドレスを入力します (これは通常、メール アドレスと同じです)。
    
    - 空のパスワード ボックスが表示されている場合は、パスワードを追加します。
    
3. [ **サインイン] を選択します**。

## <a name="sign-in-to-skype-for-business-with-a-smart-card"></a>スマート カードを使用Skype for Businessにサインインする

一部の組織では、2 要素認証と呼ばれる複数ステップのサインイン プロセスを使用して、ユーザーのセキュリティを強化しています。 このオプションを使用する必要がある場合は、サインインしてサインインする "スマート カード" が必要Skype for Business。 スマート カードには、物理カードまたは仮想カードを使用できます。
  
- **物理** クレジット カードのサイズについて。 ログイン時にスマート カード リーダーに挿入します。
    
- **仮想** 物理オブジェクトではなく、コンピューター上の特別なチップに書き込まれる電子識別子で、本質的にスマート カードがコンピューターに組み込まれます。 TPM (信頼済みプラットフォーム モジュール) Windows 8含むコンピューターでのみ使用できます。
    
### <a name="enroll-your-smart-card"></a>スマート カードを登録する

スマート カードでサインインする前に、カードを "登録" する必要があります。つまり、ユーザー資格情報をカードで識別する必要があります。 これは、カードが物理的か仮想かの場合です。 このプロセスは、管理者が既に実行Skype for Business Serverがあります。 それが行われたかどうか不明な場合は、それらを確認してください。
  
> [!NOTE]
> 各仮想スマート カードは、インストールされているデバイスにのみ関連付けられているので、使用するコンピューターごとに個別のカードを登録Windows 8必要があります。 
  
### <a name="to-manually-enroll-your-smart-card"></a>スマート カードを手動で登録するには

1. 実行中のコンピューターにログオンし、Skype for Businessします。
    
2. [Internet Explorerを使用して、組織の [証明機関 Web 登録] ページを参照します。 
    
    このリソースSkype for Business Serverまだ持ってない場合は、管理者にこのリソースの Web アドレスを確認します。 URL は次のように表示されます https://MyCA 。.[yourcompanyname].com/certsrv。
    
    > [!NOTE]
    > このサイトを使用しているInternet Explorer 10互換モードでこの Web サイトを表示する必要がある場合があります。 
  
3. 認定ページにログオンするように求めるメッセージが表示されたら、(コンピューターの管理者ではなく) ドメイン アカウントを使用してログオンします。
    
4. Web サイトの [ようこそ] ページで、[証明書の **要求] を選択します**。
    
5. [高度 **な要求] を選択します**。
    
6. [この **CA に要求を作成して送信する] を選択し、[** 次へ] を **クリックします**。
    
7. これで、スマート カード登録ステーションというページが表示されます。 要求を承認して、ActiveXコントロールをインストールし、次のように [高度な証明書要求] フォームに入力します。
    
    a. [ **証明書テンプレート] ドロップダウン リスト** から **[Smartcard user]** を選択します。
    
    b. [新 **しいキー セットの作成] を選択します**。
    
    c. スマート カードのラベルの製造元情報を検索し **、CSP** ドロップダウン リストからその製造元を選択します。
    
    d. まだ選択されていない場合は、[要求形式] として **[CSP]** を選択します。
    
    e. まだ **選択されていない** 場合は、[ハッシュ アルゴリズム] ドロップダウン リストから sha1 を選択します。
    
    f. 証明書に認識する名前を付け、[送信] を **クリックします**。
    
8. 次に、登録ステーションに接続されているカード リーダーに空白のスマート カードを挿入し、[登録] を **クリックします**。
    
9. プロンプトが表示されたら、個人識別番号 (PIN) を入力し **、[OK] をクリックします**。
    
    > [!NOTE]
    > テクニカル サポート担当者が、スマート カードの登録に使用する特別な PIN を提供していない場合は、既定のスマート カード PIN 値を使用12345678。 
  
10. スマート カードを初めて使用した場合に PIN を強制的に変更するオプションを選択します。
    
11. 次に、登録ステーションに接続されているカード リーダーに空白のスマート カードを挿入し、[登録] を **クリックします**。
    
12. プロンプトが表示されたら、個人識別番号 (PIN) を入力し **、[OK] をクリックします**。
    
    > [!NOTE]
    > テクニカル サポート担当者が、スマート カードの登録に使用する特別な PIN を提供していない場合は、既定のスマート カード PIN 値を使用12345678。 
  
13. スマート カードを初めて使用した場合に PIN を強制的に変更するオプションを選択します。
    
14. **[OK]** をクリックして、表示される証明書に情報が含されていないことを確認します。
    
15. 証明書が発行されたという通知が表示されたら、[この証明書のインストール] をクリックして登録プロセスを完了します。
