# <a name="call-diagnostic-reports-per-user-in-skype-for-business-server"></a>Skype のビジネス サーバーの (ユーザー) ごとに診断レポートを呼び出す
 
**の概要:** ビジネス サーバーの Skype で使用される、ユーザーごとのコールの診断レポートについて説明します。
  
通話診断レポートは、失敗したピアツーピア セッションと会議セッションについて、ユーザーごとの情報を提供します。現時点では、ただ 1 つのレポートが存在します。
