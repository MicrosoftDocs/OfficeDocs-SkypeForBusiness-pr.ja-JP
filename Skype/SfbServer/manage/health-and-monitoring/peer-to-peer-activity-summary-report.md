# <a name="peer-to-peer-activity-summary-report-in-skype-for-business-server"></a>[ピアツーピア アクティビティの概要] レポート (Skype for Business Server
 
**概要:** 詳細については、「ピアツーピア アクティビティの概要レポート」を参照Skype for Business Server。
  
ピアツーピア アクティビティ概要レポートでは、ピアツーピア通信セッションの概要が示されます。 ピアツーピア セッションは通常、2 人のユーザーを含むだけで、電話会議サービスを使用Skype for Business Server必要とします。 一方、電話会議には通常、2 人を超えるユーザーが参加し、会議サービスをSkype for Business Server必要があります。 会議アクティビティは、会議概要レポートで報告されます。
  
ピアツーピア アクティビティ概要レポートは、次のような質問に答えるときに役立ちます。
  
- 通常、ユーザーはピアツーピア インスタント メッセージを 1 日に何通送信しますか。
    
- アプリケーション共有とファイル転送機能を実際に利用Skype for Business Serverユーザーはいますか?
    
- ユーザーが、ネットワークの速度が 1 日の特定の時刻で遅くなるようだと訴えています。これらの時間帯でピアツーピアの音声ビデオセッションに費やされた時間は何分ですか。
    
## <a name="accessing-the-peer-to-peer-activity-summary-report"></a>ピアツーピア アクティビティ概要レポートにアクセスする

監視レポートのホーム ページからピアツーピア アクティビティ概要レポート にアクセスします。 次のいずれかの[指標を](im-report.md)クリックして、Skype for Business Serverでピアツーピア IM レポートを開きます。
  
- ピアツーピア IM セッションの合計数
    
- ピアツーピア IM メッセージの合計数
    
同様に、ピアツーピア音声およびビデオ レポートを開くには、次のいずれかの指標をクリックします。
  
- ピアツーピア音声セッションの合計数
    
- ピアツーピア音声セッションの合計時間 (分)
    
- ピアツーピア音声セッションの合計数
    
- ピアツーピア音声セッションの合計時間 (分)
    
## <a name="making-the-best-use-of-the-peer-to-peer-activity-summary-report"></a>ピアツーピア アクティビティ概要レポートを最大限に活用する

ピアツーピア アクティビティ概要レポートの下部には、ピアツーピア IM セッションの合計数やピアツーピア IM メッセージの合計数などの、合計の指標が表示されます。これにより、レポート本文にある詳細情報の概要が示されます。
  
## <a name="filters"></a>フィルター

フィルターは、細かく絞り込んだデータ セットを返したり、返されたデータをさまざま方法で表示したりする方法として利用できます。たとえば、ピアツーピア アクティビティ概要レポートでは、データをグループ化する方法を選択できます。この場合は、時間、日、週、または月を基準にアクティビティがグループ化されます。
  
次の表に、ピアツーピア アクティビティ概要レポートで使用できるフィルターを示します。
  
**ピアツーピア アクティビティ概要レポートのフィルター**

|**名前**|**説明**|
|:-----|:-----|
|**From** <br/> |時間範囲の開始日と開始時刻。データを時間単位で表示するには、次のように開始日と開始時刻の両方を入力します。  <br/> 7/17/12015 1:00 PM  <br/> 開始時刻を入力しないと、レポートは自動的に指定日の午前 12:00 に開始します。データを日単位で表示するには、次のように日付のみを入力します。  <br/> 7/17/12015  <br/> 週単位または月単位で表示するには、表示する週または月の任意の日付を入力します (その週または月の最初の日である必要はありません)。  <br/> 7/13/2015  <br/> 週は、常に日曜日から土曜日までです。  <br/> |
|**To** <br/> |時間範囲の終了日と終了時刻。データを時間単位で表示するには、次のように終了日と終了時刻の両方を入力します。  <br/> 7/17/12015 1:00 PM  <br/> 終了時刻を入力しないと、レポートは自動的に指定日の午前 12:00 に終了します。データを日単位で表示するには、次のように日付のみを入力します。  <br/> 7/17/12015  <br/> 週単位または月単位で表示するには、表示する週または月の任意の日付を入力します (その週または月の最初の日である必要はありません)。  <br/> 7/13/2015  <br/> 週は、常に日曜日から土曜日までです。  <br/> |
|**間隔** <br/> | 時間間隔です。次のいずれかを選択します。 <br/>  時間単位 (最大 25 時間の表示が可能) <br/>  日単位 (最大 31 日の表示が可能) <br/>  週単位 (最大 12 週の表示が可能) <br/>  月単位 (最大 12 か月の表示が可能) <br/>  入力した開始日と終了日が選択した間隔で使用できる値の最大数を超える場合は、最大数の値 (開始日からカウント) のみが表示されます。 たとえば、開始日が 7/17/12015 で終了日が 2015 年 2 月 28 日の日間隔を選択すると、2015 年 8 月 7 日から 12 月 7 日の午前 8 時から 12 月 7 日の午前 9 時から 12 時 00 分 (合計 3 日分) のデータが表示されます。 <br/> |
