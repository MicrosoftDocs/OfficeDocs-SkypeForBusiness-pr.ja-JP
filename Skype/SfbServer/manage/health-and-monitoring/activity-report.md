# <a name="conference-activity-report-in-skype-for-business-server"></a>Skype for Business Server の電話会議動作状況レポート
 
**概要:** Skype for Business Server で使用される電話会議動作状況レポートについて説明します。
  
電話会議動作状況レポートを使用すると、たとえば 1 日にいくつの電話会議が開催されているかや、電話会議がいつ開催されているかといった質問に簡単に答えることができます。 このような情報はそれ自体が有用であるだけでなく、トラブルシューティング ツールとしても役立ちます。 たとえば、昼頃になるとネットワークが著しく遅くなるという苦情がユーザーから寄せられているとします。 電話会議のアクティビティレポートでは、次のような理由が考えられます。電話会議の時間は、10:00 AM ~ 2:00 PM の間でいつでもスケジュールされています。
  
ネットワークが遅いせいで問題が起きている場合は、一部の電話会議の予定をトラフィックが少ない時間帯にずらしてもらうようにユーザーに働きかけることができます。
  
## <a name="accessing-the-conference-activity-report"></a>電話会議動作状況レポートにアクセスする

電話会議動作状況レポートには、次のいずれかの指標をクリックすると、 [Skype For Business Server の電話会議の概要レポート](conference-summary-report.md)からアクセスできます。
  
- 電話会議の合計数
    
- 参加者の合計数
    
## <a name="making-the-best-use-of-the-conference-activity-report"></a>電話会議動作状況レポートを最大限に活用する

既定では、電話会議動作状況レポートには指定した期間における電話会議の合計数 (たとえば、1 日あたりの電話会議の合計数や、1 日の 1 時間あたりの電話会議の合計数) が表示されます。ただし、その期間における参加者の合計数や、参加者の合計分数を表示することもできます。これを行うには、[パラメーターの表示/非表示] ボタンをクリックしてフィルター オプションを表示し、[報告元] ドロップダウン リストから次のいずれかを選択します。
  
- 参加者数
    
- 参加者の分数
    
- 電話会議数
    
## <a name="filters"></a>フィルター

フィルターは、細かく絞り込んだデータ セットを返したり、返されたデータをさまざま方法で表示したりする方法として利用できます。次の表に、電話会議動作状況レポートで使用できるフィルターを示します。
  
**電話会議動作状況レポートのフィルター**

|**名前**|**説明**|
|:-----|:-----|
|**From** <br/> |時間範囲の開始日と開始時刻。データを時間単位で表示するには、次のように開始日と開始時刻の両方を入力します。  <br/> 7/7/2015 1:00 PM  <br/> 開始時刻を入力しないと、レポートは自動的に指定日の午前 12:00 に開始します。データを日単位で表示するには、次のように日付のみを入力します。  <br/> 7/7/2015  <br/> 週単位または月単位で表示するには、表示する週または月の任意の日付を入力します (その週または月の最初の日である必要はありません)。  <br/> 7/3/2015  <br/> 週は、常に日曜日から土曜日までです。  <br/> |
|**To** <br/> |時間範囲の終了日と終了時刻。データを時間単位で表示するには、次のように終了日と終了時刻の両方を入力します。  <br/> 7/7/2015 1:00 PM  <br/> 終了時刻を入力しないと、レポートは自動的に指定日の午前 12:00 に終了します。データを日単位で表示するには、次のように日付のみを入力します。  <br/> 7/7/2015  <br/> 週単位または月単位で表示するには、表示する週または月の任意の日付を入力します (その週または月の最初の日である必要はありません)。  <br/> 7/3/2015  <br/> 週は、常に日曜日から土曜日までです。  <br/> |
|**間隔** <br/> | 時間間隔です。次のいずれかを選択します。 <br/>  時間単位 (最大 25 時間の表示が可能) <br/>  日単位 (最大 31 日の表示が可能) <br/>  週単位 (最大 12 週の表示が可能) <br/>  月単位 (最大 12 か月の表示が可能) <br/>  入力した開始日と終了日が選択した間隔で使用できる値の最大数を超える場合は、最大数の値 (開始日からカウント) のみが表示されます。 たとえば、開始日が7/7/2015 で、終了日が2/28/2015 の日単位の間隔を選択すると、8/7/2015 12:00 AM から 9/7/2015 12:00 AM までのデータが表示されます (つまり、合計で31日間のデータが表示されます)。 <br/> |
|[**報告元**] <br/> | レポートで使用する値を指定します。次の中から選択します。 <br/>  参加者数 <br/>  参加者の分数 <br/>  電話会議数 <br/> |
   
## <a name="metrics-for-conferences-by-pool"></a>電話会議 (プール別) の指標

次の表に、プールごとの電話会議動作状況レポートでの情報を示します。
  
**電話会議 (プール別) の指標**

|**名前**|**この項目での並べ替え**|**説明**|
|:-----|:-----|:-----|
|**Pool** <br/> |いいえ  <br/> |電話会議で使用されたレジストラー プールまたはエッジ サーバーの名前。  <br/> |
|**日付/時刻** <br/> |いいえ  <br/> |電話会議の開始日時。  <br/> |
|**合計** <br/> |いいえ  <br/> |参加者総数、参加者総時間 (分)、電話会議総数。  <br/> |

|**日付/時刻** <br/> |いいえ  <br/> |電話会議の開始日時。  <br/> |
