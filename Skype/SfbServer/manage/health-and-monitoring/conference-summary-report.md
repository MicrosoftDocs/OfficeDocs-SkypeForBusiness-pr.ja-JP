# <a name="conference-summary-report-in-skype-for-business-server"></a>会議の概要レポート (Skype for Business Server
 
**概要:** 詳細については、「会議の概要レポート」を参照Skype for Business Server。
  
電話会議の概要レポートは、オンライン電話会議セッションの全体像を示します。 通常、会議には 2 人を超えるユーザーが含まれるので、会議サービスの使用が必要です。 一方、ピアツーピア セッションには通常、2 人のユーザーが参加し、会議サービスを使用する必要Skype for Business Serverではありません。 ピアツーピア アクティビティは、Skype for Business Server のピアツーピア アクティビティの概要[レポートで報告されます](peer-to-peer-activity-summary-report.md)。
  
[会議の概要] レポートには、特定の期間中 (1 時間、毎日、毎週、毎月) 開催された会議の数だけでなく、それらの会議に参加したユーザーの総数と、一意の会議開催者の総数も示されます。
  
"一意" の開催者とは、少なくとも 1 つの会議をスケジュールするユーザーです。 たとえば、Pilar Ackerman が 1 件の会議を予定した場合、彼女は 1 名の一意の開催者としてカウントされます。 Ken Myer が 148 件の会議を予定した場合、彼も 1 名の一意の開催者としてカウントされます。 たとえば、次の表に 8 つの会議がスケジュールされているが、3 つの一意の開催者 (Ken Myer、Pilar Ackerman、David Ahs) を示します。
  
|**会議の開催者**|**会議の開催日時**|
|:-----|:-----|
|Ken Myer  <br/> |2015/7/7 10:00 AM  <br/> |
|David Ahs  <br/> |2015/7/7 10:00 AM  <br/> |
|Ken Myer  <br/> |2015/7/7 11:00 AM  <br/> |
|Pilar Ackerman  <br/> |2015/7/7 11:00 AM  <br/> |
|Ken Myer  <br/> |2015 年 7 月 7 日 13:00  <br/> |
|Pilar Ackerman  <br/> |2015 年 7 月 7 日 14:00  <br/> |
|Ken Myer  <br/> |2015/7/2 10:00 AM  <br/> |
|Pilar Ackerman  <br/> |2015/7/2 10:00 AM  <br/> |
   
電話会議の概要レポートは、音声/ビデオを利用した会議の数も示します。
  
## <a name="accessing-the-conference-summary-report"></a>電話会議の概要レポートへのアクセス方法

電話会議の概要レポートには、[監視レポート] ホーム ページからアクセスします。次の指標のどちらかをクリックすることで、電話会議動作状況レポートにドリルダウンできます。
  
- 電話会議の合計数
    
- 参加者の合計数
    
## <a name="making-the-best-use-of-the-conference-summary-report"></a>電話会議の概要レポートを最大限に活用する

会議概要レポートで使用されるほとんどの指標の合計値は、レポートの下部に表示されます。下にスクロールすると、指定した期間中に開催された会議の総数や、それらの会議に参加したユーザーの総数などの値が表示されます。 レポートの下部に合計されない指標の 1 つは、一意の会議開催者の合計です。 どうしてでしょうか? 理由の 1 つを次に示します。 1 か月分のデータを見ているとします。 1 日目には、34 人の一意の会議開催者がいました。2 日目には、27 人のユニークな会議開催者がいました。 つまり、この 2 日間に 61 人の一意の会議開催者がいましたか? 必ずしもそうではありません。 結局のところ、2 日目に会議を開催した 27 人全員が、1 日目に会議を開催した 34 人の中に含む可能性があります。 たとえば、この簡単なレポートでは、Ken Myer と Pilar Ackerman が 2015 年 7 月 7 日と 2015 年 7 月 2 日の両方で会議をスケジュールしました。
  
|**会議の開催者**|**会議の開催日時**|
|:-----|:-----|
|Ken Myer  <br/> |2015/7/7 10:00 AM  <br/> |
|David Ahs  <br/> |2015/7/7 10:00 AM  <br/> |
|Ken Myer  <br/> |2015/7/7 11:00 AM  <br/> |
|Pilar Ackerman  <br/> |2015/7/7 11:00 AM  <br/> |
|Ken Myer  <br/> |2015 年 7 月 7 日 13:00  <br/> |
|Pilar Ackerman  <br/> |2015 年 7 月 7 日 14:00  <br/> |
|Ken Myer  <br/> |2015/7/2 10:00 AM  <br/> |
|Pilar Ackerman  <br/> |2015/7/2 10:00 AM  <br/> |
   
会議を開催した一意のユーザーの総数を把握するには、期間を変更します。たとえば、日単位ではなく月単位でデータを調べます。
  
## <a name="filters"></a>フィルター

フィルターは、細かく絞り込んだデータ セットを返したり、返されたデータをさまざま方法で表示したりする方法として利用できます。たとえば、電話会議の概要レポートでは、データをグループ化する方法を選択できます。その場合は、電話会議が時間、日、週、または月の単位でグループ化されます。
  
次の表に、電話会議の概要レポートで使用できるフィルターを示します。
  
**電話会議の概要レポートのフィルター**

|**[名前]**|**説明**|
|:-----|:-----|
|**From** <br/> |時間範囲の開始日と開始時刻。データを時間単位で表示するには、次のように開始日と開始時刻の両方を入力します。  <br/> 2015 年 7 月 7 日 13:00  <br/> 開始時刻を入力しないと、レポートは自動的に指定日の午前 12:00 に開始します。データを日単位で表示するには、次のように日付のみを入力します。  <br/> 7/7/2015  <br/> 週単位または月単位で表示するには、表示する週または月の任意の日付を入力します (その週または月の最初の日である必要はありません)。  <br/> 7/3/2015  <br/> 週は、常に日曜日から土曜日までです。  <br/> |
|**To** <br/> |時間範囲の終了日と終了時刻。データを時間単位で表示するには、次のように終了日と終了時刻の両方を入力します。  <br/> 2015 年 7 月 7 日 13:00  <br/> 終了時刻を入力しないと、レポートは自動的に指定日の午前 12:00 に終了します。データを日単位で表示するには、次のように日付のみを入力します。  <br/> 7/7/2015  <br/> 週単位または月単位で表示するには、表示する週または月の任意の日付を入力します (その週または月の最初の日である必要はありません)。  <br/> 7/3/2015  <br/> 週は、常に日曜日から土曜日までです。  <br/> |
|**間隔** <br/> | 時間間隔です。次のいずれかを選択します。 <br/>  時間単位 (最大 25 時間の表示が可能) <br/>  日単位 (最大 31 日の表示が可能) <br/>  週単位 (最大 12 週の表示が可能) <br/>  月単位 (最大 12 か月の表示が可能) <br/>  開始日と終了日が、選択されている期間内で許容される値の最大数を超えると、(開始日から起算した) 値の最大数のみが表示されます。 たとえば、開始日が 7/7/2015 で終了日が 2015 年 2 月 28 日の日間隔を選択した場合、2015 年 8 月 7 日 12:00 AM ~ 9/7/2015 12:00 AM (合計 31 日分) のデータが表示されます。 <br/> |

**電話会議の概要レポートの指標**

|:-----|:-----|:-----|
|**毎時** <br/> **毎日** <br/> **毎週** <br/> **毎月** <br/> |不要  <br/> |フィルター ツール バーで選択した期間を示します。 場合によっては、特定の期間をクリックして、その期間の詳細情報を表示することもできます。 たとえば、日次間隔を使用している場合、2015 年 7 月 7 日をクリックすると、その日付のユーザー登録アクティビティの 1 時間に 1 回の内訳が表示されます。  <br/> |
|[**電話会議の合計数**] <br/> |不要  <br/> |実施された電話会議の総数です (会議の種類は区別されません)。この項目をクリックすると、選択した期間に対する電話会議動作状況レポートが表示されます。  <br/> |

|[**音声ビデオ会議の合計数**] <br/> |不要  <br/> |音声またはビデオを利用した会議の総数です。  <br/> |
