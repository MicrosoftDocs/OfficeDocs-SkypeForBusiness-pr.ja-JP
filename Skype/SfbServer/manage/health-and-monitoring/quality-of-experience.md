# <a name="quality-of-experience-qoe-in-skype-for-business-server-2015"></a>Skype for Business Server 2015 の QoE (Quality of Experience)
 
**の概要:**ビジネス サーバー 2015 の Skype でのエクスペリエンス (QoE) の品質を管理します。
  
QoE (Quality of Experience) は、メディアの品質と、通話およびセッションに関係する参加者、デバイス名、ドライバー、IP アドレス、およびエンドポイントの種類についての情報を示す数値データを記録します。 ビジネス サーバー 2015 の Skype をインストールするときは、QoE のグローバル構成設定の定義済みコレクションもインストールされます。 QoE の設定を構成するには、このセクションのトピックに従ってください。

- [ビジネス サーバー 2015 の Skype での経験の質を有効にします。](enable-qoe.md)
    
- [ビジネス サーバー 2015 の Skype の高品質のエクスペリエンスの設定を変更します。](modify-qoe-settings.md)
