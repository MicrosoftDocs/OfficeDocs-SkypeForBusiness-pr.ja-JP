# <a name="peer-to-peer-im-report-in-skype-for-business-server"></a><span data-ttu-id="f24e1-103">Skype for Business Server のピアツーピア IM レポート</span><span class="sxs-lookup"><span data-stu-id="f24e1-103">Peer-to-Peer IM Report in Skype for Business Server</span></span>
