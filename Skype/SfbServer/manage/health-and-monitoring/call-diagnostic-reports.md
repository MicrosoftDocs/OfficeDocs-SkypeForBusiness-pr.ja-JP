# <a name="call-diagnostic-reports-in-skype-for-business-server"></a>Skype for Business Server の通話診断レポート
 
**概要:** Skype for Business Server で使用されるマルチユーザー通話診断レポートについて説明します。
  
通話診断レポートは、エラーが発生したピアツーピア セッションと電話会議セッションに関する概要情報と診断データを提供します。
  
## <a name="in-this-section"></a>このセクションの内容

- [Skype for Business Server の通話診断の概要レポート](summary-report.md) エラーが発生したピアツーピア セッションと電話会議セッションの概要を示します。 ピアツーピア セッションは、参加者が 2 人だけのセッションです。 会議セッションには、3 名以上の参加者が参加します。
    
- [Skype for Business Server のピアツーピア アクティビティ診断レポート](peer-to-peer-activity-diagnostic-report.md) エラーが発生したピアツーピア セッションの全体的な傾向ビューを提供します。 ピアツーピア セッションでは、2 人の参加者が参加します。
    
- [Skype for Business Server の電話会議診断レポート](conference-diagnostic-report.md) 失敗した会議セッションの全体的な傾向ビューと、各会議モダリティの傾向ビューを提供します。 会議セッションには、少なくとも 3 人の参加者が参加します。
    
- [Skype for Business Server](top-failures-report.md) のトップ エラー レポート最も頻繁に発生するエラーとその傾向の一覧を示します。
    
- [Skype for Business Server のエラー分布レポート](failure-distribution-report.md) 失敗したセッションの分析を提供します。
    
- [Skype for Business Server のエラー リスト レポート](failure-list-report.md) 失敗した会議に関係する個々の参加者に関する詳細情報を提供します。
    
- [Skype for Business Server の診断レポート](diagnostic-report.md) 失敗したセッションの診断およびトラブルシューティング情報 (SIP 応答コード、診断ヘッダー、および ID を含む) を提供します。
