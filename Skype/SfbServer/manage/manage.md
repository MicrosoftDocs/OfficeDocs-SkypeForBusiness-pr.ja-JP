# <a name="manage-skype-for-business-server"></a>Skype for Business Server の管理 

**概要:** これらのリンクを参照して、Skype for Business Server を管理および保守する方法について学習します。
  
Skype for Business Server は、エンタープライズ レベルのコラボレーション要件をサポートできるインスタント メッセージング (IM)、プレゼンス、会議、ビデオ、テレフォニー ソリューションを提供します。 このトピックでは、Skype for Business Server の展開を構成、監視、および保守する方法に関する情報へのリンクを提供します。 
  
これらの記事は、新しい情報が入り次第、またユーザーのフィードバックに基づいて随時更新されます。 このページは頻繁に更新されますので、頻繁に確認してください。
