# <a name="manage-enhanced-9-1-1-and-the-location-service-in-skype-for-busines-server"></a>拡張 9-1-1 と Skype でロケーション サービスをビジネス用のサーバーを管理します。

Skype ビジネス サーバーの強化された 9-1-1 (~ 9-1-1) ビジネス クライアント用の Skype からの呼び出しをサポートします。 ~ 9-1-1 の Skype ビジネス サーバーを構成する場合、ビジネス用の Skype から緊急呼び出しには、場所情報サービス データベースからの緊急応答ロケーション (ERL) 情報が含まれます。 場所のポリシーを管理するためにこの資料の手順を使用します。

> [!Note]
> ~ 9-1-1 と位置情報サービスなどの高度なエンタープライズ VoIP 機能を展開する方法の詳細については、[エンタープライズ VoIP 機能を高度な配置](../deploy/deploy-enterprise-voice/deploy-advanced-enterprise-voice-features.md)を参照してください。

ビジネス サーバーの Skype では、拡張 9-1-1 (~ 9-1-1) の機能とユーザーまたは連絡先の場所の設定に関連する設定を適用するのに場所のポリシーを使用できます。 緊急の呼び出しの動作は、どのような ~ 9-1-1 のユーザーが有効になっているかどうかと、その場合は、場所のポリシーが決定します。 など、どのような番号は、緊急の呼び出し (たとえば、米国の 911) を構成する要素、かどうか企業のセキュリティを自動的に通知するか、および通話をルーティングする方法を定義するのには、場所ポリシーを使用することができます。

ビジネス サーバーのコントロール パネルの Skype の**ネットワーク設定**のグループからの場所のポリシーを構成できます。 ビジネス サーバーのコントロール パネルの Skype からすることができます表示、作成、変更、または場所のポリシーを削除します。 場所のポリシーに関する情報を表示するのにには、次の手順を使用します。

5.  [**編集**] メニューの [**詳細の表示**] をクリックします。
 
    > [!NOTE]  
    > のみ、一度に 1 つの場所のポリシーに関する情報を表示できます。

グローバルと呼ばれる 1 つのポリシー、既定で存在してことはできませんを削除または名前です。 ただし、グローバル ポリシーを変更することができます。 このポリシーは、サイト ポリシーまたはユーザーごとのポリシーを作成する場合を除き、すべてのユーザーまたは連絡先に適用されます。 ユーザーごとのポリシーは、特定のユーザーに適用する必要があります。
