# Lync for iPhone と Lync for iPad の削除

 

_**トピックの最終更新日:** 2013-02-19_

Lync 2013 for iPhone または Microsoft Lync 2010 for iPad をデバイスから削除するには、次の手順を実行します。

1.  モバイル デバイスのホーム画面で、Lync アイコンをタップしたまま押さえます。

2.  タイルが揺れ始め、\[**X**\] が表示されると、\[**X**\] をタップしてアプリケーションを削除します。
