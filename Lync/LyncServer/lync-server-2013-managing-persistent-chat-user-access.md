# <a name="managing-persistent-chat-user-access-in-lync-server-2013"></a>Lync Server 2013 での常設チャットのユーザーアクセスの管理

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**トピックの最終更新日:** 2012-10-06_

次のトピックでは、Lync Server 2013 コントロールパネルおよび**常設チャット**ページを使用して、常設チャットでのユーザーアクセスを管理する方法について説明します。 常設チャットサーバーを管理するには、システムの常設チャット管理者の権限とアクセス許可を持っている必要があります。 常設チャットサーバーをインストールする場合は、必要な常設チャット管理者の権限とアクセス許可を持つ1人以上のユーザーを指定します。 管理者権限とアクセス許可を追加のユーザー アカウントに付与するには、**ユーザーとユーザー グループの管理**を使用します。 詳細については、「 [Lync Server 2013 でユーザーがカテゴリ、チャットルーム、およびユーザーの権限とアクセス許可を管理できるように](lync-server-2013-enabling-a-user-to-manage-categories-chat-rooms-and-user-rights-and-permissions.md)する」を参照してください。
