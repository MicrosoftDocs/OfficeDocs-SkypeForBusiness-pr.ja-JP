# Lync Server 2013 のサポートされるサーバーの併置

 

_**トピックの最終更新日:** 2012-06-29_

Lync Server 2013 は、一部のサーバーの役割と機能の併置をサポートします。併置できるサーバーの役割と機能はさまざまな要素により異なりますが、フロントエンド プールまたは Standard Edition サーバーが展開されているかどうかもその 1 つです。
