# ベスト プラクティス アナライザー実行の前提条件

 

_**トピックの最終更新日:** 2012-06-25_

Best Practices Analyzer を使用すると、問題を検出するために Lync Server 2013 環境をスキャンしてレポートを作成したり、現在または前回のスキャンの結果を表示したりできます。Best Practices Analyzer を実行して環境をスキャンする前に、スキャンを実行する予定のあるアカウントに必要な権限とアクセス許可があること、またインストールに関するすべての前提条件が満たされていることを確認する必要があります。
