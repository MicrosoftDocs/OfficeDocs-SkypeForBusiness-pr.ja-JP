# <a name="prerequisites-for-running-best-practices-analyzer-in-lync-server-2013"></a>Lync Server 2013 でベストプラクティスアナライザーを実行するための前提条件

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**トピックの最終更新日:** 2012-06-25_
