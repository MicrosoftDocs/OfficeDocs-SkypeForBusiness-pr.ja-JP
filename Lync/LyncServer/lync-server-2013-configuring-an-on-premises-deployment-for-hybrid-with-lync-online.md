# Lync Online と内部設置型のハイブリッド展開の構成

 

_**トピックの最終更新日:** 2014-05-28_

ハイブリッド展開とは、社内に所属するユーザーとオンラインに所属するユーザーが存在し、すべてのユーザーが user@contoso.com のような同じドメインを共有する展開のことです。このセクションでは、ハイブリッド展開に必要なアプリケーションの展開と、展開を有効にするための構成について説明します。
