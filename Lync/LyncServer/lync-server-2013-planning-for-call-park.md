# <a name="planning-for-call-park-in-lync-server-2013"></a>Lync Server 2013 でのコールパークの計画

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**トピックの最終更新日:** 2012-09-07_

Lync Server コールパークアプリケーションを使用すると、エンタープライズ Voip ユーザーは通話を保留にしてから、後で電話で取得することができます。 通話を保留したユーザーは、コール パークが提供するオービット番号をダイヤルして保留した通話を取得するか、インスタント メッセージングやポケットベルなどの外部の機能を使用して、他のユーザーに番号を知らせて取得させることができます。 このセクションには、コールパークに固有の計画に関する情報が含まれています。
