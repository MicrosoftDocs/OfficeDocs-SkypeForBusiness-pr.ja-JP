# <a name="apply-office-communications-server-2007-r2-updates"></a>Office Communications Server 2007 R2 の更新プログラムを適用する

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**最終更新日:** 2012-10-19_

Lync Server 2013 に移行する前に、Office Communications Server 2007 R2 環境に更新プログラムを適用する必要があります。 Office Communications Server 2007 R2 に関する最新情報については、「 **Communications server 2007 r2** **の更新プログラム**」を[http://go.microsoft.com/fwlink/p/?linkid=3052\&kbid=968802](http://go.microsoft.com/fwlink/p/?linkid=3052%26kbid=968802)参照してください。

Office Communications Server 2007 R2 の更新プログラムをインストールするには、「Microsoft サポート技術情報の記事968802、「Communications Server 2007 R2 の更新プログラム」に記載されている**方法1の累積サーバー更新インストーラ**の手順を実行することをお勧め[http://go.microsoft.com/fwlink/p/?linkid=3052\&kbid=968802](http://go.microsoft.com/fwlink/p/?linkid=3052%26kbid=968802)します。
