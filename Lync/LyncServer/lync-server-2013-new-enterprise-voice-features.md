# Lync Server 2013 の新しいエンタープライズ VoIP 機能

 

_**トピックの最終更新日:** 2013-05-01_

Lync Server 2013では、 エンタープライズ VoIP を拡張する新しいルーティングおよび通話管理機能をいくつか導入しています。

Lync Server 2013 は、 仲介サーバーとゲートウェイ間の複数のトランクをサポートします。 *トランク* とは、ポート番号と 仲介サーバー間およびポート番号とゲートウェイ間の論理的な関連付けです。つまり、1 つの 仲介サーバーが異なるゲートウェイへのトランクを複数持つことができ、1 つのゲートウェイが異なる仲介サーバーへのトランクを複数持つことができます。トランク間ルーティングを使用すると、 Lync Server 2013 で IP-PBX と公衆交換電話網 (PSTN) ゲートウェイの相互接続、または複数の IP-PBX システムの相互接続ができます。 Lync Server 2013 は、異なるテレフォニー システム間を結びつけるもの (すなわち、相互接続) として機能します。

Microsoft Lync Server 2013 を使用すると、着信の転送、同時呼び出し、ボイス メールの処理、および発信者番号の提示などの機能が向上します。これらの機能は、 エンタープライズ VoIP 通話の操作性を高めます。

  - [Lync Server 2013 の新しいトランク機能](lync-server-2013-new-trunk-feature.md)

  - [Lync Server 2013 の新しいトランク間機能](lync-server-2013-new-intertrunk-feature.md)

  - [Lync Server 2013 の新しい通話管理機能](lync-server-2013-new-call-management-features.md)
