# Lync Server 2013 アーカイブの管理

 

_**トピックの最終更新日:** 2012-10-10_

組織にアーカイブを展開する場合、展開中に初期構成を指定しますが、日常の管理におけるアーカイブ サポートの実装方法の変更や、組織の新しい要件への対応が必要になることもあります。たとえば、組織内の特定のサイト、プール、またはユーザーに対して異なるアーカイブ サポートのセットアップが必要になる場合があります。Lync Server 2013 に所属するユーザーの場合、これは、アーカイブ ポリシーと構成を作成およびカスタマイズすることによって行います。Microsoft Exchange の統合を使用する場合は、Exchange 2013 の設定を構成する必要があります。このセクションでは、アーカイブ展開を変更するための情報と手順について説明します。
