# Lync 対応電話および Lync Server 2013 での折り返し先の使用

 

_**トピックの最終更新日:** 2013-08-09_

Lync の折り返し先機能を使用すると、携帯電話、“固定電話”、公衆交換電話網 (PSTN) に接続しているその他のデバイスなどを使用して、会議の音声部分に参加できます。Lync を使用して会議に参加しようとすると、通常は \[**音声会議への参加方法**\] ダイアログ ボックスが表示されます。

![Lync を使用して会議に参加する音声ウィンドウのスクリーンショット](images/Dn383570.e28f17f0-9f17-44ef-b893-f4ef132f47ac(OCS.15).png "Lync を使用して会議に参加する音声ウィンドウのスクリーンショット")

\[**折り返し先**\] を選ぶと、ドロップダウン リストから電話番号を選ぶことができます。Lync Server 2013 から選んだ番号に電話がかけられ、その電話を使って会議の音声部分に参加できます。

ドロップダウン リストには、\[**Lync - オプション**\] ダイアログ ボックスの \[**電話**\] タブで入力した (携帯電話、自宅の電話、その他の電話などの) 電話番号が表示されます。

![Lync 電話セットアップ オプションのスクリーンショット](images/Dn383570.03d2f25d-49e2-47b4-b1e9-b1614fc0c11c(OCS.15).png "Lync 電話セットアップ オプションのスクリーンショット")

\[**電話**\] タブで電話番号を入力していないと、ドロップダウン ボックスに番号は表示されません。ただし、いつでも \[**新しい番号**\] をクリックして Lync Server から任意の電話番号に電話をかけることができます。

![Lync 会議参加音声案内ウィンドウのスクリーンショット](images/Dn383570.27f2ac7a-cc1c-465c-b145-202ad03af4f2(OCS.15).png "Lync 会議参加音声案内ウィンドウのスクリーンショット")

折り返し先機能は、Lync Server から PSTN の電話番号を呼び出し、用途に従って使用すればとても便利です。ただし、Lync Server から Lync 対応のエンドポイント (会議室の電話など) に電話をかけると、問題が発生する可能性があります。次に、発生する可能性がある問題と、2 つの解決策について説明します。

まず、折り返し先機能で Lync 対応の電話を使用した場合には、次のような問題が発生します。Ken Myer が Lync Server から自分の番号 1-206-555-1219 (Lync Server 対応の電話番号) に電話して会議に参加しようとしたとします。最初は会議に接続されますが、数秒後に Lync に「**通話が完了していません、または通話が終了しました。**」というメッセージが表示され、会議に参加できなくなります。

![Lync 電話会議ウィンドウのスクリーンショット](images/Dn383570.c2a81727-8751-41b5-946a-03a1b75b9d95(OCS.15).png "Lync 電話会議ウィンドウのスクリーンショット")

しかし、Lync の \[会話\] ウィンドウには矛盾する内容が表示されます。\[**参加者**\] の見出しに、Ken Myer の名前だけが表示されますが、ウィンドウのタイトル バーには会議の参加者が 4 人と表示されます。

また、他の会議参加者の \[会話\] ウィンドウには、Ken Myer が表示されません。

![Lync 参加者一覧ウィンドウのスクリーンショット](images/Dn383570.fa5990cf-2694-402c-ac06-946aa66b6837(OCS.15).png "Lync 参加者一覧ウィンドウのスクリーンショット")

しかし同時に、Ken Myer は自分の Lync 対応の電話を使用して通話の音声部分に参加できます。折り返し先機能は実際には機能していますが、ユーザー エクスペリエンスが最適ではありません。

この問題を解するには、少なくとも 2 つの方法があります。この (Ken Myer のような) 方法ですでに会議に参加している場合、通常は別のモダリティに参加すると問題を解決できます。たとえば、インスタント メッセージを送信すると、\[会話\] ウィンドウに自分を含むすべての会議参加者が表示されるようになります。

![Lync 会話および参加者一覧のスクリーンショット](images/Dn383570.9b5ff6d6-9f73-467c-99a7-ef3aa8bd7e7a(OCS.15).png "Lync 会話および参加者一覧のスクリーンショット")

この時点で、予想どおりの方法で会議に参加できることになります。

または会議への参加方法を変えることで、この問題を完全に回避できます。Lync Server から Lync Server 対応の電話に電話をかける必要がある場合は、まず音声接続なしで会議に参加します。この操作を実行するには、\[音声会議への参加方法\] ダイアログ ボックスが表示されたときに \[音声を使用しない\] を選びます。

![Lync 会議不参加音声ウィンドウのスクリーンショット](images/Dn383570.280a148d-cce5-4b02-87f9-9f78f17a81c1(OCS.15).png "Lync 会議不参加音声ウィンドウのスクリーンショット")

会議に問題なく接続したら、Lync Server 対応の電話を “招待” して会議に参加させることができます。この操作を実行するには、マウスを \[人\] アイコンに置いて \[**他の人を招待**\] をクリックします。

![Lync 追加の参加者を招待するウィンドウのスクリーンショット](images/Dn383570.69b81b29-d1d2-4ed3-acb6-e37dd18e3d86(OCS.15).png "Lync 追加の参加者を招待するウィンドウのスクリーンショット")

\[**IM の送信**\] ダイアログ ボックスが表示されます。ダイアログ ボックスのタイトル (実際にはインスタント メッセージを送信していない) は無視して、Lync 対応の電話の電話番号を入力します。

![IM を送信するダイアログ ボックスのスクリーンショット](images/Dn383570.cd67a3f0-06d8-41ba-a808-c067f64bec9f(OCS.15).png "IM を送信するダイアログ ボックスのスクリーンショット")

\[**OK**\] をクリックすると、Lync Server からダイアログ ボックスに入力した番号が呼び出されます。接続が完了すると、Lync 対応の電話からすべての音声機能を使用でき、すべての会議名簿を表示してやり取りできます。
