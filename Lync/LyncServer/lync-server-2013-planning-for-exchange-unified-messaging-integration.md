# Lync Server 2013 での Exchange ユニファイド メッセージング統合の計画

 

_**トピックの最終更新日:** 2016-12-08_

Lync Server 2013 は、ボイス メッセージングと電子メール メッセージングを単一のメッセージング インフラストラクチャに結合する Exchange ユニファイド メッセージング (UM) との統合をサポートします。 Microsoft Exchange Server 2007 Service Pack 1 (SP1) および Microsoft Exchange Server 2010 では、 Exchange ユニファイド メッセージング (UM) はインストールと構成が可能ないくつかの Exchange サーバーの役割の 1 つです。

Microsoft Exchange Server 2013 では、 Exchange UM は、Exchange メールボックス上のサービスとして機能します。 Lync Server 2013 エンタープライズ VoIP の展開では、ユニファイド メッセージングによって、ボイス メッセージングと電子メール メッセージングが単一ストアに統合されます。このストアは電話 (Outlook Voice Access) またはコンピューターから利用できます。ユニファイド メッセージングと Lync Server 2013 が連動することにより、エンタープライズ VoIP ユーザーは、通話応答、Outlook Voice Access、自動応答の各サービスを利用できます。

Microsoft Exchange Server 2013 のアーキテクチャの変更に関する詳細については、 Microsoft Exchange Server 2013 ドキュメントの「ボイスのアーキテクチャの変更」( [http://go.microsoft.com/fwlink/?linkid=266730\&clcid=0x411](http://go.microsoft.com/fwlink/?linkid=266730%26clcid=0x411)) を参照してください。

これらの機能が内部設置型の Exchange UM の展開でサポートされるようにするには、以下のサーバーのいずれかを実行する必要があります。

  - Microsoft Exchange Server 2007 Service Pack 1 (SP1) または最新 Service Pack

  - Microsoft Exchange Server 2010 または最新 Service Pack

  - Microsoft Exchange Server 2013
