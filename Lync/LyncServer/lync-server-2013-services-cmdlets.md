# <a name="services-cmdlets-in-lync-server-2013"></a>Lync Server 2013 でのサービスのコマンドレット

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**トピックの最終更新日:** 2012-06-20_

Microsoft Lync Server 2013 に付属するサーバーの役割およびサービスのコマンドレットを使用すると、Lync Server 2013 の展開の一部としてインストールされているさまざまなコンポーネントに関する情報を戻すことができます。 さらに、これらのコマンドレットを使用すると、Lync Server サービスの停止および開始が容易になります。
