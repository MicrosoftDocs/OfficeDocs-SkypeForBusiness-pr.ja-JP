# Lync Server 2013 の常設チャット サーバーに対する証明書要件

 

_**トピックの最終更新日:** 2012-10-03_

常設チャット サーバー をインストールするには、ファイルのアップロードとダウンロードのために 常設チャット Web サービスを実行している各サーバーに対して、Lync Server 2013 内部サーバーが使用するのと同じ CA から発行された証明書が必要です。特に外部の CA を使用している場合は、常設チャット のインストールを開始する前に、必要な証明書があることを確認してください。
