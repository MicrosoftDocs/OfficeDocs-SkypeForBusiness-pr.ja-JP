# Lync Server 2013 での従来の承認済みホストの削除 (オプション)

 

_**トピックの最終更新日:** 2012-09-05_

リモート通話コントロールを Lync Server の展開に移行する場合、従来の展開における SIP/CSTA ゲートウェイの従来の承認済みホスト エントリ (Lync Server では*信頼済みアプリケーション エントリ* と呼びます) を削除する必要があります。そのような展開から承認済みホスト エントリを削除するには、管理ツールをそれぞれ使用する必要があります。
