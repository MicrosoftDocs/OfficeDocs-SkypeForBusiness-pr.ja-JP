# <a name="remove-a-legacy-authorized-host-in-lync-server-2013-optional"></a>Lync Server 2013 での従来の承認済みホストの削除 (オプション)

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**トピックの最終更新日:** 2012-09-05_

リモート通話コントロールを Lync Server 展開に移行する場合は、従来の展開の SIP/CSTA ゲートウェイに対して、従来の承認済みホストエントリ (Lync Server の*信頼されたアプリケーションエントリ*) を削除する必要があります。 そのような展開から承認済みホスト エントリを削除するには、管理ツールをそれぞれ使用する必要があります。
