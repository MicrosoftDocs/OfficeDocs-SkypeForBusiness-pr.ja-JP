# <a name="reviewing-the-administrator-reports-in-lync-server-2013"></a>Lync Server 2013 での管理者レポートの確認

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**トピックの最終更新日:** 2013-02-21_

管理者レポートは、展開と運用に関する詳細情報です。 レポートは、[**設計サイト**でマークされた選択に基づいて生成されます。 設計者は、ネットワーク ダイアグラムを編集し、サーバー、プール、およびロード バランサーの完全な IP アドレスと完全修飾ドメイン名 (FQDN) を定義することで、管理者レポートに値を追加できます。
