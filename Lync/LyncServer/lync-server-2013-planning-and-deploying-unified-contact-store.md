# Lync Server 2013 での統合連絡先ストアの計画と展開

 

_**トピックの最終更新日:** 2012-06-14_

統合連絡先ストアは、Microsoft Office 製品間で統一された連絡先エクスペリエンスを提供する機能です。この機能を使用すると、ユーザーはすべての連絡先情報を Exchange 2013 に格納でき、 Lync、 Exchange、Outlook、Outlook Web Access の間でグローバルにこの情報を利用できます。
