# モバイル クライアントの展開プロセス

 

_**トピックの最終更新日:** 2015-03-09_

Lync Server 2013 の展開が完了した後で、ユーザーは、特定のデバイス用の使い慣れたモバイル マーケットプレースから Lync 2013 アプリケーションをインストールできます。
