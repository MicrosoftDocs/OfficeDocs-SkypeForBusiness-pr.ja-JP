# <a name="interoperability-cmdlets-in-lync-server-2013"></a>Lync Server 2013 の相互運用性のコマンドレット

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**トピックの最終更新日:** 2012-06-27_

相互運用性のコマンドレットは、Microsoft Lync Server 2013 と、Microsoft Exchange Server 2013 などのその他のサーバー製品との間のサーバー間認証と承認を構成するために使用されます。 サーバー間認証および承認を使用すると、これらのサーバーがデータをシームレスに交換したり、共有したりできます。
