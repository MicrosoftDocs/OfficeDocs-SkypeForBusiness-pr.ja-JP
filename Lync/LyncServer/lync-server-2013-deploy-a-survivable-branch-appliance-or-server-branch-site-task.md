# Lync Server 2013 での存続可能ブランチ アプライアンスまたは存続可能ブランチ サーバーの展開 - ブランチ サイトのタスク

 

_**トピックの最終更新日:** 2014-10-28_

「[Lync Server 2013 による存続可能ブランチ アプライアンスまたはサーバーの展開 - 中央サイトのタスク](lync-server-2013-deploying-a-survivable-branch-appliance-or-server-central-site-tasks.md)」のタスクを正常に完了した後、このトピックで説明する 2 つの手順のうち 1 つをブランチ サイトで実行します。


> [!IMPORTANT]
> この手順を実行するには、RTCUniversalSBATechnicians グループのメンバーである必要があります。



## 存続可能ブランチ アプライアンスを展開するには

  - 存続可能ブランチ アプライアンスの展開は、 存続可能ブランチ アプライアンス ベンダーが Web ユーザー インターフェイス (UI) を介して有効にします。 存続可能ブランチ アプライアンスの展開については、 存続可能ブランチ アプライアンス ベンダーのドキュメントを参照してください。

## 存続可能ブランチ サーバーを展開するには

  - 他の Lync Server 2013 サーバーの役割をインストールするように Windows Server 2008 または Windows Server 2008 R2 を実行しているコンピューターに Microsoft Lync Server 2010 をインストールします。
    
    > [!NOTE]
    > Lync Server のインストールについては、「展開」のドキュメントの「<a href="lync-server-2013-deploying-lync-server.md">Lync Server 2013 の展開</a>」を参照してください。
