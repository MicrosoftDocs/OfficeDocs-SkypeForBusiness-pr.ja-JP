# グループ チャット サーバーの移行

 

_**トピックの最終更新日:** 2012-09-29_

レガシ Lync Server 2010 環境で グループ チャット サーバーを展開した場合、Lync Server 2013常設チャット サーバーを展開する必要があります。グループ チャット サーバーと 常設チャット サーバーは共存できますが、コンテンツ (たとえば、チャット ルームなど) はこれらのサーバー間で共有できません。常設チャット サーバーからレガシ グループ チャット サーバー コンテンツにアクセスするには、グループ チャット サーバーを 常設チャット サーバーに移行する必要があります。常設チャット サーバーへの移行の詳細については、「移行」ドキュメントの「[Lync Server 2010、グループ チャットまたは Office Communicatins Server 2007 R2 グループ チャットから Lync Server 2013、常設チャット サーバーへの移行](migration-from-lync-server-2010-group-chat-or-office-communications-server-2007-r2-group-chat-to-lync-server-2013-persistent-chat-server.md)」を参照してください。
