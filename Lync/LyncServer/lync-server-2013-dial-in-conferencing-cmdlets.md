## <a name="dial-in-conferencing-cmdlets"></a>Dial-In Conferencing Cmdlets

ダイヤルイン会議を許可しない場合は、Set-CsConferencingPolicy コマンドレットを使用し、Enabledial Inコンファレンスプロパティを False に設定することによって、この機能を無効にすることができます。 既定では、すべてのユーザーは、ダイヤルイン会議を含む会議をホストすることを許可されています。
