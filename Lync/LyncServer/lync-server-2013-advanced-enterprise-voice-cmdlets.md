# <a name="advanced-enterprise-voice-cmdlets-in-lync-server-2013"></a>Lync Server 2013 の高度なエンタープライズ Voip のコマンドレット

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**トピックの最終更新日:** 2016-04-12_

組織でエンタープライズ VoIP を実装する際には、組織のユーザー エクスペリエンスと効率を向上させる追加の機能とアプリケーションをオプションとして実装できます。これらの機能には、通話受付管理、Enhanced 9-1-1 (E9-1-1) 緊急通話、およびメディア構成設定が含まれます。
