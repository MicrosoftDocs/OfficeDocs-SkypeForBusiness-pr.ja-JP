# Lync Server 2013 のピアツーピア アクティビティ概要レポート

 

_**トピックの最終更新日:** 2015-03-09_

ピアツーピア アクティビティ概要レポートでは、ピアツーピア通信セッションの概要が示されます。通常、ピアツーピア セッションには、2 人のユーザーだけが参加し、 Lync Server 会議サービスの使用は要求されません。それに対し、会議には通常 2 人以上のユーザーが参加し、 Microsoft Lync Server 2013 会議サービスの使用が要求されます。会議アクティビティは、会議概要レポートで報告されます。

ピアツーピア アクティビティ概要レポートは、次のような質問に答えるときに役立ちます。

  - 通常、ユーザーはピアツーピア インスタント メッセージを 1 日に何通送信しますか。

  - ユーザーは Lync Server のアプリケーション共有およびファイル転送機能を実際に使用していますか。

  - ユーザーが、ネットワークの速度が 1 日の特定の時刻で遅くなるようだと訴えています。これらの時間帯でピアツーピアの音声ビデオセッションに費やされた時間は何分ですか。

## ピアツーピア アクティビティ概要レポートにアクセスする

監視レポートのホーム ページからピアツーピア アクティビティ概要レポート にアクセスします。「 [Lync Server 2013 のピアツーピア IM レポート](lync-server-2013-peer-to-peer-im-report.md)」を開くには、次のいずれかの指標をクリックします。

  - \[ピアツーピア IM セッションの合計数\]

  - \[ピアツーピア IM メッセージの合計数\]

同様に、ピアツーピア音声およびビデオ レポートを開くには、次のいずれかの指標をクリックします。

  - \[ピアツーピア音声セッションの合計数\]

  - \[ピアツーピア音声セッションの合計時間 (分)\]

  - \[ピアツーピア音声セッションの合計数\]

  - \[ピアツーピア音声セッションの合計時間 (分)\]

## ピアツーピア アクティビティ概要レポートを最大限に活用する

ピアツーピア アクティビティ概要レポートの下部には、ピアツーピア IM セッションの合計数やピアツーピア IM メッセージの合計数などの、合計の指標が表示されます。これにより、レポート本文にある詳細情報の概要が示されます。

## フィルター

フィルターは、細かく絞り込んだデータ セットを返したり、返されたデータをさまざま方法で表示したりする方法として利用できます。たとえば、ピアツーピア アクティビティ概要レポートでは、データをグループ化する方法を選択できます。この場合は、時間、日、週、または月を基準にアクティビティがグループ化されます。
