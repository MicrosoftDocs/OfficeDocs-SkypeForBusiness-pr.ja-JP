<span data-ttu-id="ba87f-103">_**最終更新日:** 2012-12-18_</span><span class="sxs-lookup"><span data-stu-id="ba87f-103">_**Topic Last Modified:** 2012-12-18_</span></span>

<span data-ttu-id="ba87f-104">エンタープライズ音声通話管理機能は、着信通話のルーティングと応答の方法を制御します。</span><span class="sxs-lookup"><span data-stu-id="ba87f-104">Enterprise Voice call management features control how incoming calls are routed and answered.</span></span> <span data-ttu-id="ba87f-105">Lync Server 2013 には、次のような通話管理機能が用意されています。</span><span class="sxs-lookup"><span data-stu-id="ba87f-105">Lync Server 2013 provides the following call management features:</span></span>

  - <span data-ttu-id="ba87f-106">**コールパーク:** 音声ユーザーが一時的に通話をパークして、同じ電話または別の電話から通話を受け取れるようにします。</span><span class="sxs-lookup"><span data-stu-id="ba87f-106">**Call Park:** Enables voice users to temporarily park a call and then pick it up from the same phone or another phone.</span></span>

  - <span data-ttu-id="ba87f-107">**グループピックアップ:** 通話のピックアップグループ番号をダイヤルすることによって、ピックアップグループに割り当てられている別のユーザーに対して発信された通話に応答することができます。</span><span class="sxs-lookup"><span data-stu-id="ba87f-107">**Group Pickup:** Enables users to answer calls made to another user who is assigned to a pickup group by dialing the call pickup group number.</span></span>

  - <span data-ttu-id="ba87f-108">**応答グループ:** ハントグループまたはインタラクティブな音声応答 (IVR) の質問と回答を使って、着信通話をエージェントのグループにルーティングします。</span><span class="sxs-lookup"><span data-stu-id="ba87f-108">**Response Group:** Routes incoming calls to groups of agents by using hunt groups or interactive voice response (IVR) questions and answers.</span></span>

  - <span data-ttu-id="ba87f-109">**お知らせ:** 割り当てられていない番号への通話に対してメッセージを再生するか、通話を別の場所またはその両方にルーティングします。</span><span class="sxs-lookup"><span data-stu-id="ba87f-109">**Announcement:** Plays a message for calls made to an unassigned number, or routes the call elsewhere, or both.</span></span>

<span data-ttu-id="ba87f-110">このセクションでは、エンタープライズ Voip の展開時にこれらの通話管理機能を構成する方法について説明します。</span><span class="sxs-lookup"><span data-stu-id="ba87f-110">This section describes how to configure these call management features during an Enterprise Voice deployment.</span></span>
