# Lync Server 2013 でのディレクターの設定

 

_**トピックの最終更新日:** 2014-05-05_

エッジ サーバーを展開して外部ユーザーのアクセスを有効にしている場合、ディレクターを展開するという方法があります。ディレクターはユーザー リクエストを認証する Microsoft Lync Server 2013 を実行するサーバーですが、ユーザー アカウントは置かれていません。これは要件ではありませんが、パフォーマンスの心配があり、認証リクエストを合理化したい場合は非常に便利です。これが組織にとって有益であると判断した場合、ディレクターやディレクター プールの設定方法は Enterprise Edition フロントエンド プールや Standard Edition サーバーの設定方法と似ています。トポロジ ビルダーでディレクターを定義してから、このセクションの手順を実行する必要があります。
