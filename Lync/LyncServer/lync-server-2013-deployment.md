# 展開

 

_**トピックの最終更新日:** 2012-10-18_

Lync Server 2013 通信ソフトウェアの展開では、Active Directory ドメイン サービス の準備、フロントエンド サーバーおよび他の中核的な Lync Server 2013 の内部コンポーネントの展開、および外部ユーザー アクセス、エンタープライズ VoIP など、組織で必要なその他のサーバーの役割と機能の展開を行います。

このドキュメントでは、Lync Server 2013 の 3 つの展開シナリオについて説明します。

  - Lync Server 2013、Enterprise Edition の新しい展開

  - Lync Server 2013、Standard Edition の新しい展開

  - 既存の Lync Server 2013 Standard Edition または Enterprise Edition 展開環境への、Lync Server 2010 Standard Edition または Enterprise Edition の新しい展開

既存の Microsoft Office Communications Server 2007 または Microsoft Office Communications Server 2007 R2 環境に Lync Server 2013 を展開する方法については、「[移行](migration.md)」のドキュメントを参照してください。
