# Lync Server 2013 での常設チャット サービスの監視、開始、および停止

 

_**トピックの最終更新日:** 2012-10-06_

常設チャット サービスと 常設チャット コンプライアンス サービスは、Lync Server 2013 トポロジの一部なので、Windows PowerShell コマンドレットを使用して監視、停止、および起動できます。これらの操作にはそれぞれ、**get-CsWindowsService**、**stop-CsWindowsService**、**start-CsWindowsService** を使用します。
