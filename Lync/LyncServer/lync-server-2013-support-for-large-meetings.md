# Lync Server 2013 での大規模会議のサポート

 

_**トピックの最終更新日:** 2012-10-01_

Lync Server 2013 では、共有 PowerPoint プレゼンテーションを含む音声ビデオ (A/V) 会議を使用して最大 1000 人が参加できる会議をサポートできます。このサポートには、大規模な会議をサポートするように構成され、一度に 1 つの大規模な会議のみ開催するような方法で管理される専用プールが必要です。

ここでは、専用 Lync Server 2013 プールを使用して大規模な会議をサポートする方法について説明します。スケーラビリティの考慮事項と、トポロジ、ハードウェア、ソフトウェア、および構成の要件を含む専用プールの実装要件について説明します。大規模な会議をサポートするための一連のベスト プラクティス推奨、Lync Server エンジニアリング チームによって実施されたサーバー スケーラビリティ テストのテスト方法と結果の概要、および大規模な会議のサポートについてよくある質問の回答も示します。
