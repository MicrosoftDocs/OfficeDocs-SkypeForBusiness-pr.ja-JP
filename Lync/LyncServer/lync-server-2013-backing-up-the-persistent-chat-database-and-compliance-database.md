# Lync Server 2013 での常設チャット データベースおよびコンプライアンス データベースのバックアップ

 

_**トピックの最終更新日:** 2012-08-01_

常設チャット データベースおよびコンプライアンス データベースは、定期的にバックアップする必要があります。ご使用の環境に最適なバックアップ方法については、データベース管理者に問い合わせてください。
