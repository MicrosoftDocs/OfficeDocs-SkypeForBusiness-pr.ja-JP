  - <span data-ttu-id="825e0-106">**HD ビデオ**    ユーザーは、2者間の通話とマルチパーティの電話会議で最大 HD 1080P の解像度を使用できます。</span><span class="sxs-lookup"><span data-stu-id="825e0-106">**HD video**   Users can experience resolutions up to HD 1080P in two-party calls and multiparty conferences.</span></span>
