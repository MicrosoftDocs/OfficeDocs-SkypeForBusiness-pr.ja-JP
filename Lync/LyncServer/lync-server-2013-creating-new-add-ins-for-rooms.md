# <a name="creating-new-add-ins-for-rooms-in-lync-server-2013"></a>Lync Server 2013 でのルーム用の新しいアドインの作成

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**最終更新日:** 2012-10-06_
