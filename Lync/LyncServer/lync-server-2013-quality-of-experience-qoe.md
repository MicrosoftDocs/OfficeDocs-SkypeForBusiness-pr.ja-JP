# <a name="quality-of-experience-qoe-in-lync-server-2013"></a>Lync Server 2013 の QoE (Quality of Experience)

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**トピックの最終更新日:** 2012-11-01_

QoE (Quality of Experience) は、メディアの品質と、通話およびセッションに関係する参加者、デバイス名、ドライバー、IP アドレス、およびエンドポイントの種類についての情報を示す数値データを記録します。 Lync Server 2013 をインストールするときに、QoE のグローバル構成設定の定義済みコレクションもインストールします。 QoE の設定を構成するには、このセクションのトピックに従ってください。
