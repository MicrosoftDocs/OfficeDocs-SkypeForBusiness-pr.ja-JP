# Lync Server 2013 での高度なエンタープライズ VoIP 機能の展開

 

_**トピックの最終更新日:** 2012-09-22_

高度なエンタープライズ VoIP の機能の詳細については、「[Lync Server 2013 での計画](lync-server-2013-planning.md)」ドキュメントの次のセクションを参照してください。
