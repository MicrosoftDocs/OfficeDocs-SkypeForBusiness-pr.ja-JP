# メディア品質比較レポート

 

_**トピックの最終更新日:** 2015-03-09_

メディア品質比較レポートを使用すると、さまざまな種類の音声通話の通話品質の値を比較できます (たとえば、ワイヤレス ネットワーク経由の通話と有線接続経由の通話の比較)。

## メディア品質比較レポートへのアクセス

メディア品質比較レポートは、\[監視レポート\] ホーム ページからアクセスします。
