# Lync Server 2013 のエッジ コンポーネントのハードウェアおよびソフトウェア要件

 

_**トピックの最終更新日:** 2013-03-14_

エッジ コンポーネントのハードウェアとソフトウェアの要件には、エッジ サーバーとオプションのディレクターを含む、Lync Server 2013 通信ソフトウェア コンポーネントの要件が含まれます。外部ユーザー アクセスおよびサポートされるトポロジのサポートに必要なコンポーネントの詳細については、「[Lync Server 2013 の外部ユーザー アクセスに必要なコンポーネント](lync-server-2013-components-required-for-external-user-access.md)」を参照してください。

## エッジ サーバーおよびディレクター用のハードウェア

ハードウェアの推奨事項と要件は、「サポート」セクションで定義されています。エッジ サーバーおよびオプションのディレクターの推奨されるハードウェア構成の詳細については、次のトピックを参照してください。計画については、「[Lync Server 2013　用のサーバー ハードウェア プラットフォーム](lync-server-2013-server-hardware-platforms.md)」を参照してください。

## エッジ サーバーおよびディレクターのソフトウェア要件

エッジ サーバーおよびオプションのディレクターに関するオペレーティング システムの要件は、他の Lync Server 2013 の役割と一致しています。サポートされるオペレーティング システムの詳細については、「[Lync Server 2013 でのサーバーおよびツールのオペレーティング システムのサポート](lync-server-2013-server-and-tools-operating-system-support.md)」を参照してください。

エッジ サーバーおよびオプションのディレクターに関する追加のコンポーネントとソフトウェア要件の詳細については、「[Lync Server 2013 の追加ソフトウェア要件](lync-server-2013-additional-software-requirements.md)」を参照してください。
