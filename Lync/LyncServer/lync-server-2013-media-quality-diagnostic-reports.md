# Lync Server 2013 のメディア品質診断レポート

 

_**トピックの最終更新日:** 2013-02-22_

メディア品質診断レポートは、通話の品質についての情報、および失敗した通話についての診断とトラブルシューティングの情報を提供します。

## このセクション中

  - [Lync Server 2013 のメディア品質概要レポート](lync-server-2013-media-quality-summary-report.md)   エンタープライズ VoIP ピアツーピア通話、エンタープライズ VoIP 電話会議、公衆交換電話網 (PSTN) を全面的または部分的に利用する通話などに関して、さまざまな種類のエンドポイントの全般的な品質データを提供します。

  - [メディア品質比較レポート](lync-server-2013-media-quality-comparison-report.md)   さまざまな種類の音声通話について、通話品質の値を比較します (たとえば、ワイヤレス ネットワーク経由の通話と、有線接続による通話)。

  - [Lync Server 2013 のサーバー パフォーマンス レポート](lync-server-2013-server-performance-report.md)   品質低下、パケット損失、ジッターなどの主要な品質指標の測定値に基づいて、特に多くの問題が発生したサーバーを表示します。

  - [Lync Server 2013 の場所レポート](lync-server-2013-location-report.md)   ネットワークの場所の一覧と、それぞれの場所で行われた通話のメディア品質の概要を表示します。このレポートでは IP サブネットに基づいて場所を扱います。

  - [Lync Server 2013 デバイス レポート](lync-server-2013-device-report.md)   エンタープライズ VoIP 通話で使用されたデバイスの概要と、デバイスごとの通話の平均的なメディア品質を表示します。

  - [Lync Server 2013 の通話リスト レポート](lync-server-2013-call-list-report.md)   組織内で発信または受信した通話についての詳細情報を提供します。

  - [Lync Server 2013 の通話の詳細レポート](lync-server-2013-call-detail-report.md)   組織内で発信または受信した通話についての詳細情報を提供します。

  - [サーバー メディア品質傾向レポート](lync-server-2013-server-media-quality-trend-report.md)   通話件数、低品質通話のパーセンテージ、パケット損失、ジッターなどの QoE (Quality of Experience) 指標について、最大 5 台のサーバーを視覚的に比較できます。

  - [メディア品質メトリック分布レポート](lync-server-2013-media-quality-metrics-distribution-report.md)   ジッターやパケット損失などの QoE (Quality of Experience) 指標の分布値を示すグラフを提供します。

  - [場所の傾向レポート](lync-server-2013-location-trend-report.md)   ネットワークの場所の通話品質に関する傾向情報を提供します。
