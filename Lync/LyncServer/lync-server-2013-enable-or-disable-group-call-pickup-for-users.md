# <a name="enable-or-disable-group-call-pickup-for-users-in-lync-server-2013"></a>Lync Server 2013 でユーザーのグループ通話ピックアップを有効または無効にする

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**トピックの最終更新日:** 2013-02-22_

グループ通話ピックアップ範囲がコールパークオービットテーブルに追加されると、ユーザーのグループ通話ピックアップを必要に応じて有効または無効にすることができます。 このセクションでは、SEFAUtil リソースキットツールを使用してユーザーのグループ通話ピックアップを有効または無効にする方法について説明します。

<div>


> [!NOTE]  
> SEFAUtil パラメーター、または enablegrouppickup を使用すると、グループ通話ピックアップが有効になり、グループ番号が割り当てられます。



</div>
