# <a name="configure-announcements-in-lync-server-2013"></a>Lync Server 2013 でのアナウンスの構成

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**トピックの最終更新日:** 2012-09-12_

アナウンスを構成する場合、実際には、割り当てられていない番号への通話をどのように処理するかを構成しています。 音声ファイルや音声合成 (TTS) ファイルなどのプロンプトを再生したり、プロンプトを再生せずに特定の宛先に通話を転送したりすることができます。

割り当てられていない番号の表を定義する前に、アナウンスを作成する必要があります。 音声ガイダンス、TTS プロンプト、または音声ガイダンスを使用するすべてのアナウンスに対してこの手順を実行する必要があります。
