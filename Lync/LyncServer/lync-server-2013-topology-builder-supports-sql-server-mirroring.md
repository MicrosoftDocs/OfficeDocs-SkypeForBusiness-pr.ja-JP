# <a name="topology-builder-supports-sql-server-mirroring-in-lync-server-2013"></a>SQL Server のミラーリングをサポートする Lync Server 2013 のトポロジ ビルダー

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**最終更新日:** 2012-09-26_
