# Lync Server 2013 での、割り当てられていない番号用のアナウンスの構成

 

_**トピックの最終更新日:** 2012-09-11_

アナウンス アプリケーションは エンタープライズ VoIP の機能で、割り当てられていない内線番号 (組織で有効であるがユーザーや電話に割り当てられていない内線番号) に通話があったときの動作を構成できます。たとえば、割り当てられていない番号に通話があった場合にメッセージを再生したり、別の通話先に転送したり、その両方を行ったりするように構成できます。

アナウンス アプリケーションは、エンタープライズ VoIP を展開するときに 応答グループ アプリケーションの機能としてフロントエンド サーバーまたは Standard Edition サーバーにインストールされます。オーディオ ファイルをアップロードするか音声合成 (TTS) を構成して、割り当てられていない番号の表を構成し、アナウンスを構成する必要があります。

ここでは、Lync Server のアナウンスの構成について説明します。アナウンスに関連する「計画」のセクションを既に読んでおり、エンタープライズ VoIP を備えた Enterprise Edition サーバーまたは Standard Edition サーバーを展開していることが前提です。
