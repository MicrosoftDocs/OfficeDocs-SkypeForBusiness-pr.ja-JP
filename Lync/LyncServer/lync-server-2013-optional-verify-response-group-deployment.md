# (オプション) 応答グループの展開の確認

 

_**トピックの最終更新日:** 2012-09-11_

応答グループを構成したら、構成を検証して応答グループが意図したとおりに機能するかどうかを確かめる必要があります。少なくとも、次の種類のユーザーを使用して以下のシナリオを検証してください。

**ユーザー**

  - Lync Server 2013 に所属するユーザー

  - 公衆交換電話網 (PSTN) を使用する外部ユーザー

  - Lync Server 2013 に所属するエージェント

**シナリオ**

  - Lync Server 2013 のユーザーが応答グループを呼び出す。

  - 外部ユーザーが応答グループを呼び出す。

  - エージェントが別の通話に応答しているときにユーザーが応答グループを呼び出し、キューに入る。
