# <a name="optional-verify-response-group-deployment-in-lync-server-2013"></a>オプションLync Server 2013 での応答グループの展開の確認

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**トピックの最終更新日:** 2012-09-11_

応答グループを構成した後、応答グループが想定どおりに機能するように構成を検証する必要があります。 少なくとも、次の種類のユーザーを使用して以下のシナリオを検証してください。

**Users**

  - Lync Server 2013 に所属するユーザー

  - 公衆交換電話網 (PSTN) を使用する外部ユーザー

  - Lync Server 2013 に所属するエージェント

**Scenarios**

  - Lync Server 2013 ユーザーが応答グループを呼び出します。

  - 外部ユーザーが応答グループを呼び出す。

  - エージェントが別の通話に応答しているときにユーザーが応答グループを呼び出し、キューに入る。
