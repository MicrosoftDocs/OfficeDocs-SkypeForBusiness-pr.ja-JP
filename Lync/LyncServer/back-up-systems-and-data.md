# <a name="back-up-systems-and-data"></a>システムとデータをバックアップする

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**トピックの最終更新日:** 2012-09-18_

Lync Server 2013 への移行を開始する前に、システムの完全バックアップを実行し、各プールに所属するユーザーアカウントのインベントリを含む既存のシステムをドキュメント化して、Lync Server 2010 にロールバックできるようにすることを強くお勧めします。が必要になります。 データ、設定、およびシステムのバックアップと復元には、複数のツールおよびプログラムを使用できます。

詳細と手順については、「 [Lync Server 2010 のバックアップと復元](https://go.microsoft.com/fwlink/p/?linkid=265417)」を参照してください。
