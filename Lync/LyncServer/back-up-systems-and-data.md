# <a name="back-up-systems-and-data"></a>システムとデータのバックアップ

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**最終更新日:** 2012-09-18_

Lync Server 2013 への移行を開始する前に、完全なシステムバックアップを実行し、各プールに所属しているユーザーアカウントのインベントリなど、既存のシステムを文書化して、Lync Server 2010 にロールバックすることを強くお勧めします。必要になります。 データ、設定、およびシステムのバックアップと復元には、複数のツールとプログラムが用意されています。

詳細と手順については、「 [Lync Server 2010 のバックアップと復元](http://go.microsoft.com/fwlink/p/?linkid=265417)」を参照してください。
