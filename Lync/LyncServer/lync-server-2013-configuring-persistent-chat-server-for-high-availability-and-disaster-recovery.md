# Lync Server 2013 の高可用性と障害復旧に対応した常設チャット サーバーの構成

 

_**トピックの最終更新日:** 2012-10-01_

Lync Server 2013、 常設チャット サーバー サービスでは、障害復旧のために *拡大されたプール* 構成を使用します。拡大されたプールとは、1 つの論理的な Lync Server サイト内にある 2 つの物理データ センターに分散されたコンピューターを持つプールのことです。
