_**トピックの最終更新日:** 2012-06-27_

常設チャットコマンドレットを使用すると、Microsoft Lync Server 2013 常設チャットサービス (以前のグループチャットサービス) を管理および構成できます。 常設チャットを使用すると、ユーザーはオンラインインスタントメッセージングセッションに参加できます。 これらのセッションはリアルタイムで行われますが、各セッションの内容は永続的です。つまり、これらの会話は、いつでもだれでも再開できます。
