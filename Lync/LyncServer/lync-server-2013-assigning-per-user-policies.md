# ユーザー単位のポリシーの割り当て

 

_**トピックの最終更新日:** 2012-10-14_

他のユーザーに割り当てる、グローバル ポリシーなどのポリシーに定義されている設定と異なる特別な設定を指定するため、ユーザーまたはユーザー グループに特定のポリシーを割り当てることができます。 これらのポリシーは、ユーザー単位のポリシーと呼ばれます。
