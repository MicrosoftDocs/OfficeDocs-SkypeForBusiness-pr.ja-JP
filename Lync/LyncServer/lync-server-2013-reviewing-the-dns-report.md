# <a name="reviewing-the-dns-report-in-lync-server-2013"></a>Lync Server 2013 での DNS レポートの確認

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**トピックの最終更新日:** 2013-02-21_

管理者レポートの一部である DNS レポートは、ドメインネームシステム (DNS) について、内部、境界、および外部ネットワークの推奨されるエントリと既知のエントリのすべてについて詳細に説明します。 デザイナーがネットワークダイアグラムの編集を完了しており、すべての IP アドレスと完全修飾ドメイン名 (Fqdn) が運用時の値に対して定義されている場合、DNS レポートは優れた構成リソースを提供します。 このレポートは、運用トラブルシューティングドキュメントとしても利用できます。

![DNS 管理レポート](images/Gg558641.9dd1e810-ddc7-4816-a806-4239baf9ec51(OCS.15).jpg "DNS 管理レポート")

Dns 管理チームは、DNS レポートを十分に確認して、展開中に問題を引き起こす可能性があるエラーが発生していないこと、またはトラブルシューティングセッションが複雑になる可能性があることを確認する必要があります。
