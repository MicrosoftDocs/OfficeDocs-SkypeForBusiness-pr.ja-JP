# Lync Server 2013 の境界ネットワークの VoIP コンポーネント

 

_**トピックの最終更新日:** 2012-09-21_

個別の通話または電話会議に統合コミュニケーション クライアントを使用する外部のユーザーには、仕事仲間との音声通信に エッジ サーバーが必要です。

エッジ サーバーのアクセス エッジ サービスは、組織のファイアウォールの外部にいる Lync のユーザーから着信する SIP 信号を処理します。音声ビデオ エッジ サービスを使用すると、メディアが NAT およびファイアウォールを通過できます。企業ファイアウォールの外部で統合コミュニケーション (UC) クライアントを使用する発信者には、個別の通話と電話会議の両方に音声ビデオ エッジ サービスが必要です。

音声ビデオ認証サービスは音声ビデオ エッジ サービスと同じコンピューターに配置され、音声ビデオ エッジ サービスのための認証サービスを実行します。外部のユーザーが音声ビデオ エッジ サービスに接続するには、発信を行う前に、音声ビデオ認証サービスから認証トークンを取得する必要があります。
