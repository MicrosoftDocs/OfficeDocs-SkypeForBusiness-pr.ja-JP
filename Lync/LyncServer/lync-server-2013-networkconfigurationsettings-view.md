# NetworkConfigurationSettings ビュー

 

_**トピックの最終更新日:** 2015-03-09_

NetworkConfigurationSettings ビューには、ネットワーク構成についての情報が格納されます。このビューは、Microsoft Lync Server 2013 で導入されました。
