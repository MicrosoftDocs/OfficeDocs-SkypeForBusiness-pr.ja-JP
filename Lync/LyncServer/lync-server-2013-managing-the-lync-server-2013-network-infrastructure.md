# Lync Server 2013 ネットワーク インフラストラクチャの管理

 

_**トピックの最終更新日:** 2014-02-11_

Microsoft Lync Server 2013 は、通話受付管理 (CAC) とメディア バイパスをサポートします。これらの機能を実装するには、音声とビデオの送信を規制する必要のある状況で帯域幅を管理できるように、地域、サイト、サブネットなどのネットワークを構成する必要があります。サービスの品質 (QoS) ネットワーク テクノロジの使用も、音声およびビデオ通信の最適なエンドユーザー エクスペリエンスを提供するのに役立ちます。

Lync Server コントロール パネルを使用して、CAC、メディア バイパス、QoS の設定および管理を行うことができます。次のトピックで、これらの手順を説明します。
