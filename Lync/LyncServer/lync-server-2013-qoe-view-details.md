# <a name="qoe-view-details-in-lync-server-2013"></a>Lync Server 2013 の QoE ビューの詳細

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**トピックの最終更新日:** 2012-10-03_
