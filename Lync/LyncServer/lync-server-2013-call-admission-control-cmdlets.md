# <a name="call-admission-control-cmdlets-in-lync-server-2013"></a>Lync Server 2013 での通話受付管理のコマンドレット

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**トピックの最終更新日:** 2012-03-21_

通話受付管理 (CAC) は、許容できる品質のリアルタイムの音声またはビデオセッションを確立するのに十分なネットワーク帯域幅があるかどうかを判断します。 CAC を管理するには、ネットワーク、サイト、およびサブネットの制限と設定を構成し、それらの間の相互作用を行います。
