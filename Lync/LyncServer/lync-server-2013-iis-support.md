# Lync Server 2013 での IIS のサポート

 

_**トピックの最終更新日:** 2012-06-29_

Lync Server 2013 コンポーネントの中には インターネット インフォメーション サービス (IIS) を必要とするものがあります。サポートされている IIS のバージョンには次のものがあります。

  - IIS 8.0

  - IIS 7.5

Lync Server 2013 コンポーネントの IIS 要件の詳細については、「計画」のドキュメントの「[Lync Server 2013 のインターネット インフォメーション サービス (IIS) の要件](lync-server-2013-internet-information-services-iis-requirements.md)」を参照してください。
