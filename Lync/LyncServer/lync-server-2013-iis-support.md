# <a name="iis-support-in-lync-server-2013"></a>Lync Server 2013 での IIS のサポート

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**最終更新日:** 2012-06-29_

一部の Lync Server 2013 コンポーネントには、インターネットインフォメーションサービス (IIS) が必要です。 サポートされているバージョンの IIS には、次のようなものがあります。

  - IIS 8.0

  - IIS 7.5

Lync Server 2013 コンポーネントの IIS 要件の詳細については、計画ドキュメントの「 [Lync server 2013 のインターネットインフォメーションサービス (IIS) の要件](lync-server-2013-internet-information-services-iis-requirements.md)」を参照してください。
