# Lync Server 2013 でのサーバーの役割の追加

 

_**トピックの最終更新日:** 2012-06-19_

初期の展開が稼働するようになったら、 Lync Server 2013 に対して、他のサーバーの役割 ( ディレクター、スタンドアロン 仲介サーバーなど) を追加できます。

> [!NOTE]
> サーバーの役割を追加インストールするタスクを実行する前に、各役割に関連する計画トピックを参照してください。
