Lync Server 2013 に移行する前に、Lync Server 2010 環境に更新プログラムを適用する必要があります。 Lync Server 2010 に関する最新情報については、「 **lync のリソースセンターを更新する** 」を参照してください <https://go.microsoft.com/fwlink/p/?linkid=232630> 。
