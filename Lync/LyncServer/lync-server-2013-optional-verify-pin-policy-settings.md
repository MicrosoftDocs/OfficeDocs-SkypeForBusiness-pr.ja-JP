# <a name="optional-verify-pin-policy-settings-in-lync-server-2013"></a>オプションLync Server 2013 での PIN ポリシー設定の確認

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**トピックの最終更新日:** 2012-06-20_

Lync Server 2013 Active Directory ドメインサービスの資格情報を持つユーザーは、認証されたユーザーとしてダイヤルイン会議に参加するための個人識別番号 (PIN) を入力できます。 PIN ポリシーは、ダイヤルイン会議の PIN がどのように機能するかについてのルールを定義します。

ダイヤルイン会議を展開するときには、既定のグローバル PIN ポリシーが要件を満たすことを検証してください。 変更を加える必要がある場合は、既定のグローバル ポリシーを変更するか、新しい PIN ポリシーを作成できます。 特定のサイト、特定のユーザー、または特定のユーザー グループに適用される PIN ポリシーを作成できます。
