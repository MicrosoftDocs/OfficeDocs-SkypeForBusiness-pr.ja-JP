# (オプション) Lync Server 2013 での PIN ポリシー設定の確認

 

_**トピックの最終更新日:** 2012-06-20_

Active Directory ドメイン サービス の資格情報を持つ Lync Server 2013 ユーザーは、暗証番号 (PIN) を入力して認証済みユーザーとしてダイヤルイン会議に参加できます。PIN ポリシーは、ダイヤルイン会議の PIN がどのように機能するかについてのルールを定義します。

ダイヤルイン会議を展開するときには、既定のグローバル PIN ポリシーが要件を満たすことを検証してください。 変更を加える必要がある場合は、既定のグローバル ポリシーを変更するか、新しい PIN ポリシーを作成できます。 特定のサイト、特定のユーザー、または特定のユーザー グループに適用される PIN ポリシーを作成できます。
