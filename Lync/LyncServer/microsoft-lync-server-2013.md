# Microsoft Lync Server 2013

 

_**トピックの最終更新日:** 2016-12-08_

Microsoft Lync Server 2013通信ソフトウェアは、エンタープライズレベルのコラボレーション要件をサポートできる、インスタント メッセージング (IM)、プレゼンス、会議、およびテレフォニー ソリューションを提供します。

Lync Server 2013 のドキュメントが必要な場合は、マイクロソフト ダウンロード センター ([http://go.microsoft.com/fwlink/p/?LinkID=331103](http://go.microsoft.com/fwlink/p/?linkid=331103)) から **LyncServer2013\_ITPro** というコンパイルされたヘルプ ファイル (.chm) をダウンロードしてください。

次の表では、Lync Server 2013 のドキュメンテーション ライブラリのトピックをリストしています。
