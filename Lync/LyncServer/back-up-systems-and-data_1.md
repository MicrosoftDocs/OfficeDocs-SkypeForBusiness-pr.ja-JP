# <a name="back-up-systems-and-data"></a>システムとデータのバックアップ

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**最終更新日:** 2012-10-19_

Lync Server 2013 への移行を開始する前に、完全なシステムバックアップを実行し、各プールをホームにしているユーザーアカウントのインベントリなど、既存のシステムをドキュメント化して、Office の通信にロールバックすることを強くお勧めします。必要になった場合は、サーバー 2007 R2。 データ、設定、およびシステムのバックアップと復元には、複数のツールとプログラムが用意されています。 詳細と手順については、「Office Communications Server 2007 R2 バックアップと復元ガイド<http://go.microsoft.com/fwlink/p/?linkid=168162>」を参照してください。
