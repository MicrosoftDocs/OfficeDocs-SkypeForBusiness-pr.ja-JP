# バックアップの前提条件の確認

 

_**トピックの最終更新日:** 2013-02-17_

Lync Server のバックアップを始める前に、以下の準備ができていることを確認します。

  - バックアップ ツール。詳細については、「[バックアップと復元の要件: ツールとアクセス許可](lync-server-2013-backup-and-restoration-requirements-tools-and-permissions.md)」を参照してください。

  - アクセス許可。詳細については、「[バックアップと復元の要件: ツールとアクセス許可](lync-server-2013-backup-and-restoration-requirements-tools-and-permissions.md)」を参照してください。

  - バックアップを格納するための場所。詳細については、「[バックアップ場所の設定](lync-server-2013-setting-up-a-backup-location.md)」を参照してください。

  - バックアップ用のメディア。詳細については、「[バックアップ場所の設定](lync-server-2013-setting-up-a-backup-location.md)」を参照してください。
