# <a name="clients-supported-for-group-call-pickup-in-lync-server-2013"></a>Lync Server 2013 でのグループ通話ピックアップに対してサポートされているクライアント

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**トピックの最終更新日:** 2013-02-12_

次のいずれかのクライアントを使用して、グループ通話ピックアップメンバーへの呼び出しに応答できます。

  - Lync 2013

  - Lync 2010

  - Lync Phone Edition

<div>


> [!NOTE]  
> ユーザーは、これらのクライアントのいずれかを使用して通話のピックアップメンバーをグループ化することができますが、Lync Server 2013 の累積的な更新プログラム (2 月 2 2013 日) については、ユーザーが lync server 2013 プールに所属している必要があります。



</div>

Lync Server 2013 年 2 2013 月の累積的な更新プログラムでは、次のクライアントとデバイスは、グループ通話ピックアップメンバーへの呼び出しの選択ではサポートされていません。

  - Lync Mobile

  - Windows 8 および Windows RT 用 Lync アプリ

  - Lync for iPad

  - アナログ電話

  - 公衆交換電話網 (PSTN) 番号を使用する電話

</div>
