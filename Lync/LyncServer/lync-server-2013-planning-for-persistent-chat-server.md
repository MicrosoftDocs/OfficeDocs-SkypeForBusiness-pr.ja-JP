# <a name="planning-for-persistent-chat-server-in-lync-server-2013"></a>Lync Server 2013 での常設チャットサーバーの計画

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**トピックの最終更新日:** 2012-10-11_

Lync Server 2013 の常設チャットサーバーを使用すると、複数のユーザーが、特定のトピック (テキスト、リンク、ファイルなど) に関するコンテンツを投稿してアクセスできるようにする会話に参加することができます。 ユーザーはセッション中にリアルタイムで通信することができますが、各セッションのコンテンツは永続的です。つまり、セッションが終了した後も引き続き使用できます。

このセクションでは、要件の定義、コンポーネントとサポートされるトポロジの識別、展開に関する推奨事項など、Lync Server 2013、常設チャットサーバーの展開の計画に関する考慮事項について説明します。
