<a name="manage-guest-access-to-microsoft-teams"></a>Microsoft チームにゲスト アクセスを管理します。
======================================

ゲスト アクセスは、すべての Office 365 Business Premium、Office 365 Enterprise、および Office 365 Education サブスクリプションに含まれています。Office 365 ライセンスを追加する必要はありません。
  
    
    
チーム ゲスト アクセスの設定をテナント レベルは、既定でオフにします。IT 管理者は、テナント レベルでゲストを追加、設定、ゲスト ユーザー ポリシーとアクセス許可を管理して、ゲストを招待およびゲスト ユーザーのアクティビティのレポートを抽出できるユーザーを確認することができます。これらのコントロールは、Office 365 管理センターを使用します。ゲスト ユーザーのコンテンツとアクティビティは、[同じコンプライアンスと Office 365 の残りの部分と監査の保護] で、します。
  
    
    

> [!NOTE]
> 設定のみチーム ゲスト アクセス テナントでは、ゲスト所有者が新しい招待できるサインイン チーム来園者し、それぞれチームに既存のディレクトリ ゲスト ユーザーを追加できないようにします。ためには、チームで常に Azure Active Directory 外部の設定を許可またはテナントにゲスト ユーザーの追加できないようにするのには、します。 
  
    
    

さらに、ゲストと Office 365 とチーム リソースへのアクセスを管理するのに Azure Active Directory ポータルを使用することができます。チームがゲスト アクセスを使用して Azure Active Directory ビジネス-(B2B) でのグループ作業機能インフラストラクチャの基になると id プロパティ、メンバーシップ、および多要素など、セキュリティ原則情報を格納します。認証設定します。Azure Active Directory B2B の詳細については、次を参照してください。 [Azure AD B2B での共同作業は何ですか?](https://go.microsoft.com/fwlink/p/?linkid=853011)と[Azure Active Directory B2B コラボレーション Faq](https://go.microsoft.com/fwlink/p/?linkid=853020)します。
