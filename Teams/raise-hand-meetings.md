# <a name="raise-your-hand-option-in-teams-meetings"></a>Teams 会議で手動オプションを上げる

Microsoft Teams の会議中に、参加者は [**手を上げる**] オプションを使用して、発表者とモデレーターの注意を引くことができます。

この機能は、次のような理由で重要です。

- 参加者は、会議を中断することなく、話す必要があることを知らせることができます。

- モデレーターまたは発表者は、会議に参加している人がいるかどうかを確認できます。  

- **手を上げて、** 参加者からの参加を促すことで、会議をより包括的にします。

- **手を上げる**だけで、簡単に簡単に会議の参加者の投票を行うことができます。

チーム管理者は、この機能が会議に表示されるようにするためのポリシーを作成する必要はありません。

## <a name="raise-your-hand-display"></a>手での表示を上げる

**自分の手を上げる**ことは、参加するすべての会議でいつでも利用でき、会議室のデバイスを除くすべてのクライアントでサポートされます。

- 浮き出た針は、すべての会議参加者に表示され、手のアイコンで示されます。

- 発生した針は、会議ステージ、名簿 (期間順)、およびコンテンツを共有するときに表示されます。

- この浮き出た針は、参加者の名前の横に黄色で表示されます。

参加者が認知されたら、**手の下**のオプションを使うことができます。 会議の発表者は、参加者の手を下げることもできます。
