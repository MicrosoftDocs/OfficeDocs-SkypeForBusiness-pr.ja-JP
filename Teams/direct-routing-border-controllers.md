<span data-ttu-id="829de-104">マイクロソフトは、選択したセッション ボーダー コントローラー (SBC) ベンダーと提携して、SBC がダイレクト ルーティングと連携していることを保証しています。</span><span class="sxs-lookup"><span data-stu-id="829de-104">Microsoft partners with selected Session Border Controllers (SBC) vendors to certify that their SBCs work with Direct Routing.</span></span>

- <span data-ttu-id="829de-106">SIP 相互接続プロトコルで共同で作業します。</span><span class="sxs-lookup"><span data-stu-id="829de-106">Jointly work on the SIP interconnection protocols.</span></span>

- <span data-ttu-id="829de-109">実稼働環境とプリプロダクション環境で、認定済みのすべてのデバイスで毎日のテストを実行します。</span><span class="sxs-lookup"><span data-stu-id="829de-109">Run daily tests with all certified devices in production and pre-production environments.</span></span> <span data-ttu-id="829de-110">実稼働前環境でデバイスを検証することにより、クラウド内の新しいバージョンのダイレクト ルーディング コードが認定済み SBC で機能することが保証されます。</span><span class="sxs-lookup"><span data-stu-id="829de-110">Validating the devices in pre-production environments guarantees that new versions of Direct Routing code in the cloud will work with certified SBCs.</span></span>
