<span data-ttu-id="87436-104">マイクロソフトは、選択したセッション ボーダー コントローラー (SBC) ベンダーと提携して、SBC がダイレクト ルーティングと連携していることを保証しています。</span><span class="sxs-lookup"><span data-stu-id="87436-104">Microsoft partners with selected Session Border Controllers (SBC) vendors to certify that their SBCs work with Direct Routing.</span></span> 

<span data-ttu-id="87436-105">Microsoft は各ベンダーと連携して次のことを行うことができます。</span><span class="sxs-lookup"><span data-stu-id="87436-105">Microsoft works with each vendor to:</span></span> 

- <span data-ttu-id="87436-106">SIP 相互接続プロトコルで共同作業を行います。</span><span class="sxs-lookup"><span data-stu-id="87436-106">Jointly work on the SIP interconnection protocols.</span></span>
- <span data-ttu-id="87436-107">サードパーティラボを使用して、大量のテストを実行します。</span><span class="sxs-lookup"><span data-stu-id="87436-107">Perform intense tests using a third-party lab.</span></span> <span data-ttu-id="87436-108">テストに合格したデバイスのみが認定されます。</span><span class="sxs-lookup"><span data-stu-id="87436-108">Only devices that pass the tests are certified.</span></span>
