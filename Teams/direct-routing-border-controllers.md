# <a name="list-of-session-border-controllers-certified-for-direct-routing"></a><span data-ttu-id="05938-103">ダイレクト ルーティングに対応する認定済みセッション ボーダー コントローラーのリスト</span><span class="sxs-lookup"><span data-stu-id="05938-103">List of Session Border Controllers certified for Direct Routing</span></span>

<span data-ttu-id="05938-104">マイクロソフトは、選択したセッション ボーダー コントローラー (SBC) ベンダーと提携して、SBC がダイレクト ルーティングと連携していることを保証しています。</span><span class="sxs-lookup"><span data-stu-id="05938-104">Microsoft partners with selected Session Border Controllers (SBC) vendors to certify that their SBCs work with Direct Routing.</span></span> 

<span data-ttu-id="05938-105">各ベンダーと、マイクロソフトは次のことを行います。</span><span class="sxs-lookup"><span data-stu-id="05938-105">With each vendor, Microsoft:</span></span> 

- <span data-ttu-id="05938-106">SIP 相互接続プロトコルについて、SBC ベンダーと協力して取り組みます。</span><span class="sxs-lookup"><span data-stu-id="05938-106">Jointly works with the SBC vendors on the SIP interconnection protocols.</span></span>
- <span data-ttu-id="05938-107">サードパーティのラボで、集中的なテストを実行します。そのテストに通過したデバイスのみが認定されます。</span><span class="sxs-lookup"><span data-stu-id="05938-107">Performs intense tests via a third-party lab; only devices that passed the tests are certified.</span></span> 
- <span data-ttu-id="05938-108">すべての認定済みデバイスに対して、実稼働環境と実稼働前環境で毎日テストを実行します。</span><span class="sxs-lookup"><span data-stu-id="05938-108">Runs daily tests with all certified devices in production and pre-production environments.</span></span> <span data-ttu-id="05938-109">実稼働前環境でデバイスを検証することにより、クラウド内の新しいバージョンのダイレクト ルーディング コードが認定済み SBC で機能することが保証されます。</span><span class="sxs-lookup"><span data-stu-id="05938-109">Validating the devices in pre-production environments guarantees that new versions of Direct Routing code in the cloud will work with certified SBCs.</span></span> 
- <span data-ttu-id="05938-110">SBC ベンダーとの共同サポート プロセスを実施します。</span><span class="sxs-lookup"><span data-stu-id="05938-110">Has a joint support process with the SBC vendors.</span></span>
