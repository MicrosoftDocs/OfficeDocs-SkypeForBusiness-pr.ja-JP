# <a name="how-to-look-up-your-support-pin"></a>サポート PIN を検索する方法

電話で Microsoft カスタマー サポートを有効にするサポート PIN を検索するには、次の手順に従ってください。 

1. [Office管理センター](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/homepage)にログインします。 

2. 歯車アイコンをクリックします。

3. サポート PIN が右下のウィンドウに表示されます。 必要に応じて PIN を再作成することもできます。  

   ![サポート PIN を示す図。](media/support-pin.png)
