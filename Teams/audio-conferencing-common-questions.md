# <a name="audio-conferencing-common-questions"></a>電話会議に関するよくある質問

電話会議を使用したいお客様から寄せられる質問の一部を次に示します。
  
## <a name="what-are-the-benefits-of-audio-conferencing"></a>電話会議の利点は何ですか?

会議への呼び出しは、たとえば、ユーザーが出張中で、ノート PC やモバイル デバイス上の Microsoft Teams アプリを使用して会議に出席できない場合に便利です。 ただし、コンピューターでアプリを使用するよりも、電話を使用して Teams 会議に参加する方が適しているシナリオは他にもあります。
  
- インターネット接続が制限されている。

- 会議が音声のみ。

- そのユーザーが Teams 会議に参加しようとして失敗しました。

- 通話品質は、ダイヤルインした方が優れています。

- Bluetooth デバイスを使用すると「ハンズフリー」で会議に参加できる。

- 状況によっては、そのほうが簡単で便利なことがあります。

## <a name="who-can-attend-an-audio-conferencing-meeting-and-who-can-i-hear"></a>電話会議会議に参加できるユーザー そして、誰が聞こえますか?

ダイヤルイン番号と会議 ID を持つすべてのユーザーは、会議の開催者が会議をロックしていない限り、Teams 会議に参加できます。
  
電話または Teams アプリを使用して通話を行う場合でも、通話中に他のすべてのユーザーが音声を聞くことができます。 会議の開催者は、会議の出席者が聞きたくない場合は、会議の出席者を "ミュート" することができます。
  
## <a name="can-i-add-a-toll-free-number-for-my-audio-conferencing-users"></a>電話会議ユーザーのフリーダイヤル番号を追加できますか?

はい。フリーダイヤル電話番号 (サービス番号) は一部の国/地域でのみご利用いただけます。 使用可能な番号の一覧については、「 [電話会議と通話プランの国と地域の可用性](country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md)」を参照してください。
  
## <a name="how-many-local-dial-in-numbers-are-currently-supported"></a>現在サポートされているローカル ダイヤルイン番号はいくつですか?

電話会議のライセンスを購入するときに割り当てられるローカル ダイヤルイン番号があります。 ダイヤルイン番号は会議出席依頼に含まれます。 これらのローカル番号は、組織でのみ使用できます。 組織に割り当てられた電話とその番号は、電話会議が有効になっている組織内のユーザーによって共有されます。 つまり、ユーザー A と別のユーザー B によってスケジュールされた Teams 会議のダイヤルイン番号は同じです。
  
ローカル ダイヤルイン番号と場合によっては、組織が配置されている国の国際ダイヤルイン番号が会議出席依頼に含まれます。 会議の出席者が招待に含まれている別の番号を使用する場合は、共有電話番号になります。
  
## <a name="how-many-international-dial-in-numbers-does-audio-conferencing-in-microsoft-365-or-office-365-support"></a>Microsoft 365 または Office 365 でサポートされる国際ダイヤルイン番号はいくつですか?

現在の国/地域の一覧については、「[Microsoft Teams の電話会議の電話番号](phone-numbers-for-audio-conferencing-in-teams.md)」を参照してください。
  
## <a name="what-is-the-maximum-length-of-the-audio-conferencing-meetings"></a>電話会議会議の最大長は何ですか?

最大時間は、会議に参加しているユーザーと、会議に参加するために使用した認証の種類によって異なります。
  
|**会議出席者**|**会議の終了時刻**|
|:-----|:-----|
|ユーザーは Teams アプリを使用して参加しているか、会議にダイヤルインしています。  <br/> |24 時間後に出席者リストに変更がない場合、会議は終了します。  <br/> |
|すべてのユーザーが会議にダイヤルインされますが、他のユーザーが PIN を使用して会議に参加しました。  <br/> |会議は 24 時間後に終了します。  <br/> |
|すべてのユーザーが会議にダイヤルインされますが、PIN を使用して会議に参加するユーザーは誰もいませんでした。  <br/> | 匿名ユーザーが会議を開始できる場合、会議は 4 時間後に終了します。 匿名ユーザーが会議を開始することが許可されず、認証されたすべての参加者が会議を終了した場合、最後に認証された参加者が終了してから 90 分後に会議が終了します。 <br/> |

## <a name="why-did-users-start-receiving-emails-with-their-audio-conferencing-information"></a>ユーザーが電話会議情報を含むメールを受信し始めたのはなぜですか?

[管理者](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504)が電話会議情報と電子メールの PIN を送信および更新できる新しい機能が追加されました。 無効にする方法など、詳細については、「[Microsoft Teams で電話会議の設定が変更されたときにメールの送信を有効または無効にする](enable-or-disable-sending-emails-when-their-settings-change-in-teams.md)」を参照してください。
  
## <a name="can-a-user-get-a-personal-conference-id"></a>ユーザーは個人用会議 ID を取得できますか?

Teams ユーザーには、ランダムに割り当てられた会議 ID が割り当てられます。 ユーザーは、自分だけが使用できる静的な会議 ID を予約または設定できません。
  
## <a name="can-a-user-get-operator-assistance-during-a-meeting"></a>ユーザーは会議中にオペレーターの支援を受けることができますか?

いいえ。ユーザーは、会議中に *0 を押すことでオペレーターの支援やサポートを受けることはできません。 電話会議に問題がある場合、組織の管理者は[、Office 365のサポートMicrosoft](https://support.office.com/article/Microsoft-support-for-Office-365-32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b)連絡できます。
  
## <a name="how-does-a-user-access-or-change-their-conference-id"></a>ユーザーが電話会議 ID にアクセスまたは変更する方法

Teams ユーザーは、Outlook とOutlook on the webで会議をスケジュールすることで、割り当てられた会議 ID を見つけることができます。 また、ユーザーは、セットアップ後に送信されるメール内の会議 ID を見つけることができます。
  
> [!NOTE]
> ユーザーは電話会議 ID をリセットできません。 会議 ID は、 [組織の管理者](https://support.office.com/article/admin-eac4d046-1afd-4f1a-85fc-8219c79e1504)のみがリセットできます。
  
ユーザーが組織の管理者の助けを借りずに電話会議 ID にアクセスしてリセットできるようにするソリューションに取り組んでいます。

## <a name="how-do-users-access-or-change-their-pin"></a>ユーザーが PIN にアクセスしたり、PIN を変更したりする方法

Teams ユーザーは、セットアップ後に送信される電子メールで PIN を見つけることができます。

> [!NOTE]
> Teams ユーザーはピンを変更できます。 会議出席依頼の下部に、ピンをリセットするためのリンクがあります。
  
ユーザーが組織の管理者の助けを借りずに PIN にアクセスしてリセットできるようにするソリューションに取り組んでいます。

## <a name="can-attendees-dial-out-to-international-phone-numbers-when-they-are-in-a-teams-meeting"></a>出席者が Teams 会議中に国際電話番号にダイヤルアウトできますか?

はい。出席者は、国際的にダイヤルアウトし、他の発信者を Teams 会議に招待できます。 [他のユーザーが参加できるように、Microsoft Teams 会議からのダイヤルアウトに関](dialing-out-from-a-teams-meeting-so-other-people-can-join-it.md)するページを参照してください。
  
## <a name="how-does-a-teams-user-schedule-a-meeting-with-audio-conferencing-meeting-details"></a>Teams ユーザーは、電話会議会議の詳細を使用して会議をスケジュールする方法を説明します。

ユーザーに **電話会議** ライセンスが割り当てられ、ユーザーが Outlook またはOutlook on the webで新しい Teams 会議を作成すると、ダイヤルイン電話番号と会議 ID が会議出席依頼に自動的に追加されます。
  
## <a name="how-does-a-user-schedule-and-start-a-meeting-when-all-attendees-will-be-using-a-phone-to-dial-in"></a>すべての出席者が電話を使用してダイヤルインする場合、ユーザーはどのように会議をスケジュールして開始しますか?

電話を使用してダイヤルインするすべての出席者が参加する会議をスケジュールすることは、通常のオンライン会議のスケジュールと変わるものではありません。 ただし、すべての参加者が電話を使用してダイヤルインする会議を開始するには、次の 2 つの方法があります。

- **オプション #1**: 既定では、会議の開催者とすべての参加者が電話を使用して会議に参加している場合、会議の開催者は電話会議 PIN を入力して会議を開始する必要があります。 発信者は、オンライン会議の電話番号をダイヤルするときに、特定の会議の開催者として認証するかどうかを尋ねられます。 開催者が開始する前にダイヤルインを介して会議に参加するすべての参加者はロビーに配置され、保留中に音楽を聴きます。 Teams 会議の場合、参加者は、開催者の会議ポリシーの [ユーザーを自動的に許可する] 設定の値に従って会議に参加します。

- **オプション #2**: [認証されていない発信者が会議の最初のユーザーであることを許可する] 設定は既定で無効になっています。 特定の開催者に対して設定が有効になっている場合、開催者が電話会議 PIN を入力する前に、その開催者によってスケジュールされたすべての会議を開始できます。 この設定を有効にすると、最初の参加者がダイヤルイン電話番号で参加するとすぐに会議が開始され、最初の参加者はロビーに配置されません。 詳細については、「[Microsoft Teams でユーザーの電話会議設定を管理する](manage-the-audio-conferencing-settings-for-a-user-in-teams.md)」を参照してください。
