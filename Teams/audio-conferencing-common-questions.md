# <a name="audio-conferencing-common-questions"></a>電話会議に関するよくある質問

電話会議を使用したいお客様から寄せられる主な質問の一部を次に示します。
  
## <a name="what-are-the-benefits-of-audio-conferencing"></a>電話会議の利点は何ですか?

たとえば、ユーザーが移動中で、ノート PC やモバイル デバイスで Microsoft Teams アプリを使用して会議に出席できない場合、会議に参加することは有益です。 ただし、電話を使用して Teams 会議に出席するシナリオは、コンピューターでアプリを使用するよりも優れた方法です。
  
- インターネット接続が制限されている。

- 会議が音声のみ。

- ユーザーが Teams 会議に参加しようとしましたが、失敗しました。

- ダイヤルインする場合は、通話の品質が向上します。

- Bluetooth デバイスを使用すると「ハンズフリー」で会議に参加できる。

- 状況によっては、そのほうが簡単で便利なことがあります。

## <a name="who-can-attend-an-audio-conferencing-meeting-and-who-can-i-hear"></a>電話会議会議に参加できるのはだれですか? また、だれが読み上げられますか?

ダイヤルイン番号と会議 ID を持つすべてのユーザーは、会議の開催者が会議をロックしていない限り、Teams 会議に参加できます。
  
電話または Teams アプリを使用して通話する場合でも、通話中の他のすべてのユーザーの音声が聞こえてきます。 会議の開催者は、会議の出席者の音声を聞きたくない場合に、会議の出席者を "ミュート" する機能を持っています。
  
## <a name="can-i-add-a-toll-free-number-for-my-audio-conferencing-users"></a>電話会議ユーザーのフリーダイヤル番号を追加できますか?

はい。フリーダイヤル電話番号 (サービス番号) を使用できますが、一部の国/地域でのみ使用できます。 使用可能な番号の一覧については、「 [電話会議と通話プランの国と地域の可用性](country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md)」を参照してください。
  
## <a name="how-many-local-dial-in-numbers-are-currently-supported"></a>現在サポートされているローカル ダイヤルイン番号はいくつですか?

電話会議のライセンスを購入するときに割り当てられるローカル ダイヤルイン番号があります。 ダイヤルイン番号は、会議の招待に含まれます。 これらのローカル番号は、組織でのみ使用できます。 組織に割り当てられている電話とその番号は、電話会議が有効になっている組織内のユーザーによって共有されます。 つまり、ユーザー A と別のユーザー B によってスケジュールされた Teams 会議のダイヤルイン番号は両方とも同じになります。
  
ローカル ダイヤルイン番号と、組織が所在する国からの国際ダイヤルイン番号が会議出席依頼に含まれる場合もあります。 会議出席者が招待に含まれる別の番号を使用している場合は、共有電話番号になります。
  
## <a name="how-many-international-dial-in-numbers-does-audio-conferencing-in-microsoft-365-or-office-365-support"></a>Microsoft 365 または Office 365 の電話会議でサポートされている国際ダイヤルイン番号の数はいくつですか?

現在の国/地域の一覧については、 [Microsoft Teams の電話会議の電話番号に関するページを参照してください](phone-numbers-for-audio-conferencing-in-teams.md)。
  
## <a name="what-is-the-maximum-length-of-the-audio-conferencing-meetings"></a>電話会議会議の最大長はどのくらいですか?

最大時間は、会議に参加しているユーザーと、会議への参加に使用した認証の種類によって異なります。
  
|**会議の出席者**|**会議の終了時刻**|
|:-----|:-----|
|ユーザーが Teams アプリを使用して参加しているか、会議にダイヤルインしています。  <br/> |24 時間後に出席者リストに変更がない場合、会議は終了します。  <br/> |
|すべてのユーザーが会議にダイヤルインされますが、他のユーザーが会議に参加するために PIN を使用しています。  <br/> |会議は 24 時間後に終了します。  <br/> |
|すべてのユーザーが会議にダイヤルインされますが、会議に参加するために PIN を使用したユーザーはだれもいませんでした。  <br/> | 匿名ユーザーが会議の開始を許可されている場合、会議は 4 時間後に終了します。 匿名ユーザーが会議を開始できず、認証されたすべての参加者が会議を離れた場合、会議は最後に認証された参加者が退席してから 90 分後に終了します。 <br/> |

## <a name="how-many-total-phone-participants-can-i-have-in-meetings"></a>会議に参加できる電話参加者の総数はどれくらいですか?

会議で許可される出席者の合計数については、「 [制限と仕様](limits-specifications-teams.md) 」を参照してください。

会議の制限については、「 [会議と通話](limits-specifications-teams.md#meetings-and-calls)」を参照してください。
  
## <a name="why-did-users-start-receiving-emails-with-their-audio-conferencing-information"></a>ユーザーが電話会議情報を含む電子メールの受信を開始した理由

[管理者](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504)は、電話会議情報と PIN を電子メールで送信および更新できる新機能を追加しました。 無効にする方法など、詳細については、「 [Microsoft Teams で電話会議の設定が変更されたときに電子メールの送信を有効または無効にする](enable-or-disable-sending-emails-when-their-settings-change-in-teams.md)」を参照してください。
  
## <a name="can-a-user-get-a-personal-conference-id"></a>ユーザーは個人の会議 ID を取得できますか?

Teams ユーザーには、会議 ID がランダムに割り当てられます。 ユーザーは、自分だけが使用できる静的な会議 ID を予約または設定できません。
  
## <a name="can-a-user-get-operator-assistance-during-a-meeting"></a>ユーザーは会議中にオペレーターのサポートを受けることができますか?

いいえ。ユーザーは、会議中に *0 を押してオペレーターのサポートやサポートを受けることはできません。 電話会議に問題がある場合は、組織の管理者が [Microsoft サポートに連絡してOffice 365](https://support.office.com/article/Microsoft-support-for-Office-365-32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b)。
  
## <a name="how-does-a-user-access-or-change-their-conference-id"></a>ユーザーが会議 ID にアクセスまたは変更する方法

Teams ユーザーは、Outlook とOutlook on the webで会議をスケジュールすることで、自分に割り当てられている会議 ID を見つけることができます。 また、ユーザーは、セットアップ後に送信される会議 ID を電子メールで見つけることができます。
  
> [!NOTE]
> ユーザーは会議 ID をリセットできません。 会議 ID は、 [組織の管理者](https://support.office.com/article/admin-eac4d046-1afd-4f1a-85fc-8219c79e1504)のみがリセットできます。
  
組織の管理者の助けを借りずに、ユーザーが会議 ID にアクセスしてリセットできるようにするソリューションに取り組んでいます。

## <a name="how-do-users-access-or-change-their-pin"></a>ユーザーが PIN にアクセスまたは変更する方法

Teams ユーザーは、セットアップ後に送信される電子メールで PIN を見つけることができます。

> [!NOTE]
> Teams ユーザーはピンを変更できます。 会議出席依頼の下部にピンをリセットするためのリンクがあります。
  
ユーザーが組織の管理者の助けを借りずに PIN にアクセスしてリセットできるようにするソリューションに取り組んでいます。
  
## <a name="what-in-meeting-dial-pad-commands-are-supported"></a>どのような会議内ダイヤルパッド コマンドがサポートされていますか?

会議の参加者と開催者の場合:

| コマンド名    | コマンド キー | 説明                                                                 |
|--------------   |--------------|-----------------------------------------------------------------------------|
| コマンド メニュー    | \*1          | 1 キーを押して \*、使用可能なダイヤル パッド コマンドの説明を再生します。 |
| 手を上げる/下げる| \*5          | 5 キーを押 \*して、手を上げたり下げたりします。                                           |  
| ミュート/ミュート解除     | \*6          | 6 キーを押して \*、ミュートまたはミュート解除を行います。                                     |

会議の開催者の場合のみ:

| コマンド名                   | コマンド キー | 説明                                                 |
|--------------------------------|--------------|-------------------------------------------------------------|
| すべて許可する                      | \*21         | ロビーで待機しているすべての参加者を許可するには、21 キーを押 \*します。  |
| すべてミュートする                       | \*22         | 22 キーを押して \*、すべての参加者をミュートします (開催者を除く)。 |
| Enter/Exit のお知らせを切り替える | \*23         | Enter/Exit Announcement を有効または無効にするには、23 キーを押 \*します。    |

## <a name="can-attendees-dial-out-to-international-phone-numbers-when-they-are-in-a-teams-meeting"></a>Teams 会議に参加しているときに、出席者は国際電話番号にダイヤルアウトできますか?

はい。出席者は国際電話でダイヤルアウトし、他の発信者を Teams 会議に招待できます。 [他のユーザーが参加できるように、Microsoft Teams 会議からダイヤルアウトする](dialing-out-from-a-teams-meeting-so-other-people-can-join-it.md)方法に関するページを参照してください。
  
## <a name="how-does-a-teams-user-schedule-a-meeting-with-audio-conferencing-meeting-details"></a>Teams ユーザーが電話会議会議の詳細を使用して会議をスケジュールする方法

ユーザーに **電話会議** ライセンスが割り当てられ、ユーザーが Outlook またはOutlook on the webで新しい Teams 会議を作成すると、ダイヤルイン電話番号と会議 ID が会議の招待に自動的に追加されます。
  
## <a name="how-does-a-user-schedule-and-start-a-meeting-when-all-attendees-will-be-using-a-phone-to-dial-in"></a>すべての出席者が電話を使用してダイヤルインする場合、ユーザーはどのようにスケジュールして会議を開始しますか?

電話を使用してダイヤルインするすべての出席者が参加する会議のスケジュール設定は、通常のオンライン会議のスケジュールと変わりはありません。 ただし、すべての参加者が電話を使用してダイヤルインする会議を開始するには、次の 2 つの方法があります。

- **オプション #1**: 既定では、会議の開催者とすべての参加者が電話を使用して会議に参加している場合、会議の開催者は電話会議 PIN を入力して会議を開始する必要があります。 発信者は、オンライン会議の電話番号をダイヤルするときに、特定の会議の開催者として認証するかどうかを尋ねられます。 開催者が開始する前にダイヤルインを使用して会議に参加するすべての参加者はロビーに配置され、保留音を聞きます。 Teams 会議の場合、参加者は、開催者の会議ポリシーで自動的にユーザーを許可する設定の値に従って会議に参加します。

- **オプション #2**: [認証されていない発信者を会議の最初のユーザーに許可する] 設定は既定で無効になっています。 特定の開催者に対して設定が有効になっている場合、開催者が電話会議 PIN を入力する前に、その開催者によってスケジュールされたすべての会議を開始できます。 この設定を有効にすると、最初の参加者がダイヤルイン電話番号を介して会議に参加するとすぐに会議が開始され、最初の参加者はロビーに配置されません。 詳細については、「 [Microsoft Teams でユーザーの電話会議設定を管理する」を参照してください](manage-the-audio-conferencing-settings-for-a-user-in-teams.md)。
