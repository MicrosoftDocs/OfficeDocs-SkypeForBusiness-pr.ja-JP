# <a name="microsoft-teams-rooms"></a>Microsoft Teams ミーティング

小規模の huddle 領域から大人数の会議室への会議スペースの変換は、使いやすく、展開、管理が簡単で、共同作業を行う豊富なチームエクスペリエンスを備えています。

ワンタッチ参加で会議を時間どおりに開始し、即座に会議室内ディスプレイに表示し、リモート参加者と共有できます。

Microsoft のパートナー (Yealink、Logitech、Crestron、Polycom、Lenovo、HP) から、適切なシステムやオーディオ ビデオ周辺機器を選択してください。
