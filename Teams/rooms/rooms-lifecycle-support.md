<span data-ttu-id="2d7cc-104">Microsoft Teams ミーティング アプリは年に数回更新されます。</span><span class="sxs-lookup"><span data-stu-id="2d7cc-104">The Microsoft Teams Rooms app gets updates a few times per year.</span></span> <span data-ttu-id="2d7cc-105">各更新プログラムは、一般提供 (GA) のリリース日から 12 か月間サポートされます。</span><span class="sxs-lookup"><span data-stu-id="2d7cc-105">Each update supported for twelve (12) months from its general availability (GA) release date.</span></span> <span data-ttu-id="2d7cc-106">テクニカル サポートは 12 か月を通して提供されます。</span><span class="sxs-lookup"><span data-stu-id="2d7cc-106">Technical support is provided for the entire twelve (12) months.</span></span> <span data-ttu-id="2d7cc-107">ただし、サポート体制は動的であり、最新バージョンの提供可能性によって次の 2 つのフェーズがあります。</span><span class="sxs-lookup"><span data-stu-id="2d7cc-107">However, the support structure is dynamic, with two distinct phases that depend on the availability of the latest version:</span></span>

- <span data-ttu-id="2d7cc-108">**サービス提供および重要な更新プログラム フェーズ** \- Microsoft Teams ミーティング アプリの最新バージョンを実行すると、*セキュリティおよびサービス提供*更新プログラムを含む定期更新プログラムを受け取ります。</span><span class="sxs-lookup"><span data-stu-id="2d7cc-108">**Servicing and Critical Updates phase** \- When you run the latest version of the Microsoft Teams Rooms app, you receive regular updates that contain *Security and Servicing* updates.</span></span>

- <span data-ttu-id="2d7cc-109">**セキュリティ更新プログラムのみフェーズ** \- Microsoft Teams ミーティング アプリの新しいバージョンがリリースされると、そのアプリの以前のバージョンのサポート レベルが、12 か月のライフサイクルの残りの*セキュリティ更新プログラムのみ*に削減されます。</span><span class="sxs-lookup"><span data-stu-id="2d7cc-109">**Security Updates Only phase** \- When a new version of the Microsoft Teams Rooms app releases, older versions of the app have a reduced support level with *Security updates only* for the rest of the twelve (12) month lifecycle.</span></span>

> [!NOTE]
> <span data-ttu-id="2d7cc-110">最新バージョンは、常に、サービス提供および重要な更新プログラム フェーズになります。</span><span class="sxs-lookup"><span data-stu-id="2d7cc-110">The latest version is always in the Servicing and Critical Updates phase.</span></span> <span data-ttu-id="2d7cc-111">また、重要な更新プログラムが必要なコードの問題が発生した場合は、最新バージョンをインストールして修正プログラムを受け取る必要があります。</span><span class="sxs-lookup"><span data-stu-id="2d7cc-111">When you encounter a code defect that warrants a critical update, you must also have the latest version installed to receive a fix.</span></span> <span data-ttu-id="2d7cc-112">サポートされているその他のすべてのバージョンは、セキュリティ更新プログラムを受け取ることしかできません。</span><span class="sxs-lookup"><span data-stu-id="2d7cc-112">All other supported versions will only be eligible to receive security updates.</span></span>

<span data-ttu-id="2d7cc-113">すべてのサポートは、バージョンの 12 か月のライフサイクルの終了後またはそれ以降に 2 つ以上の更新プログラムがリリースされた場合に終了します。</span><span class="sxs-lookup"><span data-stu-id="2d7cc-113">All support ends after the twelve (12) month lifecycle for a version has expired or if more than two updates have been released since then.</span></span> <span data-ttu-id="2d7cc-114">その後は、お客様が、サポートされているバージョンに更新する必要があります。</span><span class="sxs-lookup"><span data-stu-id="2d7cc-114">Then, customers must update to a supported version.</span></span>

<span data-ttu-id="2d7cc-115">すべてのリリースが、[Microsoft Teams ミーティング リリース ノート](rooms-release-note.md)に記載されています。</span><span class="sxs-lookup"><span data-stu-id="2d7cc-115">All releases are listed in the [Microsoft Teams Rooms release notes](rooms-release-note.md).</span></span>
