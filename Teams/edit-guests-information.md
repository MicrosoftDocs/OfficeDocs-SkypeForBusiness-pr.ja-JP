<a name="edit-guest-user-information"></a>ゲスト ユーザー情報を編集します。
===========================

(名前またはプロファイル写真の表示) などのゲスト アカウントを編集するには、Azure Active Directory ポータルに移動します。詳細については、 [Office 365 の理解度 id と Azure Active Directory](https://support.office.com/en-us/article/Understanding-Office-365-Identity-and-Azure-Active-Directory-06a189e7-5ec6-4af2-94bf-a22ea225a7a9)を参照してください。

現在、ゲストについては、Office 365 管理センターまたは Exchange 管理センターからは編集できません。
