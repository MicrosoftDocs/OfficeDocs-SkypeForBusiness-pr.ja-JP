# <a name="pstn-service-desk-contact-information"></a>PSTN サービスデスクの連絡先情報

PSTN サービスデスクチームに問い合わせるには、次の表に記載されている適切なエイリアスにメールを送信します。

| **既存の電話番号を移行/移植する**  | **の新しい番号を注文する** |
|:-----:|:------:|
| **米国**または**カナダ**国内の場合は、 [ptn@microsoft.com](mailto:ptn@microsoft.com)にメールを送信**してください**。| **ベルギー、フランス、ドイツ、英国、アイルランド、オランダ、スペイン、スウェーデン、スイス**、 [ptneu@microsoft.com](mailto:ptneu@microsoft.com)にメールを送信します。|
|**米国**または**カナダ****以外**の場合は、 [ptneu@microsoft.com](mailto:ptneu@microsoft.com)にメールを送信してください。| **残りの部分**では、 [gcstnmsd@microsoft.com](mailto:gcstnmsd@microsoft.com)にメールを送信します。|

> [!NOTE]
> 受信要求が大幅に増加したため、北米の応答が遅れる可能性があります。 バックログは FIFO (先入れ先出し) ベースで消去されます。

## <a name="transfer-or-port-your-existing-phone-numbers-to-microsoft-teams"></a>既存の電話番号を Microsoft Teams に転送または移植する
上の表から適切なメールエイリアスにメールを送信します。 次のような、可能な限り多くの情報を含めます。
  - [組織 ID (テナント id) または現在アクティブな Microsoft ドメイン](https://docs.microsoft.com/onedrive/find-your-office-365-tenant-id)。
  - サポートが必要な電話番号の種類ユーザー、有料サービス、有料サービスなど。
  - お客さまのリクエストに基づいて、その他の重要な詳細情報。

## <a name="get-new-phone-numbers-for-microsoft-teams"></a>Microsoft Teams の新しい電話番号を取得する
上の表から適切なメールエイリアスにメールを送信します。 次のような、可能な限り多くの情報を含めます。
  - [組織 ID (テナント id) または現在アクティブな Microsoft ドメイン。](https://docs.microsoft.com/onedrive/find-your-office-365-tenant-id)
  - サポートが必要な電話番号の種類ユーザー、有料サービス、有料サービスなど。
  - 必要な電話番号の数と、新しい電話番号の発信元となる市外局番 (s)、または国または地域。
