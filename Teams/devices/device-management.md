# <a name="manage-your-devices-in-microsoft-teams"></a>Microsoft Teams でのデバイスの管理

組織の管理センターから、Microsoft Teamsで使用されるデバイスMicrosoft Teams管理できます。 組織のデバイス インベントリを表示および管理し、デバイスの診断の更新、再起動、監視などのタスクを実行できます。 構成プロファイルを作成して、デバイスまたはデバイスのグループに割り当てすることもできます。

デバイス構成の変更、デバイスの再起動、更新プログラムの管理、デバイスと周辺機器の正常性の表示などのデバイスを管理するには、次のいずれかの管理者ロールMicrosoft 365必要があります。

- Microsoft 365グローバル管理者

管理者ロールの詳細については、「管理者ロールを使用Teams Teams管理者ロールを使用[Teams」を参照Teams。](../using-admin-roles.md)

### <a name="use-configuration-profiles-in-teams"></a>アプリケーションで構成プロファイルを使用Teams

構成プロファイルを使用して、組織内の異なる Teams デバイス (コラボレーション バー、Teams ディスプレイ、Teams 電話、Teams パネルなど) の設定と機能を管理します。 構成プロファイルを作成またはアップロードして、有効または無効にする設定と機能を含め、デバイスまたは一連のデバイスにプロファイルを割り当てできます。 

#### <a name="create-a-configuration-profile"></a>構成プロファイルを作成する

デバイスの種類に合った構成Teams作成するには:

1. 左側のナビゲーションで、[デバイス] に移動 **し>構成** プロファイルTeamsデバイスの種類>**選択します**。 たとえば、[デバイス]**パネルTeams**  >  **構成** プロファイル] を選択して、新しい構成プロファイルを作成Teams  >  します。
2. **[追加]** をクリックします。
3. プロファイルの名前を入力し、必要に応じて説明を追加します。
4. プロファイルの設定を指定し、[保存] を **クリックします**。
   新しく作成された構成プロファイルがプロファイルの一覧に表示されます。

3. [構成 **の割り当て** ] ウィンドウで、選択したデバイスに割り当てる構成プロファイルを検索します。
4. [**適用**] をクリックします。
   構成ポリシーを適用したデバイスの場合、[アクション]列に [構成の更新] と表示 **され、[構成** プロファイル] 列に構成プロファイル名が表示されます。
