# <a name="usb-audio-and-video-devices-certified-for-microsoft-teams"></a>MICROSOFT TEAMS認定のUSBオーディオ/ビデオデバイス

周辺機器の認定プログラムは、USB 周辺機器(ヘッドセット、スピーカーフォン、ウェブカメラ、モニターなど)が、オーディオやビデオの品質を高く提供します。 認定デバイスは、追加の設定が不要で機能(プラグ&再生)、Microsoft TeamsとSkype for Businessでコールコントロールを提供します。 製品は、エンド ユーザーと管理者に優れたエクスペリエンスを提供します。

Microsoft Teamsのテストおよび認定を受けたデバイスを探索するには[、Teamsストア](https://products.office.com/microsoft-teams/across-devices/devices)に向かいます。

Teamsでのコール制御のサポートの詳細については[、「Teamsのヘッドセットを使用したコントロールコール](https://support.office.com/article/Control-calls-using-a-headset-in-Teams-65d6e104-444d-4013-b8c2-f11317dd69a8)」を参照してください。

> [!TIP]
> Skype for Business認定デバイスをお探しですか? Skype for Business認定の[USBオーディオ/ビデオデバイスに向](/skypeforbusiness/certification/devices-usb-devices)かいます。

## <a name="understanding-certification-categories"></a>認定カテゴリについて

ユーザーは、Skype for BusinessからTeamsにアップグレードする際に、以前に認定されたデバイスを交換する必要はありません。  すべての認定デバイスは、Microsoft TeamsとSkype for Businessユーザーに対してサポートされていますが、わずかな違いがあります。  認定はまた、特定のデバイスカテゴリのさらなる指定を提供します。

- **Teamsアップグレードで認定Skype for Business** このレベルで認定されたデバイスは、当初、Skype for Business認定を受けました。 Microsoft は、デバイスメーカーと協力して、元のハードウェアに変更を加えることなく、可能な限り最高のTeams統合エクスペリエンスを提供してきました。 *フックスイッチ* ボタンは、呼び出し時にTeams ボタンとして機能し、時間の経過に合った機能が追加される場合があります。  このカテゴリのデバイスには、マーケティング資料にTeams認定バッジが付く場合があります。
- **Microsoft Teams認定** このレベルで認定されたデバイスは、妥協のないTeamsエクスペリエンスのために、認定された最新のデバイスを示します。 これらのデバイスは、以前の認定の利点のすべてを提供し、LEDインジケータ付き専用のTeamsボタンも備えています。 ほとんどのフォームファクターでは、TeamsボタンはTeamsロゴが付いた見つけやすい触覚ボタンになります。 マイクロソフトは、デバイスメーカーと協力して、ファームウェアの更新を通じて時間の経過とともに新しい価値を提供し続けています。

### <a name="special-usage-designations"></a>特別な使用指定

- **オープンOffice用のプレミアムマイク** すべてのヘッドセットは、明確なオーディオを提供するためにテストされていますが、オープンオフィスやその他の騒がしい環境は、より大きな課題を提起します。 この指定のヘッドセットは、ヘッドセットユーザーの声を拾うが、隣人の音を送信しない。
- **会議室のサイズ** スピーカーフォンやカメラなどの共有デバイスの使用距離はサポートされています。 推奨距離は製造元によって異なる場合があります。 お客様に明確なガイダンスを提供するために、すべての共有デバイスは、明確に定義された部屋のサイズとノイズ環境に対して認定でテストされます。

[Microsoft Teamsの認定デバイスを](https://products.office.com/microsoft-teams/across-devices/devices)参照する 。

## <a name="end-user-experience"></a>エンドユーザーエクスペリエンス

### <a name="new-teams-certification-benefits"></a>新しいTeams認定特典

- 専用TeamsボタンとLEDインジケータライト。
- Teams クライアントからのイベントやアラートのユーザー通知 (会議の開始、ボイスメール、不在着信など)。
- Teamsクライアントをアクティブにしたり、ボタンを押して通知に応答する機能 (たとえば、会議に参加する)。
- 拡張性 (ファームウェア更新プログラムを使用して提供される新しいデバイス/クライアント機能) を対象に設計された認定。

### <a name="core-certification-benefits"></a>主要な認定特典

- 複数のオーディオ周辺機器が存在する場合は、デフォルトのオーディオデバイスと優先度設定の自動選択。
- デバイスは、Teams クライアントに自動的に登録され、使用できる状態になります。
- オーディオ デバイス (ヘッドセット、スピーカーフォンなど) には次の機能があります。
  - PC または Mac デバイスで使用する場合の基本的なコール制御。 基本的なコントロールには、応答/ハングアップ、ミュート/ミュート解除、音量調節などがあります。
  - エコー、歪み、または過度のグリッチのないワイドバンドオーディオ。 エコー キャンセレーションは、デバイス間でもサポートされています。
- ビデオ デバイスは、ノイズ、カラー、画像の詳細、ジッタ、遅延、フレーム レート、および視野の要件を満たします。

## <a name="administrator-experience"></a>管理者エクスペリエンス

- プロビジョニングは不要
- すべてのデバイスは、ファームウェアの更新をサポートし、新機能とパフォーマンスの向上を可能にします。

製造元の方で、認定プログラムに参加する場合は、「要件と利用可能なプログラム [の参加方法](/skypeforbusiness/certification/how-to-join) 」を参照してください。
