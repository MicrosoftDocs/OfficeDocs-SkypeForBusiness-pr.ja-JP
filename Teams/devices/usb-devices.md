# <a name="usb-audio-and-video-devices-certified-for-microsoft-teams"></a>Microsoft Teamsの認定を受けた USB オーディオおよびビデオ デバイス

周辺機器認定プログラムは、USB 周辺機器 (ヘッドセット、スピーカーフォン、Web カメラ、モニターなど) が豊富なオーディオまたはビデオ品質を提供することを保証します。 認定されたデバイスは、追加の構成が不要で (プラグ & プレイ) 動作し、Microsoft TeamsとSkype for Businessで通話制御を提供します。 この製品は、エンド ユーザーと管理者に優れたエクスペリエンスを提供します。

Microsoft Teamsのテストおよび認定を受けたデバイスを調べるには、[Teams ストア](https://products.office.com/microsoft-teams/across-devices/devices)に進みます。

Teamsでの通話制御のサポートの詳細については、[Teamsのヘッドセットを使用した制御呼び出しに](https://support.office.com/article/Control-calls-using-a-headset-in-Teams-65d6e104-444d-4013-b8c2-f11317dd69a8)関するページを参照してください。

> [!TIP]
> Skype for Businessの認定を受けたデバイスをお探しですか? [Skype for Business認定された USB オーディオおよびビデオ デバイスに進みます](/skypeforbusiness/certification/devices-usb-devices)。

## <a name="understanding-certification-categories"></a>認定カテゴリについて

Skype for BusinessからTeamsにアップグレードするときに、以前に認定されたデバイスを置き換える必要はありません。  認定されたすべてのデバイスは、Microsoft Teams ユーザーとSkype for Business ユーザーに対してサポートされていますが、違いはわずかです。  認定では、特定のデバイス カテゴリに対するさらなる指定も提供されます。

- このレベル **で認定されたTeamsアップグレード デバイスで認定されたSkype for Business** は、最初はSkype for Businessの認定を受けていました。 Microsoft は、デバイス製造元と協力して、元のハードウェアを変更することなく、可能な限り最高のTeams統合エクスペリエンスを提供してきました。 *フックスイッチ* ボタンは、呼び出し中でない場合はTeams ボタンとして機能し、時間の経過と共にさらに多くの関数が含まれる場合があります。  このカテゴリのデバイスは、マーケティング資料にTeams認定バッジを持つ場合があります。
- このレベルで **認定されたMicrosoft Teams** デバイスの認定は、妥協のないTeamsエクスペリエンスのために、認定された最新のデバイスを示します。 これらのデバイスは、以前の認定のすべての利点を提供し、LED インジケーターを備えた専用のTeams ボタンも含まれています。 ほとんどのフォーム ファクターでは、Teams ボタンは、Teams ロゴが付いた簡単に見つけることができるタッチ ボタンになります。 Microsoft は引き続きデバイス製造元と連携し、ファームウェアの更新を通じて新しい価値を提供しています。

### <a name="special-usage-designations"></a>特別な使用の指定

- **プレミアム マイク for Open Office** すべてのヘッドセットはクリア オーディオを提供するためにテストされますが、オフィスやその他のノイズの多い環境を開く方が大きな課題となります。 この指定を持つヘッドセットは、ヘッドセット ユーザーの声を受け取りますが、近隣の音声は送信しません。
- **会議室のサイズ** スピーカーフォンやカメラなどの共有デバイスでは、使用距離がサポートされます。 推奨される距離は、製造元によって異なる場合があります。 お客様に明確なガイダンスを提供するために、すべての共有デバイスは、明確に定義された部屋のサイズとノイズ環境に対する認定でテストされます。

[Microsoft Teams認定されたデバイスを](https://products.office.com/microsoft-teams/across-devices/devices)参照します。

## <a name="end-user-experience"></a>エンドユーザーのエクスペリエンス

### <a name="new-teams-certification-benefits"></a>新しいTeams認定特典

- 専用Teamsボタンと LED インジケーター ライト。
- Teams クライアントからのイベントとアラートのユーザー通知 (会議の開始、ボイスメール、不在着信など)。
- Teams クライアントをアクティブ化したり、ボタンを押して通知に応答したりする機能 (会議に参加するなど)。
- 機能拡張用に設計された認定 (ファームウェアの更新を介して提供される新しいデバイス/クライアント機能)。

### <a name="core-certification-benefits"></a>コア認定の利点

- 既定のオーディオ デバイスの自動選択と、複数のオーディオ周辺機器が存在する場合の優先順位付け。
- デバイスは自動的にTeams クライアントに登録され、使用できるようになります。
- オーディオ デバイス (ヘッドセット、スピーカーフォンなど) には、次の機能があります。
  - PC または Mac デバイスで使用する場合の基本的な通話制御。 基本的なコントロールには、応答/ハングアップ、ミュート/ミュート解除、およびボリューム コントロールが含まれます。
  - エコー、歪み、または過度の不具合がないワイド バンド オーディオ。 エコーの取り消しは、デバイス間でもサポートされています。
- ビデオ デバイスは、ノイズ、色、画像の詳細、ジッター、待機時間、フレーム レート、視野の要件を満たしています。

## <a name="administrator-experience"></a>管理者エクスペリエンス

- プロビジョニングは必要ありません
- すべてのデバイスは、新しい機能とパフォーマンスの向上を可能にするためにファームウェアの更新をサポートします。

製造元で認定プログラムに参加する場合は、「要件と利用可能なプログラム [に参加する方法](/skypeforbusiness/certification/how-to-join) 」を参照してください。
