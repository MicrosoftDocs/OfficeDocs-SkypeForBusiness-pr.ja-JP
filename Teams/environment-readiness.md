<a name="check-your-environments-readiness-for-microsoft-teams"></a>Microsoft チームの環境の準備状況を確認します。
===========================================

クラウドへの切り替えは、各組織によって異なります、現在の状態は、マイクロソフトのチームがどのように機能に影響を与える可能性があります。

最適な操作性を移動するには Microsoft チームで、Exchange Online と SharePoint Online 組織を導入されている必要があります。されない場合は、またはがさまざまな設定では、ヘルプについては、次のリンクを参照してください。

-   場合は、組織には、任意の Office 365 ワークロード展開されませんが、「[一般法人向け Office 365 使用を開始する。](https://support.office.com/en-us/article/Get-started-with-Office-365-for-Business-d6466f0d-5d13-464a-adcb-00906ae87029)

-   組織は追加または検証済みドメインを Office 365 用に構成されていないが、 [Office 365 ドメインを確認する](https://support.office.com/en-us/article/Verify-your-Office-365-domain-to-prove-ownership-nonprofit-or-education-status-or-to-activate-Yammer-87d1844e-aa47-4dc0-a61b-1b773fd4e590)を参照してください。

-   組織が Azure Active directory のユーザーを同期していない場合は、[マイクロソフトのチームで認証との Id のモデル](identify-models-authentication.md)を参照してください。

-   組織が Exchange Online を持たない場合は、 [Exchange と Microsoft チームの対話方法を理解する](Exchange-Teams-interact.md)を参照してください。

-   組織が SharePoint Online を持たない場合は、 [Microsoft チームの SharePoint Online と OneDrive for Business の対話方法を理解する](SharePoint-OneDrive-interact.md)を参照してください。
