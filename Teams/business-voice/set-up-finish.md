# <a name="finished-teams-phone-system-with-calling-plan-setup"></a>通話プランの設定で完了したTeams 電話システム

通話プランでTeams 電話システムの基本的なセットアップが完了しました。 通話プラン バンドルを使用して電話システムを管理し、ユーザーが新しい電話、会議、電話会議のエクスペリエンスを最大限に活用するのに役立つリソースを次に示します。

次の表に、Teamsと通話プラン バンドルの電話システムを最大限に活用するのに役立つリソースの一覧を示します。

## <a name="training"></a>トレーニング

次のトレーニング リソースを使用して、Teamsと通話プラン バンドルを使用した電話システムを使用して、自分とユーザーの速度を向上します。

|リソース  |説明  |
|---------|---------|
| [Microsoft Teams のトレーニング](../training-microsoft-teams-landing-page.md)    | Teams初心者の管理者とエンド ユーザー向けのトレーニング。        |
| [Microsoft Teamsクイック スタート ガイド](https://download.microsoft.com/download/D/9/F/D9FE8B9E-22F5-47BF-A1AB-09539C41FCD0/Teams%20QS.pdf)    | Teams クライアントと一般的なアクティビティを実行する方法を説明する印刷可能な PDF ファイル。        |
| [Teamsを電話として使用する方法](https://support.microsoft.com/office/meetings-and-calls-d92432d5-dd0f-4d17-8f69-06096b6b48a8?ad=US#ID0EAABAAA=Calls)    | 通話の発信と受信、ボイスメールの設定と確認、通話の転送とマージなどの方法に関するトレーニング。        |

## <a name="customize-teams-phone-system-with-calling-plan"></a>通話プランを使用してTeams 電話システムをカスタマイズする

次のリソースを使用して、特定の要件に合わせて通話プランを使用してTeams 電話システムをカスタマイズします。

| リソース | 説明  |
|---------|---------|
| [Microsoft Teams の発信通話制限ポリシー](set-up-policies.md)    | ユーザーがプライベート通話を行ったり、通話を転送したり、同時通話を設定したり、通話をボイスメールにルーティングしたりできるかどうかを制御します。        |
| [Microsoft Teamsでポリシー パッケージを管理する](policy-packages.md)    | 同様のロールのユーザーに割り当てることができるポリシーのグループを作成します。        |
| [通話プランユーザーを使用してTeams 電話システムを作成してライセンスを付与し、電話番号を割り当てる](create-users.md)    | 通話プランを使用してTeams 電話システムにユーザーを追加し、電話番号で設定します。        |
| [通話キューを作成する - 小規模ビジネスのチュートリアル](create-a-phone-system-call-queue-smb.md)    | 通話キューを作成して、セールス、カスタマー サービス、テクニカル サポートなどのコール センター環境をサポートします。        |
