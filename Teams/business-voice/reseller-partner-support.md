# <a name="get-help-from-a-microsoft-reseller-or-partner"></a>Microsoft 販売店または Microsoft パートナーからサポートを受ける

アカウントを設定Teams 電話システム、自分で設定するか、Microsoft パートナーまたは販売者を使用することができます。

Microsoft のリセラーまたはパートナーを見つけるには、パートナーと一緒Connect[ページを参照](https://appsource.microsoft.com/marketplace/partner-dir)してください。

Microsoft パートナーや販売店には次のことができます。

- テナントと電話Teams 電話の適切なライセンスを取得する
- アプリケーションの設定に関する技術的要件を理解電話システム
- 電話プロバイダーと調整して、電話番号が電話番号と一緒に使用するために正しく設定されていることを確認電話システム
- オンプレミスのテレフォニー ハードウェアと Microsoft 365 の間の通信を設定する
- 通話キュー、電話システム、ボイス メール自動応答機能を設定する

Microsoft パートナーまたはリセラーに問い合Teams 電話システム、次のような情報を提供する必要があります。

- 外部の電話番号との通話を発信および受信する必要があるユーザー、電話機、その他のデバイスの数
- 既存の電話システムの詳細情報
- お使Teams 電話システム PSTN プロバイダーとの接続に関する詳細
- インターネット接続
- 組織に関する独自の設計上の考慮事項。たとえば、コール センター、電話システムの通話メニュー (例: 営業は 1 を押す、テクニカル サポートは 2 を押す)、会議室、公衆電話室をサポートする必要があるなど。
- 既存の電話システムがサポートしている場所の数と、それらの間の接続の設定方法
