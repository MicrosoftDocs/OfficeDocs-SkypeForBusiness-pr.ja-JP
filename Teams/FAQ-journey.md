## <a name="meeting-capabilities"></a>会議機能

### <a name="is-audio-conferencing-coverage-in-teams-different-in-skype-for-business"></a>Teams と Skype for Business では電話会議のサービス エリアに違いはありますか?

Teams では電話会議がサポートされており、そのサービス エリアには変更はありません。 両方の製品で、現在提供されているサービス エリア (90 カ国以上および 400 以上の都市) で電話会議をご利用いただけます。 現在のリストは、[オーディオ会議や予定を呼び出すための国および地域の可用性](country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md)を参照してください。

### <a name="are-third-party-audio-conferencing-providers-acp-supported-in-teams"></a>サード ・ パーティ製の音声会議プロバイダー (ACP) は、チームでサポートされますか。

チームでサード ・ パーティ製の音声会議プロバイダー (ACP) をサポートする予定はありません。 ビジネスは、Office 365 (以前は PSTN 会議) で、オーディオ会議サービスを使用するは、チームと Skype を利用するお客様に最適な音声会議経験と考えています。 ビジネス会議のための Skype で、ACP サポートを活用する必要があるお客様は、ビジネス クライアント用の Skype を使用する続行できます。 Teams クライアントで予約した会議では、Office 365 の電話会議サービスのみがサポートされます。

### <a name="whats-the-plan-for-video-interoperability-support-for-teams-meetings"></a>チームの会議のビデオの相互運用性サポートのための計画とは何ですか。

現代的なワークプレースの実現に向けた Microsoft のビジョンでは会議室のデバイスが重要な位置を占めます。 VTC システムが既存のチーム会議をサポートするビデオの相互運用性サービスをクラウドは、進行中、秋に通常使用するには、Pexip、ポリコム、およびブルー ジーンズから製品にされています。 お客様は Skype for Business Online で Polycom 社の RealConnect for Office 365 をこれまでどおりご利用いただけます。

### <a name="will-the-latest-generation-of-skype-room-systems-v2-support-meetings-in-teams"></a>最新世代の Skype ルーム システム (v2) は、チームでの会議をサポートしますか。

会議室のデバイスは、現代の職場に対する私どものビジョンに重要な Skype ルームのシステムと当社の勢いを継続しました。 Skype ルーム システムの更新プログラムは、ユーザー近接検出と Skype ルーム システムの近くにある特定の機能だけでなく、チームのミーティングに参加するのにはされています。 デュアル スクリーンとパノラマ デバイスは今後のリリースでサポートされる予定です。

### <a name="will-skype-room-systems-v1-be-updated-to-support-teams-meetings"></a>Skype Room Systems V1 は Teams 会議をサポートするように更新されますか?

現代的なワークプレースの実現に向けた Microsoft のビジョンでは会議室のデバイスが重要な位置を占めます。 Skype ルーム システムの v1 (Lync ルームのシステムとも呼ばれます) には、チームのミーティングに参加するのにはこれらのデバイスを許可するアップグレード可能ながあります。 アップグレード プロセスの一環として、私たちがもこれらのデバイスから Windows 7 から 10 にアップグレード Windows デバイスのセキュリティを強化します。 タッチ スクリーンを使用して、それらのデバイスのタッチ機能およびインク、PowerPoint の注釈、およびホワイト ボードなどの機能へのアクセスを失うし、会議を開始する室の前面のタッチが動作を停止します。
