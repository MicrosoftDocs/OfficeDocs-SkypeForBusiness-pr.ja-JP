<a name="get-tools-to-support-a-rollout-of-microsoft-teams"></a>チームの Microsoft の導入をサポートするツールを入手します。
=================================================

お客様の成功キットの Microsoft チーム用に組織を正常に展開 Microsoft チームの組織を支援するツールを提供します。キットには、お知らせ、チラシ、ポスター、およびカウント ダウン、ヒントやテクニックのメールの電子メール テンプレートが含まれています。チーム リーダーや IT 管理者の作業の開始ガイドも含まれます。[お客様の成功キット](https://go.microsoft.com/fwlink/?linkid=854598)を今すぐダウンロードします。

### <a name="learn-adjust-and-repeat"></a>説明、調整、および繰り返し

関係者をここに推奨されるため、お客様の成功キットのリソースを使用する手順を行うとエキスパートとユーザーはフィードバックが共有されます。これらの貴重な情報は、アプローチ、メッセージング、その他の導入を推進して最終的には、ビジネスの結果にモデルを調整することで役立ちます。柔軟性のアジャイル サービス管理モデルは、ビジネスのペースで成功を提供すると、今日に重要です。この文書を開いた場合は、この展開の成功を担当する主要な個人のいずれかのと、現在の条件に基づいて、計画を調整する機能が組織に導入曲線を迅速化します。

Microsoft チームは、まだ堅牢なサービスを使用する単純な処理中に共同作業を促進設計されています。この旅の開始時に選択して、ビジネスの結果を定期的に再表示は、それらの優先チームの Microsoft の導入を測定可能なビジネスへの影響を提供していることを確認します。
