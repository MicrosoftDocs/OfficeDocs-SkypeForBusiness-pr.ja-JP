# <a name="use-activity-reports-for-microsoft-teams"></a>Microsoft Teams のユーザー アクティビティ レポートを使用する 

Microsoft 365 管理センターのアクティビティ レポートを使用して、組織内のユーザーがどのようにMicrosoft Teamsを使用しているかを確認できます。 たとえば、Microsoft Teamsをまだ使用していない場合は、作業を開始する方法や、Teamsを使用して生産性と共同作業を強化する方法を理解していない可能性があります。 組織では、アクティビティ レポートを使用して、トレーニングとコミュニケーションに関する優先項目を判断できます。

## <a name="how-to-view-the-teams-reports-in-the-reports-dashboard"></a>レポート ダッシュボードでTeams レポートを表示する方法

1. [Microsoft 365 管理センター](https://portal.office.com/adminportal/home)で、[**ReportsUsage** > ] を選択します。
 
2. [**使用状況**] ページで [**レポートの選択**] を選択し、レポートの一覧 **の [Microsoft Teams**] で、表示するレポートを選択します。

## <a name="teams-activity-reports-that-are-available"></a>使用可能なTeamsアクティビティ レポート

現在、表示できるアクティビティ レポートは 2 つあります。

- [Microsoft Teams ユーザー アクティビティ レポート](#microsoft-teams-user-activity-report) 
- [Microsoft Teams のデバイス使用状況レポート](#microsoft-teams-device-usage-report) 

### <a name="microsoft-teams-user-activity-report"></a>Microsoft Teams ユーザー アクティビティ レポート

Teams ユーザー アクティビティ レポートには、ユーザーがTeamsで実行する最も一般的なアクティビティのビューが表示されます。 これには、チャネルでチャットに参加するユーザーの数、プライベート チャット メッセージを介して通信するユーザーの数、通話や会議に参加するユーザーの数が含まれます。 この情報は、組織全体と個々のユーザーに対して確認できます。

![管理センターのユーザー アクティビティ レポートのスクリーンショット。](media/teams-user-activity-report.png)

## <a name="who-can-access-the-teams-activity-reports"></a>Whoは、Teams アクティビティ レポートにアクセスできます

アクティビティ レポートには、割り当てられているユーザーがアクセスできます。

- グローバル管理者ロール
- 製品固有の管理者ロール (Exchange、Skype for Business、またはSharePoint)
- レポート リーダー ロール

### <a name="reports-reader-role"></a>レポート リーダー ロール

管理者権限を持たないが、Teamsの導入やライセンスの使用状況の追跡を担当するユーザーにレポート リーダー ロールを割り当てることができます。 ロールを割り当てる方法については、「[Azure Active Directoryを持つユーザーに管理者ロールと管理者以外のロールを割り当てる](/azure/active-directory/fundamentals/active-directory-users-assign-role-azure-portal)」を参照してください。
