# <a name="cloud-video-interop-for-microsoft-teams"></a>Microsoft Teams のクラウド ビデオ相互運用性

クラウド ビデオの相互運用機能 (CVI) は、サードパーティ製の会議室 (telepresence) と (VTCs) がマイクロソフトのチーム会議に参加する個人のビデオ デバイスを有効にする Microsoft の認定済みのサードパーティ製ソリューションです。
 
マイクロソフトのチームでは、オーディオ、ビデオ、およびコンテンツの共有を含む会議の豊富なオンライン コンテンツのコラボレーションを取得します。 これは、デスクトップと web クライアントからの他に、マイクロソフトのチームとネイティブに統合する多くのパートナー デバイスに楽しみことができます。 ただし、多くのお客様が、すでにビデオ会議とビデオの個人の通信デバイスは、アップグレードのコストをすることができますに投資します。 クラウド ビデオの相互運用機能をアップグレードする準備ができるまで、既存のソリューションを使用して保持することを許可する簡単なソリューションを提供します。

## <a name="powershell-for-cloud-video-interop"></a>PowerShell の雲のビデオの相互運用機能

次の PowerShell コマンドレットは、(部分的に)、ビデオの相互運用をクラウドの展開を自動化するために使用できます。

- **Get CsTeamsVideoInteropServicepolicy**: マイクロソフトは、クラウドのビデオの相互運用機能を使用するのには、どの取引先を指定するため、サポートされているパートナーの事前構築されたポリシーを提供しています。<br>このコマンドレットを使用すると、組織内で使用できる事前に構築されたポリシーを識別できます。 許可 CsTeamsVideoInteropServicePolicy コマンドレットを使用することで、1 つまたは複数のユーザーにこのポリシーを割り当てることができます。
- **許可 CsTeamsVideoInteropServicePolicy**: このコマンドレットでは、組織で使用するため事前に構築されたポリシーを割り当てたり、特定のユーザーにポリシーを割り当てることができます。
- **新規 CsVideoInteropServiceProvider**: 組織の利用を希望するサポートされている CVI パートナーに関する情報を指定するのにはこのコマンドレットを使用します。
- **セット-CsVideoInteropServiceProvider**: 組織を使用する、サポートされている CVI パートナーに関する情報を更新するのにはこのコマンドレットを使用します。
- **Get-CsVideoInteropServiceProvider**: 組織内で使用するすべての構成されているプロバイダーを取得するのにはこのコマンドレットを使用します。
- **削除-CsVideoInteropServiceProvider**: 組織が不要になった使用するプロバイダーのすべてのプロバイダー情報を削除するのにはこのコマンドレットを使用します。
