# <a name="data-transfers-between-carriers-and-microsoft-for-operator-connect"></a>オペレーター サービスを利用した運送業者と Microsoft の間でのConnect

Microsoft は、オペレーターサービスに関連して処理する個人データのConnect。 お客様は、エンド ユーザーに次の情報を通知する必要があります。

- Microsoft のオペレーター は、Connectの機能Microsoft Teams、Teams 経由でネットワークを呼び出すことによって他のユーザーと通信Microsoft Teams。
- この機能が組織によって有効になっている場合、Microsoft は、サービスが適切に機能し、トラブルシューティングの目的で、情報を組織の通信サービス業者 (各オペレーター) と共有します。

オペレーターサービスを使用する際に Microsoft がオペレーターと共有する情報にはConnectがあります。

- 電話番号と、該当する場合は、電話番号に割り当てられた Microsoft 割り当て SIP URI (セッション開始プロトコルの Uniform Resource Identifiers)
- 通話の詳細と通話品質データ (発信者番号や通話受信者の電話番号など)通話の招待と終了時間。特定の呼び出し元識別子。呼び出し期間。サービス接続に関する情報 (呼び出し元と呼び出しの受信者の IP アドレスとオペレーティング システムなど)。そして
- 緊急通話の場所情報。

オペレーターは、上記の情報をエクスポートおよびダウンロードできます。

この情報は、オペレーターのプライバシーに関するプラクティスに従ってオペレーターによって処理されます。 詳細については、オペレーターのプライバシー ポリシーと使用条件を参照してください。 Microsoft のプライバシーに関するプラクティスについては、このリンクを参照して、マイクロソフトのプライバシーに関するTeamsを[参照してください](https://go.microsoft.com/fwlink/?LinkId=521839)。
