# <a name="teams-devices-for-education"></a>Teamsのデバイス

## <a name="overview"></a>概要

### <a name="hybrid-and-blended-education-is-here-to-stay"></a>ハイブリッドとブレンドされた教育はここにとどまる

ラーニング、ハイブリッドとブレンドが行なう予定です。 教育者と学生はどちらも、学習の方法と場所に関して柔軟性を必要とすることで、一連のスペースで実現される流動的な経験を必要とします。 教育は、物理的な教室や厳密に仮想環境に限定されません。リモート参加は、対人学習と同じ動的な方法で行います。 ハイブリッド学習とブレンド学習が提供する柔軟性は、すべてでは新しいのではなく、多くのユーザーに新しく提供され、大きな影響を及ぼす可能性があります。 

モダンでハイブリッドな教室は境界をなくします。 学生と教育者は、あらゆる状況ですべての学生のニーズを満たして設計された、同期学習と非同期学習、対話型のマルチモーダルエンゲージメントによって強化されます。 Teamsデバイスは、音声が聞こえなかないので、さまざまなスペースをサポートするように設計されています。 

### <a name="student-and-educators-alike-excel-with-microsoft-teams-rooms"></a>学生と教育者は同様に、会議室で優Microsoft Teamsします

-   Microsoft Teams会議室を使用すると、教室の統合から、より集中したコラボレーションのための小会議室まで、安全で安全で学生中心の学習が可能です。また、特定の学生のニーズや質問に対応するための 1 対 1 の学習も可能です。

-   デジタルまたはハイブリッド学習環境で個人用および共有デバイスを使用すると、学生は一生使う重要なスキルを身に付け、習得できます。

-   Teams機能は教室に限定されません。 教育者と管理者は、自宅、オフィス、または環境を最適化した場所から、同じ学校内または学区全体内の他の教職員と簡単に接続できます。 このTeams、プロフェッショナルな開発にどこからでもアクセスできます。

## <a name="space-considerations"></a>スペースに関する考慮事項

### <a name="what-is-the-role-of-the-space"></a>スペースの役割は何ですか?

物理スペースは、そこで発生するアクティビティの種類に対応するように設計されています。 アクティビティや接続の種類によって、会議の動作が異なる場合があります。 たとえば、教室では、役員室とは大きな異なる動作が表示されます。

家具は、スペースの使用方法を定義するのに役立ちます。 教室や会議スペースが有効になるには、家具や仕上げを考える必要があります。

既存のスペースの場合、テクノロジはルームの既存のレイアウトに合わせて配置する必要があります。 たとえば、プレゼンテーションと対人指導のためにテーブルと椅子が設定されている場合は、テクノロジに従う必要があります。 マイクとカメラは、部屋の使い方を考慮せずに、単に収まる場所に追加されるのではなく、部屋に最適に配置する必要があります。 リモート参加者は、プレゼンテーション資料を対人参加者として明確に確認する必要があります。

新しいスペースでは、部屋の使用目的は、家具と技術の両方の考慮事項を駆動する必要があります。 たとえば、複数の目的 (個々の作業やグループのやり取りなど) を容易にすることを目的とした部屋を設計する場合、スペースは最大の効率を得る目的で 2 つの別々の領域に分割する必要があります。

### <a name="what-is-the-function-of-the-space"></a>スペースの機能は何ですか?

*スペース内のアクティビティは、共同作業の共同作成に焦点を当てるでしょうか。*

創造性には、ユーザーのコラボレーションを中心に設計されたレイアウトとテクノロジの展開が必要です。 適切なスペースにより、ユーザーとテクノロジの間の作業をシームレスに行い、初心者ユーザーの摩擦をなくします。 Teamsデバイスは、若い学生が十分に操作し、教育者が最初の使用から制御し、カスタマイズするのに十分なユーザーフレンドリーに設計されています。 

*対人参加者とリモート参加者の両方とのコンテンツ共有が優先されるスペースについてはどうでしょうか。*

コンテンツ共有とリモート コラボレーションは、それぞれ異なる物理的なレイアウトとテクノロジエクスペリエンスを必要とします。 適切なスペースを使用すると、ディスカッションが容易に行え、共有コンテンツを簡単に確認できます。 どのスペースを装備する場合でも、学生寮のフレックススペースから講堂まで、有効にTeamsがあります。

## <a name="technology-considerations"></a>テクノロジに関する考慮事項

適切なオーディオとビデオテクノロジの選択肢は、ユーザーが会議の中で最もMicrosoft Teamsします。

*Audio*

音声に関する考慮事項は、リモート参加者と部屋の両方の参加者が明確に聞き取り、理解するために重要です。 マイクとスピーカーのカバー範囲には、ギャップを残さずに部屋のすべてのシートを含める必要があります。

### <a name="what-are-the-right-audio-devices-for-each-space"></a>各スペースに適切なオーディオ デバイスは何ですか?

各スペースで適切なマイクとスピーカーをペアリングすると、気を散らさずに明らかに聞こえ得る最良の結果が得られます。

-   自宅の一般的な部屋や教室のデスクなど、ノイズの多い可能性のあるスペースでは、ヘッドセットは会議の参加者にのみ音を制限することで、可能な限り最高のエクスペリエンスを提供します。 ヘッドセット マイクは、会議の音声のみを受け取る目的で設計され、会議出席者だけが何が起こっているのかを聞くことができます。

-   よりプライベートな設定 (学生の寝室など) では、デバイスの選択肢を拡張して、個人用スピーカーフォンのようなものを含めできます。 これらの個人用デバイスは、ヘッドセットと同じレベルの会話品質を実現しますが、デバイスを装着する必要はありません。

-   共有スペースでは、オプションは拡張されますが、会議の参加者は他のユーザーの声をはっきりと聞く必要があるという同じ原則が残ります。

    -   1 台のオーディオ デバイスは、オーディオ デバイスに比較的近い場所に座っている参加者の数が限られている小さなスペースに対して、展開が容易で高品質なオプションです。

    -   大きなスペースでは、マイクとスピーカーを分離して、スペース内のすべての参加者に対して最大のカバレッジを確保します。 中規模から大規模の部屋向けソリューションの多くは、構成を必要とし、マイクとスピーカーの物理的なインストールのみを必要とするキットの一部として利用できます。

    -   非常に大規模で最も複雑なスペースでは、大規模なホールから講師と聴衆の声をキャプチャし、同時にすべての人に通話を呼び出すなど、さまざまな種類のマイクとスピーカーを一緒に使用できるソリューションが用意されています。 これらのソリューションは、技術的に関連するソリューションの設計と展開に関する専門知識を持つ [Microsoft](https://cloudpartners.transform.microsoft.com/teams-mrpp) パートナーが展開する場合に最適です。

*Video*

ビデオはオーディオと同じほど重要です。 顔の表現やジェスチャーなどの言葉以外の物理的なコミュニケーションを取り込むのは、最大のエンゲージメントと理解を確保するために、教室や会議のコンテンツと同じほど重要です。 すべてのユーザーが会議室またはリモートで会議を適切に体験できるよう、会議室には適切なサイズのディスプレイを備える必要があります。 また、リモートから参加するユーザーに対して、会議室のすべての会議参加者を効果的にキャプチャするには、適切なカメラ範囲と視野 (FOV) が必要です。

### <a name="what-are-the-right-cameras-for-each-space"></a>スペースごとに適切なカメラは何ですか?

高品質のビデオは、最新のビデオコラボレーションTeams必要です。 カメラのテクノロジは、すべてのスペースで利用可能な高品質のカメラで大幅に進んだ。

-   ノート PC やタブレットのようなデバイスに含まれるカメラは、ビデオ通話に参加している個人が使用する最適な開始点です。 より豊富なビデオ エクスペリエンスを個人空間に提供するユーザー向けには、外部カメラは埋め込みデバイスからのステップアップを表し、より多くのシナリオや構成で使用できます。通話参加者が自分の最高の品質ビューを共有することを保証します。 これらのデバイスは、すべての PC で簡単な USB 接続を使用して簡単にインストールできます。

-   共有スペースでは、オプションと機能が展開されます。 小さい部屋は、サウンドバーなどの 1 台のオーディオ デバイスに含まれるカメラで覆われる場合があります。 通常、これらのカメラは広い視野を備え、1 つのショットでスペース内のすべての参加者を確実にキャプチャします。 これらのカメラの多くは、画像のインテリジェントなフレーミング (会議のユーザーのみを表示し、空の椅子は表示しない) など、スペースの使用状況を追跡して理解する機能も備わっています。

-   大きな部屋では、カメラはさまざまな場所でさまざまな場所で参加者をキャプチャするために必要な技術を備えました。 これらのカメラは、グループをキャプチャしたり、特定の個人に焦点を当てたりしながら、視覚的な明快さを維持できます。 アクティブなスピーカーのみを追跡および機能できるデバイスは、大規模なスペースで定期的に使用され、多くの参加者が自分の考えを寄稿する会議中に個別に取り上けられます。

### <a name="how-do-you-interact-virtually-beyond-video-streams"></a>ビデオ ストリームを超えて仮想的にやり取りする方法

Teams会議は、参加者の音声通話とビデオ通話以外にも行きます。 コンテンツを動的に共有および操作するには、さまざまな方法があります。

一般的なシナリオは、参加者のデバイスからコンテンツを直接共有する場合です。 個々のユーザーは、個々のアプリケーションまたは画面のコンテンツ全体から、すべてのユーザーが表示する会議に直接音声と視覚のエクスペリエンスを共有できます。 これは、インストールされているデバイスを持つすべての共有スペースに拡張されます。 Microsoft Teamsまた、会議室には、会議に個別に参加する機能がなくても参加者が共有できる有線モニター接続を使用するオプションがあります。

物理的な世界のコンテンツの場合、Microsoft Teams Room システムを備えるスペースはホワイトボードまたはドキュメント カメラを処理して、空間からのコンテンツの調整されたレビューとディスカッションを可能にできます。

完全に対話的なエクスペリエンスを提供するために、対話型ディスプレイ (Surface Hub ファミリの製品や Microsoft デジタル ホワイトボードなど) を使用すると、対人学生やリモート学生向けのコンテンツを簡単に作成できます。
