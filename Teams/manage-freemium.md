# <a name="manage-the-free-version-of-microsoft-teams"></a>無料版の Microsoft Teams を管理する

無料バージョンの Microsoft Teamsでは、組織に最初にサインインするユーザーの管理者ロールは制限されています。 この制限付きロールを持つユーザーは、チーム メンバーの追加と削除を行い、他のメンバーを招待できるかどうかを指定できますが、このユーザーには高度な管理機能はありません。Microsoft Teams 管理センターへのアクセス権はありません。 詳細については、「ユーザーを他のユーザー[に招待する (Teams)」を参照してください](https://support.office.com/article/invite-people-to-teams-free-53a9b20c-2ad7-442e-967c-2e9305e96463)。

管理の強化など、Teams 機能の完全なセットを取得するには、ユーザーに適した Microsoft 365 または[Office 365](upgrade-freemium.md)サブスクリプション プランを購入して、Teams の完全なバージョンにアップグレードする必要があります。 

バージョンとその機能の詳細Teamsプランの比較に関する[ページTeamsしてください](https://products.office.com/microsoft-teams/free)。
