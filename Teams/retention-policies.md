<span data-ttu-id="80dcb-132">Teams のチャットは、チャットに参加している各ユーザーのメールボックスにある SubstrateHolds という隠しフォルダーに保存されます。また、Teams のチャネル メッセージは、チーム用のグループ メールボックス内の SubstratesHolds という隠しフォルダーに保存されます。</span><span class="sxs-lookup"><span data-stu-id="80dcb-132">Teams chats are stored in a hidden SubstrateHolds folder in the mailbox of each user in the chat, and Teams channel messages are stored in a hidden SubstratesHolds folder in the group mailbox for a team.</span></span> <span data-ttu-id="80dcb-133">Teams が使用する Azure を利用したチャット サービスでもこのデータが保存されます。既定では、このサービスはデータを無期限に保存します。</span><span class="sxs-lookup"><span data-stu-id="80dcb-133">Teams uses an Azure-powered chat service that also stores this data, and by default this service stores the data forever.</span></span> <span data-ttu-id="80dcb-134">Teams のアイテム保持ポリシーでデータを削除すると、Exchange メールボックスと基になるチャット サービスの両方からデータが完全に削除されます。</span><span class="sxs-lookup"><span data-stu-id="80dcb-134">With a Teams retention policy, when you delete data, the data is permanently deleted from both the Exchange mailboxes and the underlying chat service.</span></span>
