# <a name="ip-phones-certified-for-microsoft-teams"></a>Microsoft Teams で認定を受けた IP 電話

IP 電話の認定プログラムを使うと、認定された卓上電話や電話会議の電話機を使用しているお客様は高品質の音声通話と会議エクスペリエンスを体験できます。

## <a name="announcing-certified-for-microsoft-teams"></a>Microsoft Teams の承認済みのアナウンス

Skype for Business で認定された IP 電話の範囲をお客様に提供するだけでなく、Teams の IP 電話認定プログラムの一環として、Microsoft Teams に対して認定される新しい IP 電話ハードウェアの可用性についてもお知らせします。 この認定プログラムは、より高いパフォーマンスターゲットと音質の指標を備えた高品質バーを確保します。 電話のベンダーは、この高いバーに対応するために積極的に取り組んでおり、この暦年の新しい要件に対して修飾されます。 一部のパートナーも、チームの IP 電話と同じハードウェアで Skype for Business サービスのサポートを提供するために取り組んでいます。 チーム認定 IP 電話の利点は次のとおりです。

- ハードウェアボタンの統合と LED 通知を使用したネイティブチームのエクスペリエンス
- 予定表の統合とワンタッチ会議の参加
- Intune との統合を使用して、Microsoft デバイス管理ソリューションを使用して展開および管理する
