# <a name="supported-rate-centers-and-coverage-for-calling-plans"></a>通話プランでサポートされているレート センターとカバレッジ

この記事では、北米の通話プランでサポートされているレート センターとカバレッジの一覧を示します。 レート センターは、従来、公衆交換電話網 (PSTN) のローカル通話、課金レート、電話番号の割り当ての境界を定義する地理的領域の業界用語です。 多くの場合、業界がすべての包括的なプランまたは分単位のバンドルに移行する場合、料金センターは課金の重要性が低くなっていますが、一部のユーザーはまだ使用しています。

[北米 スプレッドシートのサポートされるレート センターとカバレッジ マトリックス](https://www.microsoft.com/download/details.aspx?id=102534)には、サポートされているレート センターが一覧表示されます。 新しい電話番号を取得する場合、または既存のプロバイダーからTeamsに電話番号を転送する場合は、スプレッドシートをダウンロードし、それを使用してレート センターを検索します。 レート センターがわからない場合は、エリア コード (NPA) とプレフィックス (NXX) に基づいてインターネットで検索できます。
新しい番号を取得していて、要求した番号がない場合は、同じレート センターから番号を提供しようとします。

新しい番号を取得していて、要求した番号がない場合は、同じレート センターから番号を提供します。
