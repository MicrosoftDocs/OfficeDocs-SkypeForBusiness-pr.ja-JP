# <a name="manually-submit-a-port-order"></a>手動で番号移行注文を送信する

一部の国や地域では、電話番号の取得、電話番号の譲渡、電話番号の譲渡、住所の変更などを行うために、サービスリクエストを手動で送信する必要がある場合があります。 国や地域ごとに何が必要かを確認したり、番号移行の詳細については、「 [組織の電話番号を管理](../manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization.md)する」を参照してください。

この記事の手順を使用して、 [Microsoft Teams 管理センターの [移植] ウィザード](transfer-phone-numbers-to-teams.md)に国または地域が表示されていない場合は、手動でポート注文を送信します。

## <a name="manually-submit-a-new-port-order-request"></a>新しいポート注文の依頼を手動で送信する

新しいポート注文を手動で送信するには、お客様の地域の PSTN サービスデスクに、完了した承認状 (LOA) フォームを直接送信します。

1. お住まいの国または地域の [Loa](../manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization.md) をダウンロードしてください。
2. フォームに入力します。
3. お住まいの地域の [PSTN サービスデスク](../manage-phone-numbers-for-your-organization/contact-pstn-service-desk.md) に送信してください。

## <a name="what-else-should-you-know-about-number-porting"></a>番号の移行について知っておくべきその他のこと

- 通話プランを使用するには、ライセンスを購入してユーザーに割り当てる必要があります。 「 [Teams のアドオンライセンス](https://docs.microsoft.com/microsoftteams/teams-add-on-licensing/microsoft-teams-add-on-licensing)」をご覧ください。

- 各ユーザーに、利用可能な新しい電話番号を割り当てる必要があります。 「[ユーザーの電話番号の割り当て、変更、削除を行う](../assign-change-or-remove-a-phone-number-for-a-user.md)」を参照してください。
