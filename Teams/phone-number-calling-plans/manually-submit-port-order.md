# <a name="manually-submit-a-port-order"></a>手動で番号移行注文を送信する

一部の国や地域では、電話番号の取得、電話番号の譲渡、電話番号の譲渡、住所の変更などを行うために、サービスリクエストを手動で送信する必要がある場合があります。 国や地域ごとに何が必要かを確認したり、番号移行の詳細については、「 [組織の電話番号を管理](../manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization.md)する」を参照してください。

この記事の手順に従って、手動でポート注文を作成して送信します。これには、 [Microsoft Teams 管理センターの [移植] ウィザード](transfer-phone-numbers-to-teams.md)にお住まいの国または地域が表示されていない場合に使用します。

## <a name="manually-submit-a-new-port-order-request"></a>新しいポート注文の依頼を手動で送信する

お客様の地域の PSTN サービスデスクまたは Microsoft 365 管理センターに、入力した承認状 (LOA) フォームを直接送信して、サービスリクエストを作成することで、新しいポート注文を手動で送信できます。

### <a name="send-your-letter-of-authorization-directly-to-the-pstn-service-desk"></a>PSTN サービスデスクに直接、承認状を送信する

お住まいの国または地域の [Loa](../manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization.md) をダウンロードし、フォームに記入して、お住まいの地域の [PSTN サービスデスク](../manage-phone-numbers-for-your-organization/contact-pstn-service-desk.md) に送信します。

- 米国およびカナダで、 [お客](mailto:ptn@microsoft.com)さまのリクエストを送信します。
- ヨーロッパで、 [お客](mailto:ptneu@microsoft.com)さまのリクエストを送信します。
- アジアで、 [リクエストを送信](mailto:ptnapac@microsoft.com)します。

### <a name="submit-a-service-request-in-the-microsoft-365-admin-center"></a>Microsoft 365 管理センターでサービスリクエストを送信する

1. Microsoft 365 管理センターの左のナビゲーションで、[ **Support**  >  **新しいサービスリクエスト**のサポート] に移動します。  [ **サポート** ] が表示されない場合は、左側のナビゲーションで [ **ナビゲーションのカスタマイズ** ] に移動し、[ **サポート** ] チェックボックスをオンにして追加します。
2. [ **ヘルプが必要ですか?** ] ウィンドウで、[ **サポートに問い合わせる**] を選択します。
3. [ **サポートに問い合わせる** ] ウィンドウで、次の操作を行います。

    1. リクエストのタイトル (「ポート注文のリクエスト」など) と説明を入力して、電話番号とメールアドレスを確認して、希望の連絡方法を選択します。
    2. [ **添付**ファイル] の下の [ **ファイルの追加**] をクリックして、完成した [loa](../manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization.md)をアップロードします。
    3. [ **お問い合わせ先**] をクリックします。

## <a name="what-else-should-you-know-about-number-porting"></a>番号の移行について知っておくべきその他のこと

- 通話プランを使用するには、ライセンスを購入してユーザーに割り当てる必要があります。 「 [Teams のアドオンライセンス](https://docs.microsoft.com/microsoftteams/teams-add-on-licensing/microsoft-teams-add-on-licensing)」をご覧ください。

- 各ユーザーに、利用可能な新しい電話番号を割り当てる必要があります。 「[ユーザーの電話番号の割り当て、変更、削除を行う](../assign-change-or-remove-a-phone-number-for-a-user.md)」を参照してください。
