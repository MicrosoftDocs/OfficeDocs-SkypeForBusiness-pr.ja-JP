# <a name="microsoft-teams-device-usage-report"></a>デバイスの使用状況レポートをマイクロソフトのチーム

チームのデバイスの利用状況レポートをマイクロソフトのチームとビジネス管理センターの Skype のでは、ユーザーがチームに接続する方法に関する情報を提供します。 レポートを使用するに移動するときに自分のモバイル デバイスからのチームを使用して数を含む、組織全体で使用されているデバイスを参照してください。  

![マイクロソフトのチームとビジネス管理センターの Skype でチームのデバイスの使用状況レポートのスクリーン ショット](../media/teams-reports-device-usage.png "マイクロソフトのチームとビジネス管理センターの Skype でチームのデバイスの使用状況レポートのスクリーン ショット")

## <a name="view-the-report"></a>レポートを表示します。

1. マイクロソフトのチームとビジネス管理センターでは、左側のナビゲーションでの Skype を、**分析とレポート**をクリックし、[**レポート**] で**デバイスの使用状況のチーム**を選択を移動します。 
2. [**日付範囲**] で、範囲を選択し、**レポートの実行**] をクリックします。
