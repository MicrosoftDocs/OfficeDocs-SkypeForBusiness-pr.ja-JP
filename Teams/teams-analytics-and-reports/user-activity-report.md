# <a name="microsoft-teams-user-activity-report"></a>マイクロソフトのチームのユーザーの活動レポート

チームのユーザー アクティビティ レポートでは、チーム、組織内のユーザーが実行するアクティビティの種類を把握できます。 たとえば、ユーザーの数が 1 対 1 の呼び出しを介して通信、チャネルのメッセージをどのように多くのユーザーが通信して、ユーザー数プライベート チャット メッセージを表示できます。

![マイクロソフトのチームとビジネス管理センターの Skype でチームのユーザー アクティビティ レポートのスクリーン ショット](../media/teams-reports-user-activity.png "マイクロソフトのチームとビジネス管理センターの Skype でチームのユーザー アクティビティ レポートのスクリーン ショット")

## <a name="view-the-report"></a>レポートを表示します。

1. マイクロソフトのチームとのビジネス管理センターは、左側のナビゲーションでは、Skype して、**分析とレポート**、および [**レポート**] で、**チームのユーザーの利用状況**を選択を参照してください。 
2. [**日付範囲**] で、範囲を選択し、**レポートの実行**] をクリックします。
