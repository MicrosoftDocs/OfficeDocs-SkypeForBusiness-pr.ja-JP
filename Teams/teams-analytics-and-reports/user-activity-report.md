# <a name="microsoft-teams-user-activity-report"></a>Microsoft Teams ユーザー アクティビティ レポート

[Teams アクティビティ レポート] では、組織内のユーザーがアクティビティで行うアクティビティの種類に関する分析情報Teams。 予定外の会議 (1:1 とグループ通話) を通じて、計画外に通信するユーザーの数を表示できます。 ユーザーが開催した会議Teamsユーザーが参加した会議Teamsを確認します。 画面、ビデオ、音声の分数、チャットコミュニケーションの統計情報 (チャネル メッセージに返信するユーザーと投稿するユーザーの数、1 対 1 またはグループ チャット メッセージに関与するユーザーの数など) の詳細を確認します。

> [!NOTE]
> ユーザー アクティビティ レポートをスケジュールする機能は、現時点では使用できません。

## <a name="view-the-user-activity-report"></a>ユーザー アクティビティ レポートを表示する

これらの変更を行うには、Teams サービス管理者であることが必要です。 「[Teams 管理者ロールを使用してチームを管理する](../using-admin-roles.md)」をご覧いただき、管理者ロールとアクセス許可を取得する方法について読んでください。

1. 管理センターの左側のナビゲーションでMicrosoft Teamsレポートの [分析] **&を** > **選択します**。 **[レポートの表示]** タブの **[レポート]** で、**[Teams ユーザーの利用状況]** を選択します。
2. [ **日付範囲] で** 範囲を選択し、[レポートの実行] **を選択します**。

    ![吹き出しTeams付き、Teamsアクティビティ レポートのスクリーンショット。](../media/teams-reports-user-activity-with-callouts.png "Teams 管理センターの Teams ユーザー アクティビティ レポートのスクリーンショット (吹き出し付き)")

## <a name="make-the-user-specific-data-anonymous"></a>ユーザー固有のデータを匿名にする

ユーザー アクティビティ レポートTeamsデータを匿名化するには、グローバル管理者である必要があります。 これにより、レポート内の表示名、電子メール、AAD ID などの識別可能な情報 (MD5 ハッシュを使用) が非表示になります。

1. [Microsoft 365 管理センター]を選択し、[**組織]** \> 設定に設定し、[サービス] タブの **[** レポート] を選択 **します**。
    
2. [ **レポート]** を選択し、[匿名識別子 **を表示する] を選択します**。 この設定は、管理センターと管理センターの使用状況Microsoft 365 管理センターにTeams適用されます。
  
3. [変更の **保存] を選択します**。
